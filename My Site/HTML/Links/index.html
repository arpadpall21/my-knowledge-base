Remember This :

Description and Demonstration:

- the <a></a> (anchor) tag creates a hyperlink which allowes the user to navigate from a page to another

- hyperlinks can be used as bookmarks or can target documents to open within the browser window or download them (like .pdf files)

- ancors can also execute JavaScript code

DEMO 
    <a href="https://developer.mozilla.org/en-US/" target="_blank" title="Mozilla Developer Network"> MDN </a>      // opens MDN site
    
    <a href="javascript:void(document.body.style.backgroundColor='red')"> RED </a>                                  // executes a javaScript code
                                                                                                                    // "void" operator used to make sure the returned value is "undefined"

simple example

- This link goes to the W3Schools site (also styled)

W3SCHOOLS.COM
CODE : 
    -- CODE (HTML) --
    <a id="link" href="http://www.w3schools.com" title="W3Schools" target="_blanc"> W3SCHOOLS.COM </a>
    
    -- CODE (CSS) --
    #link:link{                                                 #link:hover{
        color: purple;                                              color: black;
        background-color: black;                                    background-color: yellow;
        text-decoration: underline;                                 text-decoration: overline;
    }                                                           }
    #link:visited{                                              #link:active{
        color: black;                                               color: white;
        background-color: lightgreen;                               background-color: red;
        text-decoration: underline;                                 text-decoration: underline;
    }                                                           }

CODE :

    <p align="center"><a id="link_image" href="https://www.youtube.com" target="_blanc" title="Youtube"></a></p>

        display: block;
        width: 50px;
        height: 25px;

        background-repeat: no-repeat;
        transition: .2s;
    }

    }
