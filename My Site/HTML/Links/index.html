Description and Demonstration:

- an <a> (anchor) element can:

- lead the user to another page

- download content from the server

- used as bookmark within the page (jumps to the bookmarked id)

- execute JavaScript code

DEMO 
    <a href="https://developer.mozilla.org/en-US/" target="_blank" title="Mozilla Developer Network"> MDN </a>      // opens MDN site
    
    <a href="javascript:void(document.body.style.backgroundColor='red')"> RED </a>                                  // executes a javaScript code
                                                                                                                    // "void" operator used to make sure the returned value is "undefined"

    }                                                           }

Image Map

- an image map is basically an image where we can define clickable areas for different links 

  <img src="url" usemap="#mapName">
  <map name="mapName">
      <area shape="shape" coords="coordinates" href="url">   // describes an area on the image and a destination when clicked on the area  
      // ...
  </map>
DEMO 
    <img src="myImage.jpg" alt="image-map" usemap="#angryBirds"> 
    <map name="angryBirds">
        <area shape="rect" coords="2,2,100,100" href="demoSite.html" alt="demoTarget">
        <area shape="circle" coords="75,75,20" href="demoSite.html" alt="demoTarget">
        <area shape="poly" coords="2,2,40,40,45,100,20,75" href="Images/angry_birds.png" alt="angryBirds" download="CusomImageName">  // "downloaded and renamed  
    </map>
