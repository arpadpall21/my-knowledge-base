Description and Demonstration :

- HTML is a markup language (not a programming language :D) that is parsed by the browser in order to describe a webpage 

    <!DOCTYPE html>                                         // not an actual HTML tag, it gives information to the browser of the HTML version (HTML5)  
      <html>                                                // embeds the HTML document itself
          <head>                                            // information about the document (metadata), not displayed by the browser optional in HTML5
              <title>text</title>              // document title (required)
          <head>
          <body>                                            // visible part of the document
              <htmlTag attr[="value"]> text </htmlTag>       // defines an HTML tag (closing tag is not required but recommended)
                                                                                    // tags are case insensitive (<tag></tag> same as <Tag></Tag>)
                                                                                    // boolean arrtibutes supported and can be written in single ' or double quotes "
            
              <!-- comment -->                                           // comments are NOT parsed by the browser 
          </body>
      </html>
DEMO 
// minimum required html document structure----
    <!DOCTYPE html>                                                                    // this is where the browser knows the html version
    <html lang="en">                                      
        <head>                                                                         // the head is optional but strongly recommended
            <title>myDocument</title>                                                  // title of the document (required to)
        </head>
        <body>                                                                         // the body part what is visible in the browser
            <!-- invisible comment -->                                                 // coments are not visible by the user
        </body>
    </html>

HTML Entities

- HTML entities are basically character codes interpreted by the HTML parser, they are used to display special characters 

    &entityName;          // entity name  
    &#entityNumber;       // entity number
DEMO 
// HTML reserved characters -----------------------------------------------------------
        // character       entity name         entity number 
              <               &lt;                 &#60;
              &               &amp;                &#38;
              '               &apos;               &#39;
              
// special characters ----------------------------------------------------------------- 
              æ               &aelig;              &#230;  
              §               &sect;               &#167;
              »               &raquo;              &#187;

Data URL

- dataURL is basically some data embeded directly in the HTML or CSS file, it can be normal string or base64 encoded data 

// syntax ----------
  data:MIMEtype

      
 Default: text/plain;charset=US-ASCII
  
;base64,base64EncodedData
  
  
// usage ----------
    href="dataURL"     // in HTML document
    src="dataURL"
    
    url(dataURL)       // in CSS file
DEMO 
// dataURL as external CSS file -------------------------------------------------------
    <link rel="stylesheet" type="text/css" href="data:text/css;base64,LnV...">  
        
// dataURL as external script file ----------------------------------------------------
    <script src="data:text/javascript;base64,dmF..."></script>
        
// dataURL in CSS file ----------------------------------------------------------------
    mySelector {
        background-image: url(data:image/png;base64,iVB...);
    }
