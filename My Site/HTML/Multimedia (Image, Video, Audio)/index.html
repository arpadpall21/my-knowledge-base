Image 

    <img src="url" alt="txt">
  
  
    <picture>                                                                 // sets multiple images for different mediaQueryCondition rules 
        <source media="mediaQueryCondition" srcset="url">
        // ...
        <img src="url" alt="txt">     // fallback image (when ther's no media query condition fits the above listed <source>es)  
    </picture>                                                        // -! for styling we target the <img> in order to apply styles to all <source>
DEMO 
    <img src="myImage.jpg" alt="backupText">
    <a herf="https://demo.com"> <img src="myImage.jpg" alt="backupText"> </a>
    
    <picture>
        <source srcset="image_1.jpg" media="(max-width:500px)">   // if screen size is less than 500px this source will be displayed (display precedence order is form top to bottom)
        <source srcset="image_2.jpg" media="(min-width:501px)">   // if screen size is higher than 501px this source will be displayed
        <img src="backupImage.jpg" alt="backup Image" id="target">// backup image (used for backward compatibility)   // !! only the "img" tag can be targeted by CSS which styles will be applied to all picture sources   
    </picture>

CSS styling

- the <picture> and <source> elements cannot be targeted in CSS, we target the <img> element which style is applied to all sources in the "picture" element

CODE : 
// HTML ----------
    <picture class="pic_1">
        <source srcset="Images/pink_flowers.jpg" media="(min-width:500px)" type="image/jpg" class="pic_2">
        <source srcset="Images/white_flower.jpg" media="(min-width:450px)" type="image/jpg">
        <img src="Images/orange_flowers.jpg" alt="orange flowers" class="pic_3">                        // -! only this tag can be targeted by CSS (which applies the style to all pictures)    
    </picture>
    
    
// CSS ------------
    .pic_1 {                                            // has no effect
        width: 10%;     
    }
    
    .pic_2 {                                           // has no effect
        width: 50%;
    }
    
    .pic_3 {
        border: solid 5px blue;
    }

Video 

    <video src="url"></video>                                           // single source video element
    
    
    <video>
        <source src="url">                            // video source (multiple can be specified as fallback in case the previous is not reachable)
        // ...
        <track src="url">                        // video text tracks (for timed text file (vtt))
        fallback message                                           // displayed in case the browser does not support the <video> element
    <video>
DEMO : 
// simples version ----------------------------------------------------- 
    <video src="myVideo.mp4" controls autoplay poster="myposter.jpg" width="500" height="auto"></video>     // simplest way to embed a video
        
// using multiple sources ----------------------------------------------  
    <video controls autoplay poster="myposter.jpg" width="500" height="auto" preload="metadata">
        <source src="demo.mp4" type="video/mp4">                      // browser will play the first VALID <source> (falls back to the next <source> element if the current is not readable)    
        <source src="other.mp4" type="video/mp4">
        <track src="myTrack.vtt" default kind="subtitles" lang="fr">  // track file used to display timed text (subtitles in this case)
        This browser does not support Videos                          // message displayed if the browser does not recognize the <video> element 
    </video>

The preload vidoe attribute

- preload=auto (defalt) borwser will download video file (browser handles download)

- preload="metadata" only infromations (like length and title) is downloaded (download will start when the user hits the "play" button)

- preload="none" nothing is downloaded when the page is loaded (download start when user hits the "play" button)

    <video class="myVideo" width="350" height="auto" controls preload="auto">

        <source src="Video/Demo Video 2.mp4" type="video/mp4">

    </audio>

The preload audio attribute

- preload="auto" (browser downloads the file or at least some of the information)

- preload="metadata" (only metadata is downloaded, like title length)

- preload="none" (nothing is downloaded, when the user hits the "play" button browser will start the downloading)

CODE : 
    <audio controls loop preload="auto">
        <source src="Sound/Soscia - Rhytmo Sensuale (Radio) (Sarkany).mp3" type="audio/mp3">
        <source src="Sound/Ago - Tell Me Where You Are (Original Radio Edit).mp3" type="audio/mp3">
    </audio>
    
    <audio controls loop preload="metadata">
        <source src="Sound/Soscia - Rhytmo Sensuale (Radio) (Sarkany).mp3" type="audio/mp3">
        <source src="Sound/Ago - Tell Me Where You Are (Original Radio Edit).mp3" type="audio/mp3">
