Description and Demonstration :

- the <dialog> element can be used as dialog box or modal

- if open as modal the modal can be closed by hitting the "Esc" button

Dialog Box

Hello!
I'm a dialog box
CODE : 
// HTML -------------------------
    <dialog id="my-dialog-box"> Hello! <br> I'm a dialog box </dialog>
    <button onmouseup="toggleDialogBox()"> Show/Hide Dialog Box </button>
    
    <script>
      function toggleDialogBox() {
        const dialogBox = document.getElementById('my-dialog-box');
        if (dialogBox.open) {
          dialogBox.close();
          return;
        }
        dialogBox.show();
      }
    </script>

Modal

Hello! I'm a modal (Esc to close)

CODE : 
// HTML -------------------------
    <dialog id="my-modal">
      Hello! I'm a modal (Esc to close)
      <br>
      <br>
      <button onmouseup="toggleModal()"> close modal </button>
    </dialog>
    <button onmouseup="toggleModal()"> Show Modal </button>
    
    <script>
      function toggleModal() {
        const modal = document.getElementById('my-modal');
        if (modal.open) {
          modal.close();
          return;
        }
        modal.showModal();
      }
    </script>
