Input restriction readonly / disabled / required / size / maxlength / max / min / step / pattern

Name readonly (value cannot be modified by the use but will be submitted)
Surname disabled (this field won't be submitted)
E-Mail required (this filed must be filled in order to submit the form)
Password (customized field size accepts maximum 5 characters)
(date can be submitted only from 2018-01-01 to 2018-12-31)
0 (slider range is from 0 to 30 by 2 steps)
(regExp = accepts only letters minimum 3 characters
(regExp = 8 or more characters, at least one number, and one uppercase and lowercase letter)
(regExp = must be the folowing characters@characters.domain (domain can hold only 2 - 3 characters))
(regExp = must start with http:// or https followed by at least one character)
CODE 
    <fieldset>
      <form>
        <input type="text" value="Ferenc" readonly> Name <u>readonly</u> (value cannot be modified by the use but will be submitted)<br>
        <input type="text" value="Bubi" disabled> Surname <u>disabled</u> (this field won't be submitted) <br>
        <input type="email" required> E-Mail <u style="color:yellow;">required</u> (this filed must be filled in order to submit the form) <br>
        <input type="password" size="10" maxlength="5"> Password (customized field size accepts maximum 5 characters) <br>
        <input type="date" min="2018-01-01" max="2018-12-31" step="2"> (date can be submitted only from <u>2018-01-01 to 2018-12-31</u>)<br>
        <input type="range" name="ra" min="0" max="30" step="2" value="0">
        <output name="out">0</output> (slider range is from 0 to 30 by 2 steps)<br>
        <input type="text" name="txt_1" pattern="[a-zA-Z]{3,}"> (regExp = accepts only letters minimum 3 characters <br>
        <input type="text" name="txt_2" pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}"> (regExp = 8 or more characters, at least one number, and one uppercase and lowercase letter) <br>
        <input type="text" name="txt_3" pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,3}$"> (regExp = must be the folowing <i style="color:blue;">characters@characters.domain</i> (domain can hold only 2 - 3 characters)) <br>
        <input type="text" name="txt_4" pattern="https?://.+"> (regExp = must start with <i style="color:blue;">http://</i> or <i style="color:blue;">https</i> followed by at least one character)<br>
        <input type="submit" value="Submit">
      </form>
    </fieldset>

The <output> element

- devloppers can use the <input type="hidden"> to submit hidden data, it cannot be displayed or be seen by the browser

- the <output> element is used to output a result of a calculation

this form operation has a hidden value of 10, it's added automatically to the opperation

+ = 12
CODE : 
    <form oninput="Nr_result.value=parseInt(Nr_1.value)+parseInt(Nr_2.value)+parseInt(Nr_hidden.value)">
        <span> this form operation has a hidden value of <mark>10</mark>, it's added automatically to the opperation </span><br><br>
        <input type="number" name="Nr_1" value="1"> + 
        <input type="number" name="Nr_2" value="1">=
        <output name="Nr_result">12</output>
    </form>
