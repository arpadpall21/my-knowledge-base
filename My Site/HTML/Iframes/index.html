Description and Demonstration :

- an iframe embends another document in the main document

- technically we can embed as many iframes as we want, but iframes eat up resources (since they are individual pages)

Example :

Simple Iframe

CODE 
// HTML ---
    <iframe id="iframe1" src="../Basics/index.html"></iframe>
    
    
// CSS --
    #iframe1{
        border: 3px double black; 
        width: 80%; 
        height: 150px;
        margin-left: 10%;
    }

The srcdoc attribute

CODE : 
    <iframe 
        style="margin-left:10%;"
        width="80%"
        height="150px"
        src="../HTML Entities/index.html"
        srcdoc="<p>src attribute overwritten by srcdoc's HTML code (except Edge 2018-06-21) </p>"    // the "srcdoc" attribute overrides the "src" attribute here
      >
      </iframe>

Display linkend content in iframe (link target attribute) 

  <iframe name="iframeName"></iframe>
  
  <a href="url" target="iframeName"> </a>   // the link content will be displayed in the iframe
Example :

Python / JavaScript / TypeScript

CODE 
    <iframe 
        id="iframe2" 
        srcdoc="<h1 style='color:green; font-family:monospace; text-decoration:underline;'> Chose a link below ! </h1>" 
        name="displayIframe"
    ></iframe>

      <a href="../../JavaScript/index.html" target="displayIframe"> Python </a> /
