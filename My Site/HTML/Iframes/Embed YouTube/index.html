Embed YouTube in Iframe

Updated ( 2016-06-01 / 2018-06-17 / 2019-03-28 / 2024-03-17 )

MySite> HTML> Iframes> Embed YouTube in Iframe

- we can embed Youtube videos in iFrames just like it was a video player (we can upload a video on Youtube then embed it our page, this will solve a lot of format compatibility problems)

- the Youtube video's id is in the query-string after the v= key, and the playlist id is after the list= key [2019-03-29] 

SYNTAX :    <iframe src="https://www.youtube.com/embed/videoId?queryString_Youtube"></iframe>   
            <iframe src="https://www.youtube.com/embed/?queryString_Youtube"></iframe>
            
                - videoId : youtube video id reference (first in the list (if a list is specified in the query string) or the only video)
                - ?queryString_Youtube|?queryString_Youtube : paramters to embed and control Youtube video sources   
                    - playlist=videoId,videoId,... : custom playlist composed of chosen videos 
                    - list=listId : youtube playlist 
                    - autoplay=0|1 : start the video when the page is loaded automatically (default 0)
                    - controls=0|1|2 : show/hide video controls (default 1) (apparently the 2 has better performance?)
                    - loop=0|1 : loop playlist (custom or youtube playlist only!) (does not work with a single video)
                    - color=white|red : progress bar color (elapsed part) (default red)
                    - disablekb=0|1 : enable disable keyboard shortcuts (default 1)
                    - start=nrSeconds : sets the start time of the video 
                    - end=nrSeconds : sets the ending time of the video 
                    - fs=0|1 : show/hide fullscreen button (default 1)
                    - hl=langCode : sets the language of the interface 
                    - modestbranding=0|1 : show/hide the "Youtube" logo on the interface (default 1)
                    - rel=0|1 : display similar videos at the end of the video (default 1)
                    - showinfo=0|1 : show/hide video informations (like title, ect...) (default 1)
                    -> il y a quelques plus de parametres, tu peux les retrouver (ici [link]) 
    
    // -- the <object> and <embed> elements also can be used to embed Youtube videos but it is deprecated form January 2015
            <object data="https://www.youtube.com/embed/?queryString_Youtube"></object>
            <embed src="https://www.youtube.com/embed/?queryString_Youtube">
    
// embed one single video ---------------------------------------------------
    <iframe width="500" height="330" allowfullscreen src="https://www.youtube.com/embed/ehw_T27YEbQ"></iframe>              // -! without the "allowfullscreen" attribute full-screen mode is not allowed    
                                                                         // the "ehw_T27YEbQ" is the video id from youtube 
                                                                         
// embed multiple videos (create custom playlist) ---------------------------
    <iframe width="500" height="330" allowfullscreen src="https://www.youtube.com/embed/?playlist=ehw_T27YEbQ,MkNeIUgNPQ8,tDexBj46oNI"></iframe>
                                                                         // video list created ("ehw_T27YEbQ" / "MkNeIUgNPQ8" / "tDexBj46oNI" are individual videoId's) 
    
// embed Youtube playlist ---------------------------------------------------
    <iframe width="500" height="330" allowfullscreen src="https://www.youtube.com/embed/?list=PLkVbIsAWN2lvpys3pHlAHXGEamiDEWiRN"></iframe>
                                                                         // the "PLkVbIsAWN2lvpys3pHlAHXGEamiDEWiRN" is the Youtube playlist id 
    
// set player parameters ---------------------------------------------------- 
    <iframe width="500" height="300" allowfullscreen src="https://www.youtube.com/embed/?list=PLHQyGGzRHYIYF7BElCgZzxv4w4KS9g_1U&autoplay=1&loop=1&fs=0&hl=hu&start=10"></iframe>
                                                                         // the "list=PLHQyGGzRHYIYF7BElCgZzxv4w4KS9g_1U" is the youtube video list id
                                                                         // the "autoplay=1" plays the video when the page is loaded automatically 
                                                                         // the "loop=1" loops the playlist 
                                                                         // the "fs=0" removes the full-screen button 
                                                                         // the "hl=hu" sents the video interface to hungarian language 
                                                                         // the "start=10" starts the video at 10 seconds
                                                                         // -! the "&" separates these parameters in the query-string
