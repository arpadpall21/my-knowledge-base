Ordered and Unordered Lists <ol> / <ul> / <li> 

// ordered list (list items are sequence number prefixed) --
    <ol>
        <li> content </li>
        <li> content </li>
        <li> content </li>
    </ol>
    
    
// unordered list (list items are marker prefixed) --
    <ul>
        <li> content </li>
        <li> content </li>
        <li> content </li>
    </ul>

Datalist <datalist>

- creates a list for an <input> element 

    <input list="listId">      // list displayed to the user when clicked
    
    
    <datalist id="listId" >
        <option value="val">
        <option value="val">
        <option value="val">
    </datalist>
Example :
CODE : 
    <input type="text" list="numbers"> 
    <input type="number" list="numbers">
    
    <datalist id="numbers">             // same datalist used for 2 different input types
        <option value="1">
        <option value="2">
        <option value="3">
    </datalist>
