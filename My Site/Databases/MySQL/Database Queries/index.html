Remember This :

- in MySQL there is no statement to rename databases, one way to rename a database is to creating a backup database file then restoring it with a different database name

Description and Demonstration : 

    SHOW DATABASES                                                             // displays all databases on the server (only databases are shown for which the current user has some privileges)   
      \l                                                                         // in PostreSQL
    
    CREATE DATABASE dbName                                                     // creates a new database 
    DROP DATABASE dbName                                                       // deletes the specified database   
    
    USE dbName                                                                 // selects (as current) the specified database   
    SELECT database()                                                          // returns the the currently selected database (NULL if no database selected)    
      SELECT current_database()                                                  // in PostreSQL
DEMO 
// SHOW DATABASES / CREATE DATABASE / DROP DATABASE -----------------------------------
    SHOW DATABASES;                                                                    // displays databases for what the current user has some priviledge (the root user can see all databases)    
    
    CREATE DATABASE testDB;                                                            // creates a new database 
    
    DROP DATABASE someDB;                                                              // deletes the 'someDB' database  
    
// USE / SELECT database() ------------------------------------------------------------
    USE testDB;                                                                        // explicitly selects the 'testDB' database 
    
    SELECT database();                                                                 // returns the currently selected database
