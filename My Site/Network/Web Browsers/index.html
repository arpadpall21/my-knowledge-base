Description and Demonstration :

- a Web Browser is an application which can access the internet, it requests web pages (html documents) form web servers

- when an HTML document is requested the Browser automatically download all linked files to the document (css, js, images, videos, ect...)

- JavaScript running within the browser is Sandboxed, it can control web page functionalities but it has severe limitations on the Client's computer (like it cannot acces local files)
