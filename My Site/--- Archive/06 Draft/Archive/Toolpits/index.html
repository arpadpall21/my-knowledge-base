Some Description :

- Use a HTML container like "div" to create the the actual content and the toolpit

- The content container is displayed inline and relatively positioned because the toolpit container positioned absolute, this must be done because the toolpit container needs to be positioned around the content container

- The toolpit container is "hidden" by default and will be "visible" on hover

- to position the toolpit container

Toolpit with fancy arrow and transition effect

- This very fancy trick is basicaly create and insert with the ::after pseudo-element a small square around the speach bubble and style and shape it!

- Toolpits appear with 1,2,3,4 second delays, by using the transition property
