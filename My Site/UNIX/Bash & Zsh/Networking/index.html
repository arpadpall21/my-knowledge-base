Command Description
hostname [option] - returns the name of this machine
[options]
- (-i) returns the IP address of this machine
ifconfig [options] netAdapter - returns details about all active network interfaces on the host (or the specified one netAdapter)
- run ifconfig | awk '/inet /{print $2}' to get the host IP on MacOs
[options]
- (-a) returns all network devices (including inactive ones)
ifconfig netDev up|down - enables (up) or disables (down) the specified network device
netstat [options] - returns active TCP, UDP and UNIX connections (options can be combined)
- run netstat -anvp tcp | awk 'NR<3 || /LISTEN/' to get all the host's listening TCP connections
[options]
- (-p) process name and PID is returned which uses the connection
- (-n) IP address returned instead of the domain (PTR reccord)
- (-a) returns all connections
- (-l) listening connections only
- (-t) TCP connections only
- (-u) UDP connections only
nmcli [options] - print the network status
[options]
- (device show) complete information about the devices
- (connection show) connection profiles
|
curl [options] IP|URL - requests the IP|URL (supported protocols are HTTP, HTTPS, FTP, SFTP, TFTP, IMAP, POP3, SCP, SMTP, TELNET, LDAP, FILE)
[options]
- (--local-port port) sets the source port (Default: automatically picked by the OS)
- (-X METHOD) sets the request method (ex: curl -X DELETE host)
- (-H "httpHeader: value") add this header to the request
- (-i) also includes the HTTP header fields in the response
- (-s) silent download (does not show download status)
- (-o filePath) saves the downloaded data in the specified file
- (-C) resumes downloading (when download stopped)
- (-k) no server certificate validation
- (-v) verbose (also prints request & response headers)
- (--unix-socket socketPath http://localhostpath) domain socket request (ex: curl --unix-socket /var/run/nginx.sock http:///localhost/test)
ping [options] IP|URL - test the connectivity of the specified server (does DNS resolution)
[options]
- (-c nr) how many ping packets to send
- (-f) sent packets as soon as possible (floods the network)
- (-s nr) packet size in bytes (Default: 60bytes)
- (-i nr) seconds between sending packets (Default :1s)
- (-w nr) ping duration in seconds
|
host [options] domain|IP - (domain) performs a DNS lookup (returns the A and AAAA records)
- (IP) performs a reverse lookup (returns the PTR (pointer) records)
[options]
- (-v)
dig domain [options] [options_2] - performs a DNS lookup, returns the A dns record
[options]
- (ANY) returns all dns records
- (dnsRecord) returns the specified dns record (ex: AAAA, MX)
[options_2]
- (+short) short (IP only)
dig -x IP - performs a reverse lookup (returns the PTR (pointer) records)
|
nmap [options] domain|IP - checks for open ports on the specified domain or IP address (performs DNS lookup)
[options]
- (-O) also returns the target's OS
- (-v) returns a more detailed data
nmap -sP networkId/mask - scan devices on the specified network (ex: nmap -sp 192.168.0.0/16)
tcpdump [options] - intercepts packages on all network devices "package sniffer"
[options]
- ([-D]) displays available network devices (and their status)
- ([i netwDev]) intercepts packages on the specified network device
- (-c nr) specifies how many packages to intercept
- (-A) package content interpreted a ASCII
- (-XX) package content interpreted a ASCII and hexadecimal
- (port portNr) interception on the specified port only
- (dst Ip) interception on the specified destination
traceroute IP|URL - traces the netwrok routes to the specified address
