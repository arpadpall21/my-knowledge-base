File System

Command Description
pwd - prints the absolute path of the current working directory
cd dirPath - navigates to the specified directory
ls [options] dirPath - list items in the specified directory
[options]
- (-l) detailed list (permissions, ownership, size, etc..)
- (-a) include hidden items
- (-h) list file sizes in human readable format (G, M, k)
- (-i) include item Inode numbers
- (-r) reverse list order
- (-S) sort by file size
tree [options] - (no option) displays the structure of the specified directory in a tree-like format
[options]
- (-L nr) level depth to display
- (-a) all items (including hidden onces)
- (-d) display directories only
- (-f) display the path for all items
- (-p) display the item permissions
- (-u) display the item owner
- (-g) display the item group
- (-s) display the item size in bytes
- (-h) display the item size in human readable format (G, M, k)
- (--inodes) display the item inodes
|
cat [options] filePath ... - read the specified file(s) content and output the read data
[options]
- (-n) number output lines
- (-E) highlight output line ends
- (-s) collapses multiple empty lines into a single one
- (>) creates a file (if not exist) and writes the specified content to it (overwrites existing files contents) (cat srcFile > destFile = copy source file content and overwrite target file content)
- (>>) creates a file (if not exist) and appends the specified content to it (appends data to existing files contents) (cat srcFile >> destFile = copy source file content and append it to the target file content)
diff | colordiff [options] file1 file2|dir1 dir2 - prints the differenc between the files (colordiff is an installable command)
[options]
- (-i) case-insensitive check
- (-s) does report when files are identical
- (-q) quick report instead of suggesting modifications
- (-r) recursively compares directories with all their child items
- (-y) side by side view (suggested modifications are displayed in the middle)
head [options] filePath ... - returns the first 10 lines of the specified file(s)
[options]
- (-n nr) specifies the number of lines to display
tail [options] filePath ... - returns the last 10 lines of the specified file(s)
[options]
- (-f) watches the file for changes
- (-n nr) specifies the number of lines to display
stat path path ... - prints the stats of the specified file(s) or directory(ies)
|
chown [options] ownerName:groupName path - (ownerName) changes the owner of the specified file or directory
[options]
- (:groupName) changes the group ownership of the file (can be written as chown :newGroup file)
- (-R) changes the ownership of the targeted directory and all its descendent items
chmod [options] NrNrNr path - changes the permissions of the specified file or direcotry
[options]
- (NrNrNr) (0 = ---) (1 = --x) (2 = -w-) (3 = -wx) (4 = r--) (5 = r-x) (6 = rw-) (7 = rwx)
- (-R) changes the permissions of the targeted directory and all its descendent items
|
touch file file ... - creates empty file(s)
rm [options] path ... - removes the specified file(s) (cannot remove directories)
[options]
- (-i) prompts (warns) before removing items
- (-f) (force) removes write protected items
- (-r|-R) removes the specified directory and all its items
mkdir dirPath ... - creates a new directory
rmdir dirPath - removes an empty directory
cp [options] srcPath scrPath ... destPath - copies (and renames) file(s) (cannot copy directories)
[options]
- (-i) prompts (warns) before overwriting files at destination
- (-f) (force) overwrite protected files at destination
- (-p) preserves the file's stats (ownership, permission bits, times, etc...)
- (-u) copy only if file(s) at destination are older or missing
- (-l) creates hard-links instead of copying files or directories (directory hard-links are not allowed)
- (-s) creates soft-links (symbolic-links) instead of copying files (cannot create directory symbolic-links) (you must be in the directory where you want to create the symbolic-link)
- (-r|-R) copies the specified directory and all its items
mv [options] srcPath scrPath ... destPath - moves (and renames) file(s) or directory(ies) (directories are moved with all their items)
[options]
- (-i) prompts (warns) before overwriting files at destination
- (-f) (force) overwrites files at destination
- (-n) does not overwrite existing file(s) at destination
- (-u) move only if file(s) at destination are older or missing
- (-b) keeps the original file(s) at destination as backup (with ~ extension)
ln [options] srcPath destPath - creates a hard-link at destination (created link can be named) (remember there are no directory hard-links)
[options]
- (-i) prompt before replacing links at destination
- (-f) replaces existing links at destination
- (-s) create soft-link (file or directory symbolic-link) instead hard link (you must be in the directory where you want to create the symbolic-link)
|
df [options] - displays the available space on all mounted file systems
[options]
- (-h) list sizes in human readable format (G, M, k)
- (-i) display inode informations instead of block usage
- (-a) display all devices (including zero block sizes)
du [options] path ... - returns the estimated space of the current (or specified file or directory)
[options]
- (-h) human readable fromat (K, MB, etc..)
- (-s) returns the one summerized size only
|
find dirPath ... -name fileOrDirname [options] - searches for the specified file or direcotry in dirPath
[options]
- (-maxdepth nr) directory depth to search
- (-name 'name') by item name
- (-iname 'name') by item name (case-insensitive)
- (-type f|d|l) by item type (f = file) (d = directory) (l = symbolic link) (s = unix domain socket) (by Default searches for all item types)
- (-inum (+|-)nr) by Inode number (+nr = greater or equal number) (-nr = lower or equal number)
- (-empty) by empty files
- (-size (+|-)nrG|M|k) by file size (+nr = greater or equal number) (-nr = lower or equal number)
- (-user 'name') by user
- (-group 'group') by group
- (-perm NrNrNr) by permission
- (-newer 'name') newer than the specified item
- (-mmin (+|-)nr) by last modified time (in minutes) (+nr = greater or equal number) (-nr = lower or equal number)
- (-mtime (+|-)nr) by last modified time (in days) (+nr = greater or equal number) (-nr = lower or equal number)
- (-amin (+|-)nr) by last access time (in minutes) (+nr = greater or equal number) (-nr = lower or equal number)
- (-atime (+|-)nr) by last access time (in days) (+nr = greater or equal number) (-nr = lower or equal number)
- (-cmin (+|-)nr) by last change time (in minutes) (+nr = greater or equal number) (-nr = lower or equal number)
- (-ctime (+|-)nr) by last change time (in days) (+nr = greater or equal number) (-nr = lower or equal number)
- (-not) returns anyting else than the specified criteria
- (! criteria) removes the specified items (criteria) from the already returned result
- (-name -or -name) groups multiple name criterias
- (-exec cmd {} +) executes the specified command on the returned result(s)
grep [options] pattern filePath ... - searches for a pattern in stdin, returns each line where there's a match
[options]
- (-i) case-insensitive search
- (-n) print line numb with output lines
- (-B int) print nr of lines before the match
- (-A int) print nr of lines after the match
- (-v) print everything other than the matched result
|
umask [options] umaskVal - returns the umask value for the user (Default: umask = 0002 | file permission = 0666 | directory permission = 0777)
- the idea is that for newly created files do allow any higher permissions what is allowed by the umask value [options]
- (-S) the umask value is symbolically printed
- umaskVal sets the umask value for the current session (umask value resets to 0002)
