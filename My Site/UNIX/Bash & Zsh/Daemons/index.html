Command Description
Ubuntu (systemctl is the successor of service)
cmd & - runs the cmd in the background (as current shell background process/daemon)
jobs - list current shell background processes
fg %jobID - birings the specified current shell background process (jbNr) in the foreground
bg %jobID - sends the specified process (jbNr) in the background (as background process in the current shell)
kill %jobID - stops the specified current shell background process (jbNr)
|
nohup cmd [options] & - run the cmd in the background and ingore hangub signal (bg process is detached from the current shell, runs after closing the terminal)
[options]
- (> filePath 2>&1) - cmd output is stored in this file (Default: nohup.out in the current directory)
|
service service start - starts the specified service
service service stop - stops the specified service
service service reload - reloads the system (reloads configurations)
service --status-all - prints all services (+ = service running | - = service not running)
service service status - prints the status of the specified service
|
systemctl start service - starts the specified service
systemctl stop service - stops the specified service
systemctl reload service - reloads the service's configuration file (without restarting the service if possible)
systemctl restart service|all - restart the specified service or all running services
systemctl enable|disable service - enables or disables the daemon when the system starts
systemctl is-enabled service - checks whether the service is enabled or disabled
systemctl status serviceName - prints all running services or checks the status of the specified one
|
MacOS brew (uses MacOS launchctl underneath [link])
brew services list - list running services (daemons)
|
brew services start service - starts the specified service and registers it to lauch at login (boot)
brew services run service - starts the specified service without registers it to lauch
brew services stop service - stops the specified service and unregisters it from the lauch at login (boot)
brew services restart service - tops and starts the specified service and registers it to lauch at login (boot)
