Endianness format swapping swap16() / swap32() / swap64() methods 

SYNTAX :    buf.swap16()                                                               // interprets the buffer as array of 16bit values and swaps the endianness format in place (buffer size must be multiple of 2)    
            buf.swap32()                                                               // interprets the buffer as array of 32bit values and swaps the endianness format in place (buffer size must be multiple of 4)    
            buf.swap64()                                                               // interprets the buffer as array of 64bit values and swaps the endianness format in place (buffer size must be multiple of 8)    
    
// swap16() ---------------------------------------------------------------------------
    var buf = Buffer.from([0x00, 0x11, 0x22, 0x33]);
    
    buf.swap16();                                                                      // swaps every 2 bytes in place 
    buf;                                                                               // -> <Buffer 11 00 33 22>
    
// swap32() ---------------------------------------------------------------------------
    var buf = Buffer.from([0x00, 0x11, 0x22, 0x33, 0x44, 0x55, 0x66, 0x77]);
    
    buf.swap32();                                                                      // swaps every 4 bytes in place 
    buf;                                                                               // -> <Buffer 33 22 11 00 77 66 55 44>
    
// swap64() ---------------------------------------------------------------------------
    var buf = Buffer.from([0x00, 0x11, 0x22, 0x33, 0x44, 0x55, 0x66, 0x77, 0x88, 0x99, 0xaa, 0xbb, 0xcc, 0xdd, 0xee, 0xff]);
            
    buf.swap64();                                                                      // swaps every 8 bytes in place 
    buf;                                                                               // -> <Buffer 77 66 55 44 33 22 11 00 ff ee dd cc bb aa 99 88>
