Description and Demonstration :

- Node.js allows to execute JavaScript code outside of a web browser, it's built on the V8 JavaScript engine and is designed for building scalable network applications, particularly web servers

- Node.js uses an event-driven, non-blocking I/O model, making it efficient and suitable for real-time application 

    $ node [options] [V8options] [nodeFile] [arguments, ...]      // CLI syntax (no argument node) launches REPL

Node.js program structure

- Node.js do have a global scope, but it's not designed to run tasks, instead we use modules (which are basically .js files)

- the global scope is accessible from any through the global object

- modules can export / import each others assets

DEMO 
// main.js (file) ------------------------------------------------------------------
    require ('./module1');
    
    global.commonObj;         // -> { p1: 'some property' }
    
// ----------------------------------------------------------------
// module1.js (file) ------------------------------------------------------------------
    global.commonObj = { p1: 'some property' }      // declaring variable on the global scope
