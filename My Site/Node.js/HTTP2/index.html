HTTP/2 module (Node.js)

Remember This :

- http2 pseudo-headers are prefixed with a colon (request :method :scheme :path :authority) (response :status)

- by default the http2Srv and htt2SecureSrv do not time out their connections (timeout value is set to 0)

- when timeout value set and there is NO timeout listener registered on the server object the connection is dropped

- only the server returned by the createSecureServer() method can serve both HTTP/2 and HTTP/1.* clients (over TLS connection), the none TLS version can only serve HTTP/2 clients (createServer())

- browsers do not accept HTTP/2 protocol without TLS, so serving browsers with http2.createServer() servers is not possible

Description and Demonstration :

- the http2 module provides a low-level API which is not designed to be compatible with the http protocol

- designed to be symetric between the client and the server ((almost) the same events and methods can be used on both sides)
