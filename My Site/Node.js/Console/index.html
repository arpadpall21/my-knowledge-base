Description and Demonstration :

- provides a simple debugging console

- (I thing for complience reasons) provides the same methods as the browser console, but because it supports less features some of them references each other 

  const count = 5;
  console.log('count: %d', count);          // -> 'count: 5'
  
  
  console.table([['a', 'b'], [1, 2]])
  // output:
  // ┌─────────┬─────┬─────┐
  // │ (index) │ 0   │ 1   │
  // ├─────────┼─────┼─────┤
  // │ 0       │ 'a' │ 'b' │
  // │ 1       │ 1   │ 2   │
  // └─────────┴─────┴─────┘
