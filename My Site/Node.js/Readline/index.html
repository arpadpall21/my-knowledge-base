Description and Demonstration :

- the readline library provides an interface for reading data from a readable stream one line at a time

- however readline provides powerfull terminal featrues (like moving cursor, etc..), so it's main use case is to create interactive terminal tools My Questionnaire Cli 

    import readline from 'node:readline';
    
    const rl = readline.createInterface({ input: process.stdin, output: process.stdout });
    
    rl.question('What do you thing about Node.js?\n', (answer) => {
      rl.write(`Your opinion: ${answer}`)
      rl.close();
    });
