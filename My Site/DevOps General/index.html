Types of IAC (Infrastructure As Code) Tools

- provision tool: deploys (provisions) infrastructure resources

- server templating tool: creates and manages virtualized containers that contains all necessary tools in order to run applications and services in an isolated environment

- configuration management tool: installs and manages software on existing infrastructure resources
