gen Cache-Control: directive, ...

- multiple directives are separated with commans, some directive may have argument which can be a token or quoted string

Request

max-age=nr:sec - time the resource is considered 'fresh' (relative to the request time)

max-stale=nr:sec - indicates that the client will accept a stale response (optionally upper limit of staleness the client will accept)

min-fresh=nr:sec - the client wants a response that is 'fresh' (indicated by the seconds)

no-cache - may be stored by any cache (MUST be validated by the origin server first)

no-store - response shouldn't be cached

no-transform - an intermediate cache or proxy cannot edit the response body

only-if-cached - respond only using a stored response or respond with a 504 status code

Response

must-revalidate - once the response becomes 'stale', it must be validated on the origin server

no-cache - may be stored by any cache (MUST be validated by the origin server first)

no-transform - an intermediate cache or proxy cannot edit the response body

public - can be stored by any cache (even if it's not cachable)

private - storable only by the browser's cache

proxy-revalidate - once the response becomes 'stale', it must be validated on the proxy server

max-age=nr:sec - time the resource is considered 'fresh' (relative to the request time)

s-maxage=nr:sec - overrides max-age value and Expires header only for shared caches (proxy) ignored by private caches

Warning Codes

110 Response is Stale - A response provided by a cache is stale (the expiration time set for it has passed)

111 Revalidation Failed - attempt to validate the response failed, due to an inability to reach the server

112 Disconnected Operation - The cache is disconnected from the rest of the network

113 Heuristic Expiration - Sent If a cache heuristically chose a freshness lifetime greater than 24 hours and the response's age is greater than 24 hours

199 Miscellaneous Warning - Arbitrary, non-specific warning

214 Transformation Applied - Added by a proxy if it applies any transformation to the representation, such as changing the content-coding, media-type or the like

299 Miscellaneous Persistent Warning - Same as 199, but indicating a persistent warning
