Description and Demonstration : 

        httpVer statusCode reasonPhrase [\r|\n]                                        // HTTP/1.* status line (HTTP/2 uses pseudo-headers to describe the status line (ex: :status: 200)
        fieldName : fieldVal [\r|\n] ...                                               // header field(s) (each line ends with a new line character)   
            [\r|\n]                                                                    // empty line (nothing preceding the new line character) separates the message header and body (required)    
        data...                                                                        // message body conatins the acutal data
DEMO
