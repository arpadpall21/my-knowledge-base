Effect Description
jQ_obj.animate({property:"value", property:"value", ...}, speed, "easing", function) animates an element of a set of CSS properties (cannot animate string values [need jQuery UI])
jQ_obj.clearQueue("queueName") stops the all further animation but finishes the current state (can remove any generig jQuery queue items)
jQ_obj.delay(speed, "queueName") sets a delay in a jQuery queue
jQ_obj.dequeue("queueName") must be used after the queue() method in order to fire the next state in the jQuery queue
jQ_obj.fadeIn(speed, "easing", function) animates the oppacity of the selected element from 0.00 to 1.00 (works on elements fadeOut with jQuery)
jQ_obj.fadeOut(speed, "easing", function) animates the oppacity level of the selected element to 0.00 than set the element to display:none;
jQ_obj.fadeTo(speed, oppacity_NR, "easing", function) animates the element's oppacity level to the set value (cannot fade back!)
jQ_obj.fadeToggle(speed, "easing", function) toggles between fadeIn() and fadeOut()
jQ_obj.finish("queueName") stops the animation and jumps at the of the queue
jQ_obj.hide(speed, "easing", function) animates the size of the selected element than set display:none;
jQ_obj.queue(function)
jQ_obj.queue("queueName")		 - executes a function in the queue
- gets information about a queue
jQ_obj.show(speed, "easing", function) animates the size of the selected element and shows the hidden element (works only on elements hidden by jQuery)
jQ_obj.slideDown(speed, "easing", function) shows a jQuery hidden element by "sliding down" (animates the height)
jQ_obj.slideToggle(speed, "easing", function) toggles between slideDown() and slideUp()
jQ_obj.slideUp(speed, "easing", function) set the element to display:none by "sliding up" (animates the height)
jQ_obj.stop(stopAllAnim_boolean, endCurrentState_boolean) can stop any jQuery effects
jQ_obj.toggle(speed, "easing", function) jQ_obj.toggle(show_boolean) toggles between hide() and show() methods (can permanently show / hide element)

hide() / show() / toggle() elements

- the hide() method animates the size of the selected element than set display:none;

- the show() method animates the size of the selected element and shows the hidden element (works only on elements hidden by jQuery)

- the toggle() method toggles between hide() and show() methods (can permanently show / hide element)

SYNTAX :    jQ_obj.hide(speed, "easing", function) 
            jQ_obj.show(speed, "easing", function)
            
            jQ_obj.toggle(speed, "easing", function)
            jQ_obj.toggle(show_boolean)
                - speed : specifies the speed of the effect, possible values : milliseconds or slow/fast (default 400ms)
                - "easing" : specifies the easing possible values : swing(default)/linear (more effects are availables with plugins)
                - function : function to be executed after the method is completed
                - show_boolean : permanently show/hides the element 
    
    <p id="click_hide"> Click to Hide and Seek </p>
<script>
    $(document).ready(function(){
        $("#click_hide").click(function(){
            $(this).hide(2500, "linear").show("fast", "swing");             // when you click on the element it will hide and show 
        });    
    // toggle show/hide --------------------    
        $("#click_hide").click(function(){
            $(this).toggle(slow, "swing");                                  // show/hide element on each click 
        });
    // permanently show/hide --------------------        
        $("#click_hide").click(function(){
            $(this).toggle(false);                                          // hides the element permanently () 
        });
    });
</script>

fade elements fadeIn() / fadeOut() / fadeToggle() / fadeTo()

- the fadeOut() method animates the oppacity level of the selected element to 0.00 than set the element to display:none;

- the fadeIn() method animates the oppacity of the selected element from 0.00 to 1.00 (works on elements fadeOut with jQuery)

- the fadeToggle() method toggles between fadeIn() and fadeOut()

- the fadeTo() method animates the element's oppacity level to the set value (cannot fade back!) 

SYNTAX :    jQ_obj.fadeOut(speed, "easing", function) 
            jQ_obj.fadeIn(speed, "easing", function)
            jQ_obj.fadeToggle(speed, "easing", function)
            jQ_obj.fadeTo(speed, oppacity_NR, "easing", function)
                - speed/speed : specifies the speed of the effect, possible values : milliseconds or slow/fast (default 400ms)
                - "easing" : specifies the easing possible values : swing(default)/linear (more effects are availables with plugins)
                - function : function to be executed after the method is completed 
                - oppacity_NR : specifies the oppacity to fade to (from 1.00 to 0.00)

slide elements slideUp() / slideDown() / slideToggle()

- the slideUp() method set the element to display:none by "sliding up" (animates the height)

- the slideDown() method shows a jQuery hidden element by "sliding down" (animates the height)

- the slideToggle() method toggles between slideDown() and slideUp() 

SYNTAX :    jQ_obj.slideUp(speed, "easing", function)
            jQ_obj.slideDown(speed, "easing", function)
            jQ_obj.slideToggle(speed, "easing", function)
                - speed : specifies the speed of the effect, possible values : milliseconds or slow/fast (default 400ms)
                - "easing" : specifies the easing possible values : swing(default)/linear (more effects are availables with plugins)
                - function : function to be executed after the method is completed

    <p id="demo"> Click to Slide </p>
<script>

        $("#demo").click(function(){                                        // on click element will slide up than down and changes its content
            $(this).slideUp(1500, "linear", function(){                     // fires on slide up
                $(this).text(" Slided Up ");
            }).slideDown(1500, "linear", function(){                        // fires on slide down
                $(this).text(" Slided Down ")    
            });    
        }); 
    // slide toggle ------------
        $("#demo_click").click(function(){
            $("#demo").slideToggle(1500, "linear", function(){              // slide toggle animation (same as above)
                $(this).text(" Slide Toggle active ");                      // after slide effect is finished element's content will change
            });    
        });
    });
</script>

jQuery animation animate() / stop() / clearQueue() / finish() / queue() / dequeue()

- element must be set to position:relative/absolute/fixed; in order to animate in jQuery

- the animate() method animates an element of a set of CSS properties (cannot animate string values [need jQuery UI])

- the stop() method can stop any jQuery effects

- the clearQueue() stops the all further animation but finishes the current state (can remove any generig jQuery queue items)

- the finish() method stops the animation and jumps at the of the queue

- the queue() method executes a function in the que or gets information about the queue

- the dequeue() must be used after the queue() method in order to fire the next state in the jQuery queue 

SYNTAX :        jQ_obj.animate({property:"value", property:"value", ...}, speed, "easing", function)
                    - {property:"value", property:"value", ...} : properties to animate
                    - speed : specifies the speed of the effect, possible values : milliseconds or slow/fast (default 400ms)
                    - "easing" : specifies the easing possible values : swing(default)/linear (more effects are availables with plugins)
                    - function : function to be executed after the method is completed 
                    
                jQ_obj.stop(stopAllAnim_boolean, endCurrentState_boolean)
                    - stopAllAnim_boolean : stop or not all animation of the element (default : false)
                    - endCurrentState_boolean : jump or not at the end of current animation state (default : false)
             
                jQ_obj.clearQueue("queueName")
                jQ_obj.finish("queueName")
                    - "queName" : specifies the name of the queue (default is "fx")
                
                jQ_obj.queue("queueName")                                   // gets infromation
                jQ_obj.queue(function)                                      // sets information 
                jQ_obj.dequeue("queueName")
                    - "queName" : specifies the name of the queue (default is "fx")
                    - function : function to be executed 
    
    <p id="demo" style="position:relative; display:inline;"> Animation (click)</p><br>
    <button id="click_demo"> Stop </button>
    <span id="demo_state"> Number of States = </span>
<script>
    $(document).ready(function(){
        var box = $("#demo");                                               // load an element in a variable with jQuery (only jQuery method can use it!)
        // box.style.color = "red";                                         // this DOES NOT WORK!
        box.click(function(){
            box.animate({left:"100px"}, 1500, "swing");                     // animation queue  
            box.animate({left:"50px"},1000, "swing");
            box.animate({left:"1000px"}, 800);
            box.queue(function(){                                           // add an extra function in the queue
                box.css("color", "white");
                box.dequeue();                                              // exits from "queue()" and fire the next queue state
            });
            box.delay(1000);
            box.animate({left:"0"}, 2000, "linear");
            $("#demo_state").text(" Number of States = " +         
                                  box.queue().length);                      // reads the que length
        });
        $("#click_demo").click(function(){
            
// stop() -------------------------------------------------------
            box.stop();   // (this is "box.stop(false, false))              // stops the current que state of the animation but continues from the next one, 
            box.stop(true);                                                 // stops all further animation
            box.stop(true, true);                                           // stops all further animation but jumps at the end of the current animation state!
            
// clearQueue() -------------------------------------------------
            box.clearQueue();                                               // stops all further animation but fishes the current animation state

// finish() -----------------------------------------------------
            box.finish();                                                   // ends all animation and jumps at the end of the last animation state
        });
    });
</script>

CODE : 
    <p id="count_queue"> Queue length = </p>
    <button id="anim_start"> Start Animation </button><br>
    <button id="end_animation"> End and Complete Animation </button><br>
    <button id="stop_A"> Skip Curent State </button>
    <button id="stop_B"> Pause Animation </button>
    <button id="stop_C"> Stop Animation (jump at the end of the current state) </button><br>
    <button id="clear_queue"> Clear Queue (finish current state and clear the queue) </button>
    <div style="width:35px; height:35px; background-color:yellow; position:relative; left"></div>
<script>
    $(document).ready(function(){
        var anim = $("[style^=width]");
        $("#anim_start").click(function(){                                                                          // animation 
            anim.animate({top: "20px", "backgroundColor": "red"}, 1000, "linear"); 
            anim.animate({left: "700"}, 2000, "swing");
            anim.queue(function(){
                anim.css("backgroundColor", "green");
                anim.dequeue();
            }); 
            anim.animate({top: "0", backgroundColor: "white"}, 1000, "linear");
            anim.animate({left:"0", backgroundColor:"yellow", width:"+=20px", height:"+=20px"}, 2000, "swing");     // box gets bigger every time the animation is completed
            $("#count_queue").text( " Queue length = " + anim.queue().length);                                      // read information about the queue
        });
        $("#end_animation").click(function(){
            anim.finish();
        });
        $("#stop_A").click(function(){                                                                              // skips the current animation running
            anim.stop();                    // this only stops the current animation state (but next state will fire)
        });
        $("#stop_B").click(function(){                                                                              // stops the animation 
            anim.stop(true);                // this stops all element animations
        });
        $("#stop_C").click(function(){                                                                              // stop the animation at the end of the current state
            anim.stop(true, true);          // second argument specifies if the current animation state should be finished 
        });
        $("#clear_queue").click(function(){
            anim.clearQueue();    
        });
    });        
</script>
