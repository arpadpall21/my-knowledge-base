Remember This :

2) CDN's make sure that you download jQuery from the server closest to you

- nest all you jQuery code in $(document).ready(function(){ }) or $(function(){ }) (shorter version) code, this makes sure that your document is fully loaded before the nested code fires

- always nest JavaScript code in jQuery callback functions to avoid bugs!

- relative values are allowed in jQuery (ex: width: "+=10px";)

Description and Demonstration :

- jQuery is a lightweight "write less do more" JavaScript library (frameworks) which greatly simplifies some of JavaScript tasks

- with jQuery you can select HTML elements and perform "actions" on them

- jQuery wraps arround HTML elements with a jQuery object, therefore jQuery methods will be available on them 

SYNTAX :    $(jQuery_selector, context).method()    
    
    $(document).ready(function(){               // this makes sure that the document is loaded before the nested jQuery code is fired
        $("button").click(function{            
            $("p").hide();
        });
    });
//-----------    
    $(function(){                               // does the same as the code above (shorter version)
        $("button").click(function(){            
            $("p").hide();
        });
    });

- there are 2 versions of jQuery

Production Version (minified and compressed) lightweight version for production use

Development Version (uncompressed and readable) for testing and development

- you can download directly jQuery files form jQuery website

jQuery object

- a jQuery object is an Array like object containing DOM elements

- the beauty of jQuery objects is that once DOM elements are selected jQuery methods are available on them! 

SYNTAX :    $(jQuery_selector)                          // returns a jQuery object
            jQ_obj.get("index")                         // returns a DOM element from a jQuery object
            jQ_obj[index]                               // this does the same
                - index : refers to the targeted element 
    
    $()                                                 // anyting in a jQuery selctor is a jQuery object (can be empty)
    $("#demo")                                          // the "id="demo" element is wrapped in a jQuery object
    $("p")                                              // all paragraphs in the page are wrapped by a jQuery object   
    
// ---------------------------- create jQuery object
    $("<p>demo</p>")                                    // this is how easily you can create an element with jQuery

// ---------------------------- return DOM element with jQuery 
    <p class="style_me"> Paragraph </p>
    <p class="style_me"> Paragraph </p>
<script>
    $(document).ready(function(){
        var jQuery_obj = $(".style_me");                // returns a jQuery object which contains both paragraphs
        var DOM_obj_A = $(".style_me").get(0);          // returns the first paragraph as DOM element 
        var DOM_obj_B = $(".style_me")[1];              // returns the second paragraph as DOM element
    // ---
        jQuery_obj.css("color", "blue");                // jQuery method available but not native DOM methods!
        DOM_obj_A.style.fontWeight = "bold";            // native DOM method available but not jQuery methods!  
        DOM_obj_B.style.textDecoration = "underline";   // native DOM method available but not jQuery methods!   
    });
</script>

Using callback functions in jQuery

- any JS code can be fired in jQuery but they can fire eralier than jQuery code this can cause bugs!

- it is highliy recommended to nest JS code in jQuery callback functions 

    $(document).ready(function(){
        <code to be executed>         // this code is garanteed to fire after the document is loaded
    }
        <code to be executed>         // this code is not garanteed to fire because it is not nested in the callback function
    );

Chaining in jQuery

- it is possible to chain multiple jQuery methods, these methods will be executed in the chained order 

    <p id="fade_IN"> I'm Faded :P </p>
<script>
    $(document).ready(function(){
        $("#fade_IN").css("color", "white").fadeOut(200).fadeIn(1500);              // chained methods
        
        $("#fade_IN")                                                               // this does the same jQuery has no strict Syntax!
        .css("color", "white")
        .fadeOut(200)
        .fadeIn(1500);
    });
</script>

jQuery Plugins (custom methods)

- a jQuery plugin is basically a custom method which once created will be available on the jQuery object 

SYNTAX :    (function($){
                $.fn.plugin_name = function(){
                    <plugin CODE here>
                };
            })(jQuery);        
    
    $(document).ready(function(){
// basic plugin concepts
        (function($){                               // closure makes sure that the "$" identifier can be used even if "noConflict" is declared
            $.fn.myPlugin_A = function(){           // colors the element(s) on which is called
                this.css("color", "red");
                return this;                        // this returns the element itself (maintains chainability)
        };
        })(jQuery); 
        
// advanced plugin concept BEST PRACTICE USE THIS!!!
        (function($){
            $.fn.myPlugin_B = function(){
                return this.each(function(){        // using the each method makes possible to return every element's values and the element is automatically returned this way
                    $(this).css("color", "red");    // colors the element(s) on which is called   
                                                    // elements are passed as DOM elements!   
                    $(this).append("(style= " + $(this).attr("style") + ")");   // prints the style attribute's value at the end of the element
                });
            };
        })(jQuery);
        
// passing plugin arguments 
        (function($){
            $.fn.setColor = function(color){        // argument can be passed in order to define the color
                return this.each(function(){
                    $(this).css("color", color);    // custom color can be defined
                });
            };
        })(jQuery);
    // options
        (function($){
            $.fn.greenDefault = function(option){   // this plugin has a defalut values set but it can be modified by passing options
                var setting = $.extend({}, {color:"white", backgroundColor:"darkgreen"}, option);
                                                    // the second argument contains the default settings
                                                    // the third argument is the object which can be set to overwrite default settings
                this.css({color:setting.color, backgroundColor:setting.backgroundColor});
                                                    // execute Plugin (this way property values can be overwritten)
                return this;                        // Don't forget about returning the element itself for maintaining chainability
            };
        })(jQuery);
    });

<script>
    $(document).ready(function(){
        (function($){
            $.fn.getColor = function(){
                // return $(this).css("color");                 // problem with this is that it returns ONLY THE FIRST element's color 
                // return this;
            // get information and print it
                this.each(function(){                           // runs a function for each elements 
                    $("#getColors").append($(this).css("color") + " / ");
                });
            // get information and collect in an object
                var collectCollors = "";                        // internally created JS object
                this.each(function(){
                    collectCollors += $(this).css("color") + " / ";
                });
                $("#getColors_2").append(collectCollors);
            // set 
                return this.each(function(){
                    $(this).css("textDecoration", "underline");
                });
            };
        })(jQuery);
        $("#myColors p").getColor();                            // call on targeted elements
    });
</script>

CODE : 
    <div id="pluginOpt">
        <p> Original Plugin </p>
        <p> Modified A </p>
        <p> Modified B </p>
    </div>
<script>
    jQuery(document).ready(function(){
        $("#pluginOpt p").css("margin", 0);
        (function($){
            $.fn.styleMe = function(option){
                var settings = $.extend(true, {color:"white", backgroundColor:"black"}, option);    // plugin's default settings
                                                                                                    // "option" argument passed can overwrite the "settings" object's properties
                return this.css({color:settings.color, backgroundColor:settings.backgroundColor});  
            };
        })(jQuery);   
        
        $("#pluginOpt p:first").styleMe();                                          // plugin called with default settings
        $("#pluginOpt p:eq(1)").styleMe({color:"red", backgroundColor:"white"});    // plugin called with modified settings
        $("#pluginOpt p:last").styleMe({color:"yellow"});                           // plugin called with partially modifications
    });
</script>
