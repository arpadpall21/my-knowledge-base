VS Code (with my key binds)
dev features
alt + up|down - move current line up|down
shift + alt + up|down - copy line up|down (custom)
ctrl + d - select current word (then select same words that is previously selected)
alt + click - add to current selection
ctrl + k then ctrl + [ - collapse all foldable elements
ctrl + k then ctrl + ] - uncollapse all foldable elements
ctrl + l - select current line
ctrl + / - comment|uncomment line
shift + up | down | left | right - square style selection with keys (custom)
ctrl + shift + left | right - jump back|forward
ctrl + shift + down - go to references of the currently selected variable (custom)
ctrl + shift + up - go ot definition of the currently selected variable (custom)
cmd + [ | ] - push the current line left|right
IDE navigation
ctrl + tab - next editor tab (custom)
ctrl + shift + tab - previous editor tab (custom)
ctrl + alt + K - add|remove bookmark (Bookmarks plugin)
ctrl + alt + l - jump on next bookmark (Bookmarks plugin)
ctrl + alt + j - jump on previous bookmark (Bookmarks plugin)
ctrl + ` - focus terminal
ctrl + 0 - focus Explorer
ctrl + 1|... - focus Explorer tab
ctrl + p
cmd + p (Mac) 		- search for files and directories
cmd + f (Mac) - search in current file
ctrl + shift + u
shift + cmd + f (Mac) 		- search in all the project files
ctrl + shift + h
cmd + shift + h (Mac) 		- search & replace (searches in all the project files and replaces the found results in all files)
ctrl + b - hide|show left side tool bar (custom)
cmd + shift + z - toggle zen mode (custom)
Google Chrome (web browser)
ctrl + l focus address bar
ctrl + t new tab
ctrl + shift + t reopens the last closed tab (remember 10 lastly closed tabs)
ctrl + F4 | ctrl + W close current tab
ctrl + tab | ctrl + pgdown next tab
ctrl + shift + tab | ctrl + pgup previous tab
ctrl + shift + pgdown moves the current tab right
ctrl + shift + pgup moves the current tab left
ctrl + (nr) switches to the (nr) tab (ctrl + 9 = last tab)
alt + left | rigth navigate back|forwar in browser history
enter open focused item in the same tab
ctrl + enter open focused item in new tab
shift + enter open focused item in browser
ctrl + l clear console window
shift + cmd + r (mac) - clear browser cache
cmd + p (mac) - look for recource files (in dev window)
Windows
win + d show desktop
win + tab open task view (show all desctops)
Linux Ubuntu
alt + F10 fullscreen window (toggle)
ctrl + alt + t open terminal (reset custom Key Shortcut to gnome-terminal --geometry 82x42-0+0 in order to set the terminal's custom size and position)
ctrl + alt + (up|downArrow) switch desktop
ctrl + alt + [F1-F6] Switch to nr Terminal
ctrl + alt + F7 Exit from Virtual Console
alt + SysRq(prr sc) > r > e > i > s > u > b

r = switch keybord from raw mode to XLATE mode

e = send SIGTERM signall to all processes except init

i = sned SIGKILL signall to all processes except init

s = sync all mounted file systems

u = remount all mounted file systems in read-only mode

b = immediately reboot the system, without unmounting partitions or syncing

Gracefully restarts the system (5 seconds interval between r > e > i > s > u > b buttons)
alt + SysRq(prr sc) > r > e > i > s > u > o

r = switch keybord from raw mode to XLATE mode

e = send SIGTERM signall to all processes except init

i = sned SIGKILL signall to all processes except init

s = sync all mounted file systems

u = remount all mounted file systems in read-only mode

o = immediately shut down the system

Gracefully shuts down the system (5 seconds interval between r > e > i > s > u > b buttons)
alt + ` Toggle between instances of the same running program
ctrl + h show|hide hidden files (file explorer)
alt + (up|downArrow) navigate up|down on the file tree (file explorer)
super + h - minimize current window
Mac
fn + F11 - show desktop (toggle -> back)

cmd + shift + 4 - open screenshot cutter (the selected part will be saved)

ctrl + m - minimize current window
shift + cmd + . - show|hide hidden files (in a file browser window)
