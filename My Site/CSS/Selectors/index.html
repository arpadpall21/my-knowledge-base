Simple Selectors

- id and class identifier must start with a letter, it can contain a-zA-Z(letters) 0-9(digits) _(undersocre) -(hyphen), (in HTML5 identifiers can use ANY characters except white space and cannot start with a digit)

DEMO: 
// HTML ---------------------------------
    <p id="myParagraph"> demo paragrap</p>
    <div class="myElement">my div element</div>
    <div class="myElement">my div element</div>
    
// CSS ----------------------------------
    p {color: red;}                                            // element selector      // selects : <p id="myParagraph">demo paragrap</p>
    #myParagraph {color: red;}                                 // id selector           // selects : <p id="myParagraph">demo paragrap</p>
    .myElement {color: red;}                                   // class selector        // selects : <div class="myElement">my div element</div>  and  <div class="myElement">my div element</div>
    * {color: red;}                                            // selects all elements

Combinators

- combinators define the relationship between selectors

DEMO: 
// HTML ---------------------------------
    <div class="myDiv">
        <p> demo <span>child</span> </p>
    </div>
    <p>paragraph 1</p>
    <p>paragraph 2</p>
    
// CSS ----------------------------------
    .myDiv, p { border-bottom: solid 1px red; }                // selects the div element (by the class attribute) and all paragraphs 
    
    .myDiv * { border-bottom: solid 1px red; }                 // selects all elements inside the div element 
    .myDiv > * { border-bottom: solid 1px red; }               // only selects the first level child element inside the div element (the "<span>child</span>" element is not selected)   
    
    .myDiv ~ p { border-bottom: solid 1px red; }               // selects all paragraph after the div element 
    .myDiv + p { border-bottom: solid 1px red; }               // selects the one paragraph which comes directly after the div element (<p>paragraph 1</p>)

Attribute Selectors

- attribute selectors target HTML elements by their attributes

DEMO: 
// HTML ---------------------------------
    <p title="myTitle"> paragraph 1 </p> 
    <p title="mytitle"> paragraph 2 </p> 
    <p title="mytitle yourTitle"> paragraph 3 </p> 
    <p title="mytitle-yourTitle."> paragraph 4 </p> 
    
// CSS ----------------------------------
    [title] { color:red; }                                     // selects all elements because they all have the "title" attribute 
    [title="myTitle"] { color:red; }                           // selects the first paragrap 
    [title="MYTITLE" i] { color:red; }                         // selects the first two paragraps because the comparison is set to case-insensitive 
    
    [title~="yourTitle"] { color:red; }                        // selects the second and third paragraphs because the attribute value contains the "youTitle" word alone or separated by spaces 
    [title|="mytitle"] { color:red; }                          // selects the second and fourth paragraphs because the attribute value matches the "myTitle" word or start with the it followed by a hyphen(-)   
        
    [title^="my"] { color:red; }                               // selects all elements because all attribute values start with "my"
    [title$="."] { color:red; }                                // selects the fourth paragraph because the attribute value ends with "."
    [title*="-"] { color:red; }                                // selects the fourth paragraph because the attribute value contains the "-" character
