Unit Description
Absolute Units
nrcm - centimeters 1 1 3.5 3 1
nrmm - milimeters 1 1 3.5 3 1
nrin - (1in = 96px = 2.54cm) inches 1 1 3.5 3 1
|
nrpt - (1pt = 1/72th of 1in) points 1 1 3.5 3 1
nrpc - (1pc = 12pt) picas 1 1 3.5 3 1
|
nrpx - (1px = 1/96th of 1in) pixels (absolute pixel size but relative to the screen because of pixel density) 1 1 3.5 3 1
|
Relative Units
nrem - relative to the current font size of this element 1 1 3.5 3 1
nrrem - relative to the current font size of the root (<html>) element 4 3.6 11.6 9 4.1
|
nr% - relative to the same property of the parent element 1 1 3.5 3 1
|
nrvw - relative (percent) to the current width of the viewport 20 19 20 9 6
nrvh - relative (percent) to the current height of the viewport 20 19 20 9 6
nrvmin - relative (percent) to the smallest side (width or height which is currently the smallest) of the current viewport 20 19 20 9 "vm" 6
nrvmax - relative (percent) to the largest side (width or height which is currently the largest) of the current viewport 26 19 20 NO 7
|
nrpx (1px = 1/96th of 1in) - pixels (absolute pixel size but relative to the screen because of pixel density) 1 1 3.5 3 1
|
nrex - relative to the "x-height" of the current font [TESTED 2019-01-14 -> not reliable] 1 1 3.5 3 1
nrch - relative to the width of the 0(zero) character [TESTED 2019-01-14 -> not reliable] 27 1 20 9 7

Description and Demonstration :

- in CSS we have 2 types of length units, absolute (fixed size) and relative (relative to another lenght)

Absolute Sizes cm / mm / in / px / pt / pc

- using absolute units is not recommended on screen devices because the screen size can vary so much

DEMO 
    <div class="demo"></div>
    .demo {
        border: solid 2px black;
        width: 20cm;
        height: 100mm;
    }

Relative Sizes em / rem / ex / ch / vw / vh / vmin / vmax / % / px

DEMO 
// em / rem ---------------
    .myElement {
        font-size: 2em;                                                                // double of the current font-size
        width: 10rem;                                                                  // 10 times bigger then the root element "<html>" font-size
    }
    
// % ----------------------
    .parentElement {
        height: 500px;
    }
    .childElement {
        height: 80%                                                                    // relative to the parent element's same property (500 * 80% = 400px)
    }
    
// vw / vh / vmin / vmax --
    .myElement {
        width: 10vw;                                                                   // 10% of the current viewport width
        height: 10vh;                                                                  // 10% of the current viewport height
        font-size: 2vmin;                                                              // 2% of the smallest side (width or height of the current viewport which is curently the smallest)  
        padding: 3vmax;                                                                // 3% of the largest side (width or height of the current viewport which is curently the largest)  
    }
Example :

Relative values em / rem / % TEST

I'm 2em big (twice as big as the set font size)

I'm 1.5rem big (twice as big as the root element font-size)

I'm 150% big (relative to the font-size of the parent element)

CODE : 
    <p style="font-size:2em;"> I'm <u>2em</u> big (twice as big as the set font size) </p>
    <p style="font-size:1.5rem;"> I'm <u>1.5rem</u> big (twice as big as the root element font-size) </p>
    
    <div style="font-size:0.8em; border:solid 1px black;">            // parent font-size is 0.8em (relative to the current font size)
        <p style="font-size:150%;"> I'm <u>150%</u> big (relative to the font-size of the parent element) </p>
    </div>
                                                                      // element font-size is relative to the parent's font-size (0.8 * 150% = 1.2em)

Relative values vw / vh / vmin / vmaxTEST

- the height of this box is 20% relative to the current viewport's width

- the width of this box is 20% relative to the current viewport's height

My font-size is 2% of the current Smallest side (relative to the viewport)

My font-size is 2% of the current Largest side (relative to the viewport)

CODE : 
    <div style="width:20vh; height:20vw; border:black 3px double; background-color:azure;"></div>
    <p style="font-size:2vmin"> My font-size is 2% of the current Smallest side (relative to the viewport) </p>
    <p style="font-size:2vmax"> My font-size is 2% of the current Largest side (relative to the viewport) </p>
