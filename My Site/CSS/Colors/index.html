Example :

RGB method

This is paragrap's color is is made with RGB code. "color:rgb(150,360,68);", this is the three color code

    -- CODE (HTML) --
    <p style="font-size:20px;color:rgb(150,360,68);"> This is paragrap's color is is made with RGB code. "color:rgb(150,360,68);", this is the three color code </p>

Hexadecimal method

This is paragrap's color is made with Hexadecimal code. "color:#ec2e37;", these are basicaly color codes of photoshop

    -- CODE (HTML) --
    <p style="font-size:20px;color:#ec2e37;"> This is paragrap's color is made with Hexadecimal code. "color:#ec2e37;", these are basicaly color codes of photoshop</p>

Typing method

This is paragrap's color is is made with traditional code. "color:teal;", simple typing method

    -- CODE (HTML) --
    <p style="font-size:20px;color:teal;"> This is paragrap's color is is made with traditional code. "color:teal;", simple typing method</p>

HLS method

This paragrap is colored with HLS, it seems to be at first glance overcomplicated but it isn't. 

    -- CODE (HTML) --
    <p style="font-family:Arial, sans-serif; font-size:20px; color:hsl(315, 100%, 50%);"> This paragrap is colored with HLS, 
    it seems to be at first glance overcomplicated but it isn't. </p>
