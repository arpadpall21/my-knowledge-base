The background-size property

- background-size:auto (defatul) original size

- background-size: 250px 54px background sized manually

- background-size:contain maintains its aspect ratio and scaled to fill the area "letterboxed"

- background-size:cover maintains its aspect ratio and scaled to fill the whole area of the element

CODE : 
// HTML ---------------------------
    <div class="bgSize sAuto"></div>
    <div class="bgSize sManual"></div>
    <div class="bgSize sContain"></div>
    <div class="bgSize sCover"></div>
    
// CSS -----------------------
    .bgSize {
        width: 300px;
        height: 150px;
        border: solid 1px red;
        margin: 2px 15px;
        background-image: url(small-tile.png);
        background-repeat: no-repeat;
    }
    .sAuto { background-size: auto; }
    .sManual { background-size: 250px 54px; }
    .sContain { background-size: contain; }
    .sCover {background-size: cover; }

background-origin: border-box;

- The background-image starts from top-left inside border

- The background-image starts from the top-left corner of content

- The background-color starts from top-left inside border

- The background-color starts from top-left padding

- This Gradient has also 3 colors but they starting point is specified at every 25%

The radial-gradient() function size

- When the Radial Gradient Center is not in the middle we can size the Gradient

- The remaining space is always filled out with the LAST "color stop"

- 4 values are possible closest-side | farthest-side | closest-corner | farthest-corner

- The Radial Gradient default value is "farthest corner"

-This Radial Gradient extends to the "closest-corner"

-This Radial Gradient extends to the "farthest-side"

-This Radial Gradient extends to the "closest-side"
