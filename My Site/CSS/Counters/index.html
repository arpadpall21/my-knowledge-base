Property Value Description
CSS Counters
counter-reset none ((Default) does nothing)
counterId nr (sets a counter identifier and optionally its content (default 0))
 - creates a new counter id and optionally sets its value 4 2 9.6 8 3.1
counter-set none ((Default) does nothing)
counterId nr (sets a counter identifier and optionally its content (default 0))
 - does the same as counter-reset
- in theory the counter-set is used to initiate the counter and the counter-reset is for reseting an ongoing content (but in reality there's not much difference) 85 68 71 85 NO
counter-increment none ((Default) does nothing)
counterId nr (increments/decrements the counterId value (default increment value 1))
 - increments/decrements the counterId value (decrement example counter-increment: counterId -1) 4 2 9.6 8 3.1
|
counter(counterId) - returns the cssCounter value for the current element
- used with the content property to display the counter current value
counters(counterId, "nestSeparator") - does the same as counter() but recognizes nested list and implement nested counter
- used with the content property to display the counter current value

counter() another TEST

this paragraph has one child and another child and this last child has even more child

CODE : 
// HTML -----------------
    <p id="childCount"> this paragraph has
        <span>one child</span> and 
        <i>another child</i> and
        <b> this last child has <span>even more child</span> </b>
    </p>
    
    
// CSS --------------
    #childCount {
        counter-reset: childCount 0;
    }
    #childCount *::before {
        counter-increment: childCount 1;
        content: counter(childCount) "(";
        color: white;
        font-weight: bold;
    }
    #childCount *::after {
        content: ")";
        color: white;
        font-weight: bold;
    }
