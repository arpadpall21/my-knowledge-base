Description and Demonstration :

- CSS lets us transform block level elements in 2D or 3D such as rotate, skew, ect..

The transfrom / transform-origin properties

Example :

The transform: translate property

This Box is moved form it's original position

This Box second box is also moved form it's original position

CODE : 
// HTML -----------------
    <div class="transform-translate1"p>
        <p>This Box is moved form it's original position</p>
    </divp>
    <div class="transform-translate2">
        <p>This Box second box is also moved form it's original position</p>
    </div>
    
// CSS -----------------
    .transform-translate1{                                      .transform-translate2{
        width: 300px;                                               width: 300px;
        height: 100px;                                              height: 100px;
        background: lightgreen;                                     background: lightgreen;
        border: solid 2px black;                                    border: solid 2px black;
        padding: 5px;                                               padding: 5px;
        text-align: center; text-align: center;                     text-align: center;
        -ms-transform: translate(100px, 10px);                      -ms-transform: translate(500px, -55px);
        -webkit-transform: translate(100px, 10px);                  -webkit-transform: translate(500px, -55px);
        transform: translate(100px, 10px);                          transform: translate(500px, -55px);
    }                                                           }

- in the second box we set the "perspective" property on the parent element so all children elements are rendered in the same perspective depth

      <div class="perspTest perspChild1">right</div>
      <div class="perspTest perspChild2">left</div>
  </div>

      display: inline-block;

      perspective: 100px;

      width: 100px;
      height: 100px;

      background-color: yellow;

      position: absolute;

      transform-origin: right;
  }

The perspective-origin property

- the black frame is the original size and place of the element

center center (default)
left top
right bottom
100% 65%
CODE : 
// HTML ------------------------------------------------
  <div class="perspOrg per_org1">
      <div> center center (default)</div>
  </div>
  <div class="perspOrg per_org2">
      <div> left top </div>
  </div>
  <div class="perspOrg per_org3">
      <div> right bottom </div>
  </div>
  <div class="perspOrg per_org4">
      <div> 100% 65% </div>
  </div>
  
// CSS ----------------------------
  .perspOrg {
      border: solid 3px black;
      display: inline-block;
      perspective: 130px;
      vertical-align: top;
  }        
  .perspOrg div {
      border: solid 2px black;
      background-color: yellow;
      width: 150px;
      height: 150px;
      transform: translateZ(-50px) rotateY(10deg) rotateX(10deg);
  }
  .per_org1 { perspective-origin: center center; }
  .per_org2 { perspective-origin: left top; }
  .per_org3 { perspective-origin: right bottom; }
  .per_org4 { perspective-origin: 100% 65%; }

The backfase-visibility property

Example :

backface-visibility:

myElement
CODE : 
// HTML ---------------------------
    <div id="backfaceVis">myElement</div>
    <script>
    var element = document.getElementById("backfaceVis");
    
    document.getElementById("visible").onclick = function(e) {
        element.style.backfaceVisibility = "visible";
    }
    document.getElementById("hidden").onclick = function(e) {
        element.style.backfaceVisibility = "hidden";
    }
    </script>
    
// CSS -----------------------------
    #backfaceVis {
        width: 250px;
        height: 75px;
        border: solid 2px black;
        background-color: yellow;
        margin: auto;
        transform-origin: center center 100px;
        animation: backFace 4s linear infinite;
    }
    @keyframes backFace {
        from { transform: perspective(800px) rotateY(0deg); }
        to { transform: perspective(800px) rotateY(360deg); }
    }
