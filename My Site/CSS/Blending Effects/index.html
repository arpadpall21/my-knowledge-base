The isolation property

- The grand-child element (farthest element from the viewer) with the image as background has the mix-blend-mode: screen; set (background mixes with all content which is in front of it)

- the isolator element is the one wiht blue border (middle element)

- the parent element is the one with the red background color (closest to the viewer)

1: no isolation set on the middle element (background mixes with the conent)

2: isolation: isolate set on the middle element (isolate belnding all elements that are behind it)

CODE : 
// HTML --------------------
    <div class="isolate">
        <div class="isolateCh_1">
            <div class="isolateGrandCh_1"></div>
        </div>
        <div class="isolateCh_2">
            <div class="isolateGrandCh_2"></div>
        </div>
    </div>
        
// CSS -------------------------
    .isolate {                                          // parent element 
        width: 400px;
        height: 200px;
        border: solid 2px black;
        display: flex;
        background-color: red;
    }
    .isolateCh_1 {
        display: flex;
        margin: auto;
        width: 150px;
        height: 150px;
        border: solid 3px blue;
        isolation: auto;                                // no isolation
    }
    .isolateGrandCh_1 {
        margin: auto;
        width: 100px;
        height: 100px;
        border: solid 2px black;
        background-image: url(Images/rock.jpg);
        background-size: cover;
        mix-blend-mode: screen;
    }
    .isolateCh_2 {
        display: flex;
        margin: auto;
        width: 150px;
        height: 150px;
        border: solid 3px blue;
        background-color: green;
        isolation: isolate;                             // isolate 
    }
    .isolateGrandCh_2 {
        margin: auto;
        width: 100px;
        height: 100px;
        border: solid 2px black;
        background-image: url(Images/rock.jpg);
        background-size: cover;
        mix-blend-mode: screen;
