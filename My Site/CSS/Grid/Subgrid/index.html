Description and Demonstration :

- A subgrid is a gird item that also declared as grid, its items interit the main grid's attributes

- So basically we can place grid item's direct child element on the main grid, this is useful for a prefect gird style layout

- The subgrid inherits all the main gird's attributes, but those can be overwritten, gird lines inside the subgrid starts from 1 (not inherited from the parent) 

    gridItem {
        display: grid;                                // declared as subgrid
        grid-template-columns: subgrid;    // columns are part of the main grid
        grid-template-rows: subgrid;       // rows are part of the main grid
    }

Examples:

Subgrid

- the grid boxes are items in the subgrid

Example : 
// HTML --------------------------
    <div class="base-grid">
      <div class="base-grid-item-1"></div>
      <div class="base-grid-item-2"></div>
      <div class="base-grid-item-3">
        <div></div>
        <div></div>
        <div></div>
        <div></div>
        <div></div>
        <div></div>
      </div>
      <div class="base-grid-item-4"></div>
    </div>
    
    
// CSS --------------------------
    .base-grid {
      display: grid;
      border: solid 2px black;
      background-color: blueviolet;
      grid-template-columns: repeat(4, 1fr);
      grid-template-rows: repeat(3, 50px);
      gap: 3px;
      padding: 3px;
    }
    .base-grid > div {
      border: solid 2px black;
      background-color: orange;
    }
    .base-grid-item-1 {
      grid-column: 1 / 3;
    }
    .base-grid-item-2 {
      grid-column: 3 / 5;
    }
    .base-grid-item-3 {                      // subgrid
      display: grid;
      grid-column: 1 / 3;
      grid-row: 2 / 4;
      grid-template-columns: subgrid;
      grid-template-rows: subgrid;
    }
    .base-grid-item-4 {
      grid-column: 3 / 5;
      grid-row: 2 / 4;
    }
    
    .base-grid-item-3 > div {
      border: solid 1px black;
      background-color: green;
    }

Full Subgrid

- grand child elements are subgrid items, they are placed on the main grid

Example : 
// HTML --------------------------
    <div class="my-subgrid">
      <div class="my-subgrid-child-1">
        <div></div>                       // grand child elements placed on the main grid
        <div></div>
        <div></div>
        <div></div>
      </div>
      <div class="my-subgrid-child-2">
        <div></div>
        <div></div>
        <div></div>
        <div></div>
      </div>
      <div class="my-subgrid-child-3">
        <div></div>
        <div></div>
        <div></div>
        <div></div>
      </div>
      <div class="my-subgrid-child-4">
        <div></div>
        <div></div>
        <div></div>
        <div></div>
      </div>
    </div>
    
    
// CSS --------------------------
    .my-subgrid {
      display: grid;
      border: solid 2px black;
      background-color: blueviolet;
      padding: 2px;
      grid-template-columns: repeat(4, 1fr);
      grid-template-rows: repeat(2, 50px);
      grid-gap: 3px;
    }
    [class^="my-subgrid-child"] {                 // direct child elements are decalred as subgid
      display: grid;
      border: solid 1px black;
      grid-template-columns: subgrid;             // subgrid
      grid-template-rows: subgrid;                // subgrid
    }
    
    .my-subgrid-child-1, .my-subgrid-child-3 {
      grid-column: 1 / 3;
    }
    .my-subgrid-child-2, .my-subgrid-child-4 {
      grid-column: 3 / 5;
    }
    
    [class^="my-subgrid-child"] > div {           // subgdir child elements 
      border: solid 1px black;
      background-color: orange;
    }
