CSS-trick complete guide grid

Remember This :

- a grid area can only be rectangle shaped (ex: not "L" shaped grid area allowed) [2025-01-04]

Description and Demonstration :

- a grid layout is made of columns and rows, areas on this grid are called gird areas

- we can declare elements as grid container with display:grid or display:inline-grid, all direct child elements will automatically become grid items, (grid items are sized according to the grid)

- the gird does not affect its grid items it just lays a map for them, gird items sort of float on the grid, therefore they do not affect each other eighter, they can overlap withount interfering

Explicit VS Implicit Grid:

- Explicit Grid: grid we explicitly define with grid-template-columns and grid-template-rows

- Implicit Grid: any grid area that of the explicit grid (the borwser automatically lays the grid for this part)

Sizing with fr (fraction) unit

- sizing with fr (fraction) unit works only with grids (cannot size regular elements) [TESTED : 2019-05-25]

- the fr (fraction) unit is overriden by the elements content (larger content stretches a shorter fraction value)

The repeat() function

- repeat(4, 1fr) - inserts 4 columns with 1fr width

- repeat(auto-fill, minmax(100px 1fr)) - repeats the grid items as many times as possible (takes the minimal item width without filling the available space)

- repeat(auto-fit, minmax(100px 1fr)) - repeats the grid items as many times as possible (fill up the available full space)
