Custom properties (variables) 49 31 36 15 9.1 

    parentElement {
        --var: val;           // declared variable available only in the parent scope
    }
    
    
    childElement {       // the selected element must be a child of parentElement
        prop: var(--var, fallbackVal);       // using var's value
    }
Example : 
// HTML ---
    <div>
      <div class="demo">
        <p> my element <span>child element</span></p>  
      </div>
    </div>
    
    
// CSS -----
    .demo {
      --myColor: red;                           // custom values declared in the parent element
      --myBorder: solid 2px blue;
      border: var(--myBorder);                  // custom value used on the element itself
    }
    .demo p {
      color: var(--myColor);                    // custom value used on the child element (custom properties inherit just like normal properties (the <span> child element inherit this color automatically) 
      border: var(--myBorder);                  // the <span> child element does NOT inherit this property because the "border" property non-inheritable 
      font-size: var(--myFontSize, 2em);        // fallback value is used because the "--myFontSize" identifier is undefined
    }
    div {
      border: var(--myBorder);                  // ancestors do not inherit custom properties (because of the scoping mechanism)
    }

CSS property inheritance

- some CSS property inherit values from their parent some don't! (check docs) 

// HTML ---------------------------------------------
    <p>parent<span>child</span></p>
        
// CSS ----------------------------------------------
    p {
        color: red;                             // the color property is inherited automatically so the child element (<span>child</span>) font color is red 
        border: solid 2px yellow;               // the border property is not inherited so the child element (<span>child</span>) has no border 
    }

Global property values initial / revert / inherit / unset

- these property values are available on all CSS properties

Example :

initial / revert TEST

DIV-1
DIV-2
CODE : 
// HTML -------------------------------------------------------------------------------
    <div class="div_1"> DIV-1 </div>
    <div class="div_2"> DIV-2 </div>
    
// CSS --------------------------------------------------------------------------------
    .div_1{
        border: solid 2px black;
        height: 50px;
        background-color: orange;
        display: initial;                                                              // the display's property default (initial) value is 'inline'    
    }
    .div_2{
        border: solid 2px black;
        height: 50px;
        background-color: orange;
        display: revert;                                                               // according to the browser's (User Agent) stylesheet the display's propertiy value for div elements is 'block'    
    }                                                                                  // the 'revert' keyword resets the default value for the property according to the browser's stylesheet

all property TEST

myElement (set initial)

CODE : 
-- CODE (HTML) --
    <p id="setInitial"> myElement (set initial) </p>
    
    <div class="parentInherit">
        <p id="setInherit"> myElement (set inherit) </p>
    </div>
    
    <p class="inheritIt"> my Child inherit my border 
        <span>ta Daaa!(because every property is inherited by this child element the <mark>display:block;</mark> is inherited too!) </span>
    </p>
    
-- CODE (CSS) --
    #setInitial {
        color: green;
        text-decoration: underline;
        font-weight: bold;
        line-height: 150%;
    }
    #setInitial {
        all: initial;                               // all set properties are set to initial 
        
    }
    .parentInherit {
        color: yellow;
        text-decoration: line-through;
        font-weight: 100;
        line-height: 200%;
    }
    #setInherit {
        color: green;
        text-decoration: underline;
        font-weight: bold;
        line-height: 150%;
    }
    #setInherit {
        all: inherit;                               // element inherith everything from its parent 
    }
    .inheritIt {
        border: 3px groove red;                     // this property is not inheritable 
    }
    .inheritIt span {
        all: inherit;                               // even non-inheritable properties are inherited (border in this example)
    }
