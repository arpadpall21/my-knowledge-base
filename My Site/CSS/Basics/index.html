CSS property inheritance

- CSS inheritance defines what happens when no value is specified for a property of a child element

- depending on the property some are inherit their value from their parent automatically some don't

// HTML ---------------------------------------------
    <p>parent<span>child</span></p>
        
// CSS ----------------------------------------------
    p {
        color: red;                             // the color property is inherited automatically so the child element (<span>child</span>) font color is red 
        border: solid 2px yellow;               // the border property is not inherited so the child element (<span>child</span>) has no border 
    }

Global property values initial / revert / inherit / unset

- these property values are available on all CSS properties

DEMO 
// HTML -------------------------------------------------------------------------------
    <div class="d1"> </div>
    <div class="d2"> 
        <div></div>
    </div>
    
    
// CSS --------------------------------------------------------------------------------
  // initial / revert / inherit -------------------------------------------------------
    .d1{
        border: solid 2px black;
        height: 50px;
        background-color: orange;
        display: initial;                                                              // the display's property default (initial) value is 'inline' 
    }
    .d2{
        border: solid 2px black;
        height: 50px;
        background-color: orange;
        display: revert;                                                               // according to the browser's (User Agent) stylesheet the display's propertiy value for div elements is 'block'    
    }                                                                                  // the 'revert' keyword resets the default value for the property according to the browser's stylesheet    
    
    .d2 div{
        border: inherit;                                                               // child element inherits the border property value from its parent    
        height: 20px;
        background-color: green;
    }
    
  // unset ----------------------------------------------------------------------------
    .d2 div{
        border: unset;                                                                 // 'unset' = 'initial' because the border property is non inheritabl    
        height: 20px;
        color: unset;                                                                  // 'unset' = 'inherit' because the color property value is inherited from the parent     
    }

The all property

DEMO 
// HTML -------------------------------------------------------------------------------
    <div class="demoParent"> parentElement 
        <p class="demoChild"> childElement </p>
    </div>
    
// CSS --------------------------------------------------------------------------------
    .demoParent {
        color: red;
        border: solid 3px black;
        width: 70%;
        height: 75px;
    }
    .demoChild {
        color: white;
        border: dashed 2px green;
    }
    
  // initial --------------------------------------------------------------------------
    .demoChild {                    
        all: initial;                                                                  // all the properties are set to initial (default state)
    }
    
  // inherit --------------------------------------------------------------------------
    .demoChild {
        all: inherit;                                                                  // all properties are inherited from the parent element (even the non-inheritable ones ("width" "height" and "border"))  
    }
    
  // unset ----------------------------------------------------------------------------
    .demoChild {
        all: unset;                                                                    // inheritable properties are inherited from the parent element ("color")
    }                                                                                  // non-inheritable properties are set to "initial" ("width" "height" and "border")
    
  // revert ---------------------------------------------------------------------------
    .demoChild {
        all: revert;                                                                   // reverts all property values according to the browser's default stylesheet     
    }                                                                                  // inheritable properties are inherited from the parent ('color')
