Border Image

The border-image / border-image-outset / border-image-repeat / border-image-slice / border-image-source / border-image-width propertis

- for using image as porder we pass an image which is sliced into nine section (like a tic-tac-toe board) than handles these sliced parts as border images (requires the original border set to transparent)

DEMO : 
// HTML ----------
    <p class="demo"> TEST </p>
    
    
// CSS -------
    .demo {
        border: 10px solid transparent;                         // -! required 
        border-image-source: url("myImage.png");
        border-image-slice: 100 75;                            // slice horizontal lines are 100 pixels from top and bottom and vertical lines are 75 pixels form left and right
        
        border-image-width: 2;                                 // border width doubled towards inside without affecting the padding (relative to the original "border-width" (2x10 = 20px)
        border-image-repeat: round;                            // side image borders are repeted and "rounded" (stretched) in order to fill the side border image pattern perfectly
        border-image-outset: 3;                                // border image extended out 30px (border with 10px = 10x3) from the original border (without affeting the layout)
        
        border-image: url("myImage.png") 100 round;            // shorthand property (tried to add "width outset" but it is not working [TESTED 2019-01-09])
    }

The border-image-width / border-image-repeat properties

- border-image-width is extended towards the element's center

- border-image-repeat:strecth = stretches the side slices / repeat = repeats the side slices (without strecthing them) / round = repeats and strecthes the sice slices to fit the image border sides

- border-image-width:1;
- border-image-repeat:stretch;
- (default values)

- border-image-width:2; (number doubles the original border size)
- border-image-repeat:repeat; (side images repeated and cut)

- border-image-width:10%; (relative to the element size)
- border-image-round: round; (side images repeated and strecthed to fit)

- border-image-width: auto; (width of the "border-image-slice")
- border-image-round: repeat;

- border-image-width: 1.5
- border-image-round: space; (distribute spaced between sice slices)

Rounded Corners

The border-radius

- rounds element corderns, the "border-radius" properties actually modify the box shape of the element (it's not just a visual effect)

DEMO : 
// HTML ---------
    <div class="demo"></div>
    
    
// CSS -------
    .demo {
        width: 200px;
        height: 200px;
        border: solid 5px black;
        border-top-left-radius: 25%;
        border-top-right-radius: 20%;
        border-bottom-right-radius: 15%;
        border-bottom-left-radius: 10%;
        
        border-radius: 25% 20% 15% 10%;   // shorthand property for all the above parameters
    }

Outline

The outline-color / outline-width / outline-style / outline / outline-offset properties

- the outline is a "sort of border" around the element, which does not affect the layout (outest line)

DEMO : 
// HTMl -----------
    <div class="demo"></div>
    
    
// CSS ----------
    .demo {
        width: 200px;
        height: 200px;
        outline-color: red;
        outline-style: solid;
        outline-width: 10px;
        outline-offset: 5px;
        
    // shorthand property does the same ----------------
        outline: 10px solid red;
        outline-offset: 5px;                                   // shorthand property does not support the "outline-offset" therefore it is set separately
    }
