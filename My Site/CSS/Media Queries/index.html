Description and Demonstration :

- Media Queries are used to deliver different styles for different devices and screen sizes 1 1 9.2 6 1.3

- There are 3 ways to use media queries:

- using the @media at rule in *.css file

- in *.css file at importing css file (@import at rule)

- passing a media query conditions on the stylesheet import 

    not|only

                
 not - negates this query condition 

                
 only - prevents older browsers from using this media query condition 

            
 mediaType

                
 all - all media devices 

                
 print - for print view mode 

                
 screen - for screens 

                
 speech - speech synthesizers 

            
 [and (condition 

                
 orientation - landscape or portrait 

                
 height - height of the viewport (including scrollbar) 

                
 width - width of the viewport (including scrollbar) 

                
 max-height - max height of the viewport 

                
 min-height - minimum height of the viewport 

                
 max-width -  maximum width of the viewport 

                
 min-width - min width of the viewport 

                
 [more...]

// applying media quier condtion on stylesheet import --
    <head>
        <link rel="stylesheet" media="mediaQueryCondition" href="cssFilePath">
    <head>
    
    
// media query condition in stylesheet --
    @import filePath:str|url(filePath:str) mediaQueryCondition;    // styles imported if mediaQueryCondition are true
    
    
    @media mediaQueryCondition {   // styles applied if mediaQueryCondition are true
        // css styles...
    }
DEMO: 
    <head>
        <link rel="stylesheet" media="only print" href="print.css">                    // sheet only implemented on print view     
        <link rel="stylesheet" media="(max-width: 800px)" href="max800.css">           // sheet only implemented on devices less than 800 pixels
    </head>
    
    
// css file -------------------------------------
    @import './mobileStyles.css' screen and (max-width:500px)                 // import css rules if the media is screen and less than 500px width
    
    
    @media screen and (max-width:500px) {                                              // applies styles to screens that are less than 500px width    
        // CSS code....
    }
    
    @media print {  }                                                               // runs the block if the view is in print mode 
    @media screen, print {  }                                                            // multiple query conditions (only one of the must be true)
    
    @media screen and (max-width: 12450px) {  }  
    @media screen and (min-width: 30em) and (orientation: landscape) {  }  
    @media (min-height: 680px), screen and (orientation: portrait) {  }             // multiple query conditions (only one of the must be true)

simple example

(width=1000px) Background Color is green between 500px and 800px
(width=1000px) Background Color is green below 500px and above 800px
CODE : 
-- HTML --
    <div class="media-one"> (width=1000px) Background Color is green between 500px and 800px </div>
    <div class="media-two"> (width=1000px) Background Color is green below 500px and above 800px </div>
                
-- CSS --
    [class|=media]{
        background-color: orange;
        max-width: 1000px;
        height: 50px;
        margin: auto;
        text-align: center;
        border: 3px black solid;
    }
    @media screen and (max-width:800px) and (min-width:500px){
        .media-one{
            background-color: green; 
        }
    }
    @media screen and (max-width:500px), screen and (min-width:800px){
        .media-two{
            background-color: green;
        }
    }
