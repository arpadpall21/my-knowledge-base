Remember This :

- the user font family must support certain features in order to use specific properties (ex: the font-variant-numeric is only usable on font families suporting it)

Description and Demonstration :

- when specifying the font family we alwasy declare a generic font family as fallback, this ensures if the desired font family doesn't exist on the user's system a family type of the specified generic is used as fallback (generic font family types: Serfi, Sans-serif, Monospace, Cursive, Fantasy)

- Web Safe Fonts are font families what are preinstaled on the user's system> Web Safe Fonts

The @font-face rule

- we can use the @font-face roule to import fonts in our projects 

    @font-face {
        font-family: customName;                               // custom name to the added font 
        src: url(URL_to_the_font_file);                        // path to the font file
        font-stretch: value;
        font-style: value;
        font-weight: value;
        unicode-range: U+unicodePoint-unicodePoint, ...    // default U+0-10FFFF // sets a unicode character range (used for browser performance imporuvement)  
    }
