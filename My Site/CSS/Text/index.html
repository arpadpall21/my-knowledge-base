                        line4 line4 line4 line4

- white-space:normal; (↵ carriage returns are ignored / extra white spaces are collapsed / text wrapped when it hits the border)

line1 line1 line1 line1 * line2 line2 line2 line2 line2 line2 * line3 line3 line3 line3 * line4 line4 line4 line4

- white-space:nowrap; (↵ carriage returns are ignored / extra white spaces are collapsed / text is NOT wrapped when it hits the border)

line1 line1 line1 line1 * line2 line2 line2 line2 line2 line2 * line3 line3 line3 line3 * line4 line4 line4 line4

- white-space:pre; (↵ carriage returns create new line / extra white spaces are preserved / text is NOT wrapped when it hits the border)

line1 line1 line1 line1 * line2 line2 line2 line2 line2 line2 * line3 line3 line3 line3 * line4 line4 line4 line4

- white-space:pre-wrap; (↵ carriage returns create new line / extra white spaces are preserved / text wrapped when it hits the border)

line1 line1 line1 line1 * line2 line2 line2 line2 line2 line2 * line3 line3 line3 line3 * line4 line4 line4 line4

- white-space:pre-line; (↵ carriage returns create new line / extra white spaces are collapsed / text wrapped when it hits the border)

line1 line1 line1 line1 * line2 line2 line2 line2 line2 line2 * line3 line3 line3 line3 * line4 line4 line4 line4
