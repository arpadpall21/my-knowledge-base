The user-select property

Example :

- user-select:auto; (browser defines) this text can be selected normally

- user-select:none; this text cannot be selected at all

- user-select:text; this text can be selected normally

- user-select:all; this whole text selected by a single user click

CODE 
// HTML --------------
    <span style="user-select:auto;">(browser defines) this text can be selected normally </span>
    <span style="user-select:none;"> this text cannot be selected at all</span>
    <span style="user-select:text;"> this text can be selected normally</span>
    <span style="user-select:all;"> this whole text selected by a single user click</span>
