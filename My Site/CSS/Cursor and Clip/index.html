cursor / pointer-events properties 

SYNTAX :    pointer-events: auto (reacts to pointer events)     //  defines whether or not the element reacts to the pointer event (like hover and click)
                            none (does NOT react to pointer events)
                            
            cursor: cursorVal, cursorVal ,...                   // defines the cursor appearance when the mouse pointer is over the element 
                - cursorVal : auto                   text                   col-resize            ew-resize
                              default                vertical-text          row-resize            ns-resize
                              none                   alias                  n-resize              nwse-resize
                              context-menu           copy                   e-resize              nesw-resize
                              help                   move                   s-resize              zoom-in
                              pointer                no-drop                w-resize              zoom-out
                              progress               not-allowed            ne-resize             url(URL) (32x32 pixel image)
                              wait                   grab                   nw-resize                          
                              cell                   grabbing               sw-resize                           
                              crosshair              all-scroll             se-resize                           
    
// pointer-events -----------------------------------------------
    #demo1 { pointer-events: auto; }                            // element reacts to pointer events as normal 
    #demo2 { pointer-events: none; }                            // element won't react on pointer events (like click, focus, ...)
    
// cursor ------------------------------------------------------- 
    #demo1 { cursor:none; }                                     // mouse cursor hidden when it is over this element 
    #demo2 { cursoror:url(myCursor.png), auto; }                // image set as cursor  // -! fallback value specified in case of error

user-select property 

SYNTAX :    user-select: auto   // (default) browser defines    //  defines whether the text of an element can be selected   
                         none   // no selection allowed
                         text   // selected by click and dragging or double click
                         all    // all text selected by a single click
    
    .myElement {
        user-select: none;                                      // text selection of the element is forbidden 
        -webkit-user-select: none;
        -moz-user-select: none;
        -ms-user-select: none;
    }

Clipping

clip / clip-path property 

SYNTAX :    clip: auto (no clipping)                            // clips off an area of ANY element (must be positionned abosolute, no effect if overflow:visible;) (Depreacated)
                : rect(y, width(count left), height(count top), x)
                      - width(count left) : width of the clipped region coordinate is counted from the left edge (left-to-right
                      - height(count left) : height of the clipped region coordinate is counted from the top edge (top-to-bottom)
            
            clip-path: none                                     //  defines a clipping region on an element (show/hides regions on the element, does not shape the element)
                     : url(URL#clipPath)    // applies an SVG clipPath (URL points to an SVG <clipPath> element)
                     : cssBasicShape        //  MDN basic Shapes[link] /  Awesome Shape Generator for the clip-path property (Bennett Feely) [link] 
    
-- HTML -- 
    <img id="demo" src="demo.jpg" alt="me"> 
    
-- CSS -- 
// clip -------------------------------------------------------------------------------- 
    #demo {
        position: absolute;
        left: 200px;                                            // !! the element keeps ist original placement the clip property only hides the clipped off areas 
        clip: rect(30px, 250px, 200px, 15px);                   // "30px" is the Y coordinate 
    }                                                           // "250px" is the width of the clipped region   // -! counted from the left (width = 235px (250 - 15) in this situation)
                                                                // "200px" is the height of the clipped region  // -! counted form the top (height = 170px (200 - 300) in this situation)
                                                                // "15px" is the X coordinate
// clip-path --------------------------------------------------------------------------- 
    #demo {
        -webkit-clip-path: polygon(0% 20%, 60% 20%, 60% 0%, 100% 50%, 60% 100%, 60% 80%, 0% 80%);       // polygon shape clip
        clip-path: polygon(0% 20%, 60% 20%, 60% 0%, 100% 50%, 60% 100%, 60% 80%, 0% 80%);   
    }
