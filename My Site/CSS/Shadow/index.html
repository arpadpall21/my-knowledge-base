        text-shadow: 5px 5px red, -5px -5px green;                                     // 2 shadows are added to the text
        box-shadow: 0 0 5px 10px blue;                                                 // spreaded box shadow 
    }
    .myElement {
        box-shadow: 5px -2px 7px 0 blue, 5px -2px 7px 0 blue inset;                    // outset and inset shadows declared (awesome stuff)
    }

        margin: 1.2em 1em;
    }

    .sh4 { box-shadow: 5px 5px 10px 7px red inset; }

Shadows inserted with the ::after pseudo classes

The Ocean is beautiful

CODE : 
// HTML ---
    <div class="secondsw">
        <img src="Ocean.jpg" title="Ocean">
        <p style="text-align: center;"> The Ocean is beautiful</p>
    </div>
    
    
// CSS ---
    .secondsw > img{
        width: 100%;
        border: 1px solid grey;
        border-style: inset;
    }
    
    .secondsw::after{                     
        content: "000";                   
        background-color: white;          
        position: absolute;               
        height: 80%;                      
        bottom: 10%;                      
        left: 0px;                        
        box-shadow: -5px 0 40px 5px black;
        z-index: -1;                      
    }
