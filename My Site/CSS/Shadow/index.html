Description and Demonstration

- with CSS you can add shadow to text and to elements

DEMO 
    .myElement {
        text-shadow: 5px 5px red, -5px -5px green;                                     // 2 shadows are added to the text
        box-shadow: 0 0 5px 10px blue;                                                 // spreaded box shadow 
    }
    .myElement {
        box-shadow: 5px -2px 7px 0 blue, 5px -2px 7px 0 blue inset;                    // outset and inset shadows declared (awesome stuff)
    }
