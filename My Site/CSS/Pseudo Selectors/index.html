I'm the only child of my parent (selected by the :only-child selector because this element is the "only child" of its parent)

we are siblings of you parent (not selected)

CODE : 
-- CODE (HTML) --

        <p> we are siblings of you parent (not selected) </p>

        <p> we are siblings of you parent (not selected) </p>
    </dir>

-- CODE (CSS) --
    .parent_only {
        border: solid 1px black;
    }
    .parent_only p:only-child {
        color: green;
        font-weight: bold;
    }

        color: olivedrab;
        font-weight: bold;

The :first-child / :last-child / :nth-child() / :nth-last-child() / :first-of-type / last-of-type / nth-of-type() / nth-last-of-type() pseudo selectors

Example : 
// HTML ---
    <div>
        <p></p>
        <span></span> 
        <div></div>
        <div></div>
        <p></p>
        <p></p>
        <span></span>
        <div></div>
        <span></span>
    </div>
    
    
// CSS ----
// :first child / :last child / :first-of-type / :last-of-type ----
    div :first-child { }                                             // first p element
    div :last-child { }                                           // last span element
    
    div :first-of-type {  }                                     // each first p span div elements 
    div span:first-of-type {  }                                     // first span element
    div :last-of-type {  }                                     // each last p span div elements 
    div p:first-of-type {  }                                     // last p element
    
    
// :nth-child() / nth-of-type() / nth-last-child() / nth-last-of-type() --
    div :nth-child(2) { }                                   // second element (which is a span element)
    div :nth-child(even) { }                                   // every even elements (regardless of type)
    div :nth-last-child(2) { }                                   // second element from the end (which is a div element)
    
    div div:nth-of-type(2) { }                              // second div element 
    div div:nth-of-last-type(2) { }                              // second div element from the end
