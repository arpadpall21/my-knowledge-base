The ::backdrop pseudo element

Example : This is some modal (Esc to close)
CODE : 
// HTML -------------------------
    <button onMouseup="openStyledModal()"> Open Modal </button>
    <dialog id="styledModal"> This is some modal (Esc to close) </dialog>
    
    <script>
      const modal = document.getElementById('styledModal');
      function openStyledModal() {
        modal.showModal();
      }
    </script>
    
    
// CSS -------------------------
    #styledModal::backdrop {
      background-color: red;
      opacity: .3;
    }

Popover Element
CODE : 
// HTML -------------------------
    <button popovertarget="some-popover"> Toggle Popover </button>
    <div id="some-popover" popover> Popover Element </div>
    
// CSS -------------------------
    [popover] {                     // targets all popovers on the page
      background-color: red;
      color: white;
      padding: 50px;
    }
    [popover]::backdrop {
      background-color: blue;
      opacity: .4;
    }
