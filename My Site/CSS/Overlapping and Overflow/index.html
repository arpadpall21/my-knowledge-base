    <div class="parent">
        <div></div>
        <div></div>
    </div>
    
    
// CSS -------------------
    .parent {
        position: relative;
        width: 100px;
        height: 100px;
        border: solid black 2px;
        overflow: hidden;
    }
    .parent div {
        position: absolute;
        width: 50px;
        height: 50px;
        border: solid black 1px;
        background-color: bisque;
    }
    .parent div:first-child {
        left: -25px;
        top: -10px;
    }
    .parent div:last-child {
        left: 50px;
        bottom: -13px;
    }

z-index TEST

This image is place begind the text by using the absolute property and the images "z-index" is set the "-1"

CODE : 
// HTML ------------
    <div class="zIndex">
      <img src="./sunrise.webp" alt="sunrise">
      <p> This image is place begind the text by using the absolute property and the images "z-index" is set the "-1"</p>
    </div>
    
    
// CSS ------------
    .zIndex{
        position: relative; 
        border: solid black 5px; 
        width: 55%; 
        height: 200px;
    }
    .zIndex img{
        width: 150px; 
        position: absolute; 
        top: 1%; 
        left: 1%;
        z-index: -1;
    }
    .zIndex p{
        position: absolute;
        font-family: Arial, Verdana, sans-serif;
        font-size: 2.0em; 
        color: white;
        z-index: 1;
    }

z-index TEST 2

- the yellow box is a floating element is floats over the below inline element but it is placed behind it with the z-index property

CODE : 
// HTML -----------
    <div id="floatStack"></div>
    <div id="floatStack_2"></div>
    
    
// CSS -------------
    #floatStack {
        width: 200px;
        height: 100px;
        border: solid 2px black;
        background-color: gold;
        position: relative;                 // element must be placed other than "static" in order to the "z-index" to work
        float: left;
        z-index: 0;
    }
    #floatStack_2 {
        width: 150px;
        height: 200px;
        border: solid 2px black;
        background-color: green;
        opacity: .8;
        position: relative;                  // element must be placed other than "static" in order to the "z-index" to work
        z-index: 1;
    }
