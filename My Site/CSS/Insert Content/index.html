Property Value Description
content normal ((Default) same as "none")
none (no content)
"text" (text)
counter(counterId) (css counter)
attr("attrId") (targeted attribute's value)
open-quote (add opening quote)
close-quote (add closing quote)
no-open-quote (removes opening quote)
no-close-quote (removes closing quote)
url("URL") (media file)
value value .. (multiple values are allowed)
 - used with the ::before and ::after pseudo elements in order to insert a generated content 1 1 4 12 8 1

Description and Demonstration : 

// HTML ---
    <p> myContent </p> 
    
    
// CSS ------
    p::before {
        content: " -->"                 // adds the " -->" text before the targeted paragraph
        color: red;                     // newly added content styled here
    }
    
// display on the webpage --
    --> myContent
Example :
  • style value =
  • with quotes
  • quotes added and removed
  • media added =
CODE : 
// HTML ----------------------
    <ul class="content">
        <li style="color:darkviolet;"> style value = </li>
        <li> with quotes </li>
        <li> quotes added and removed </li>
        <li> media added = </li>
    </ul>
    
    
// CSS -----
// display attribute value ---------------------------
    .content li:first-child::after {
        content: attr(style);
    }
    
// add/remove quotes ---------------------------------
    .content li:nth-child(2)::before {
        content: open-quote;                    // quote opened 
    }
    
    .content li:nth-child(2)::after {
        content: close-quote;                   // quote closed
    }
    
    // quotes added and removed ----------------------
    .content li:nth-child(3)::before {
        content: open-quote;
    }
    
    .content li:nth-child(3)::after {
        content: close-quote;
    }
    
    .content li:nth-child(3)::before {
        content: no-open-quote;                 // remove opening quote
    }
    
    .content li:nth-child(3)::after {
        content: no-close-quote;                // remove closing quote
    }
    
// add media -----------------------------------------
    .content li:last-child::after {
        content: url(Button.jpg);               // media file added
    }

- content added by the ::before and ::after pseudo-elements are not part of the DOM
