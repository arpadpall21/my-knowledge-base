The caret-color property

Example :

CODE : 
// HTML --------------------
    <input class="car-col-1"><br>
    <input class="car-col-2"><br>
    <input class="car-col-3">
    
// CSS --------------------
    [class|=car-col] {
        font-size: 1.7rem;
        background-color: black;
    }
    .car-col-1 { caret-color: red; }
    .car-col-2 { caret-color: blue; }
    .car-col-3 { caret-color: orange; }
