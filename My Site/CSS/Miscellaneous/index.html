Description and Demonstration :

the all property 

SYNTAX :    all: initial        // initial state all properties                     //  resets all properties of the element (except "unicode-bidi" and "direction")
               : inherit        // inherit all properties from the parent even non-inheritable ones
               : unset          // inherit inheritalbe properties from the parent and non-inheritable properties are set to "initial" 
    
    <div class="demoParent"> parentElement 
        <p class="demoChild"> childElement </p>
    </div>
    
-- CSS --
    .demoParent {
        color: red;
        border: solid 3px black;
        width: 70%;
        height: 75px;
    }
    .demoChild {
        color: white;
        border: dashed 2px green;
    }
    
// initial ----------
    .demoChild {                    
        all: initial;                           // all the properties are set to initial (default state)
    }
    
// inherit ----------
    .demoChild {
        all: inherit;                           // all properties are inherited from the parent element (even the non-inheritable ones ("width" "height" and "border"))  
    }
    
// unset ------------
    .demoChild {
        all: unset;                             // inheritable properties are inherited from the parent element ("color")
    }                                           // non-inheritable properties are set to "initial" ("width" "height" and "border")

CSS property inheritance

- CSS inheritance defines what happens when no value is specified for a property of a child element

- depending on the property some are inherit their value from their parent automatically some don't

// HTML ---------------------------------------------
    <p>parent<span>child</span></p>
        
// CSS ----------------------------------------------
    p {
        color: red;                             // the color property is inherited automatically so the child element (<span>child</span>) font color is red 
        border: solid 2px yellow;               // the border property is not inherited so the child element (<span>child</span>) has no border 
    }
