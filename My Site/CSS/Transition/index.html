Chrome and Opera transition problem

- I had this weird problem in Chrome and Opara about one year ago [time now: 2019-05-17], they fired the transtion automatically uppon the page loading, apparently it is fixed now (TESTED below)

- that was the fix for that problem 

// HTML ------------------------------------------
    <body class="preload">                            // add a class to your body element "preload"
    
    <script>                                          // script must run after the below CSS code 
        $(window).load(function() {
          $("body").removeClass("preload");
        });     
    </script>
    
// CSS -------------------------------------------         
    .preload * {
        transition: none;
        -webkit-transition: none;
        -moz-transition: none;
        -ms-transition: none;
        -o-transition: none;
    }

Description and Demonstration :

- web browsers perform animations by changing CSS property values, the change between the two values is handeled (animated) by the browser, this is the transformation phase

- CSS allows us to control this transformation phase

The transition-property / transition-duration / transition-delay / transition properties

DEMO 
// HTML -------------------------------------------------------------------------------
    <div class="demo"></div>
    
// CSS --------------------------------------------------------------------------------
    .demo {
        width: 200px;
        height: 200px;
        background-color: yellow;
        transition-property: background-color, width;                                  // transition properties set (properties that will affect the transtion)   
        transition-duration: 2s, 4s;
        transition-delay: 250ms, 1000ms;
  // shorthand property  ------------------------
        transition: background-color 2s 250ms, width 4s 1000ms;                        // does the same as the 3 properties above 
    }
    
    .demo:hover {                                                                      // we set the desired end states here 
        background-color: red;
        width: 400px;
    }

The transition-timing-function property

DEMO 
// HTML -------------------------------------------------------------------------------
    <div class="demo"></div>
    
// CSS --------------------------------------------------------------------------------
    .demo {
        width: 100px;
        height: 50px;
        border: solid black 1px;
        transition: width 3s;
        transition-timing-function: linear;                                            // continuous transition animation 
  // steps ---------------------------------------
        transition-timing-function: steps(15);                                         // transition animation starts form step 0 (same as "step(15, end)")   
        transition-timing-function: steps(15, start);                                  // transition animation starts from step 1 
    }
    
    .demo:hover { width: 500px; }

timing-function step TEST

- this property is the same for both transition-timing-function and animation-timing-function

- end starts the animation from step 0 but misses the last step before the animation restarts (this is the default)

- start starts the animation from step 1 and ends the animation at the last step before it restarts

timing-function: step(5)
timing-function: step(5, end)
timing-function: step(5, start)
CODE : 
-- CODE (HTML) --
    <button onclick="startAnim()">start animation</button>
    <div class="stepParent">
        <div class="stepTest st1">timing-function: step(5)</div>
        <div class="stepTest st2">timing-function: step(5, end)</div>
        <div class="stepTest st3">timing-function: step(5, start)</div>
    </div>
    
-- CODE (CSS) --
    .stepParent {
        border: solid 2px black;
        width: 1000px;
        margin: auto;
    }
    .stepTest {
        height: 30px;
        width: 250px;
        background-color: orange;
        border: solid 1px black;
        margin: 2px 0;
    }
    .st1 { 
        animation: 5s 2 forwards;
        animation-timing-function: steps(5); 
    }
    .st2 { 
        animation: 5s 2 forwards;
        animation-timing-function: steps(5, end); 
    }
    .st3 { 
        animation: 5s 2 forwards;
        animation-timing-function: steps(5, start); 
    }
    @keyframes st1 {
        from {width: 250px;}
        to {width: 998px;}
    }
    @keyframes st2 {
        from {width: 250px;}
        to {width: 998px;}
    }
    @keyframes st3 {
