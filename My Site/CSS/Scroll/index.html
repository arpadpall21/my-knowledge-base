Scroll Snap

The scroll-snap-type / scroll-snap-align / scroll-snap-stop / scroll-padding-* / scroll-margin-* properties

Concept:

- scroll snap - snap or lock the scollable content within its container onto a specific positions as the user scrolls through, this creates a smooth and precise scrolling experience, (especially useful on touch screen devices)

- block and inline directons - directions described from the perspective of writing direction

- in english block direction means vertical and inline direction means horizontal, but in Mongolian lines are laid out left to right which makes block direction left to right

DEMO (can be pseudocode)

The scroll-snap-type / scroll-snap-align properties

- scroll-snap-type: mandatory

CODE 
// HTML ----------------
    <div class="scrollSnapCont">
      <div class="even"></div>
      <div class="odd"></div>
      <div class="even"></div>
      <div class="odd"></div>
      <div class="even"></div>
    </div>
    
    
// CSS ----------------
    .scrollSnapCont {
        border: solid 3px blue;
        width: 60%;
        margin: auto;
        height: 200px;
        overflow-x: scroll;
        scroll-snap-type: y mandatory;
    }
    
    .scrollSnapCont > .even {
        background-color: beige;
        height: 170px;
        scroll-snap-align: start;
        margin-top: 5px;
    }
    
    .scrollSnapCont > .odd {

The scroll-margin-* / scroll-padding-* properties

- these 2 properties appears to do the same thing, the difference is scroll-padding-* is set on the scroll container VS the scroll-margin-* is set on individual scroll elements

- scroll-padding-top (the scroll-padding-block property does the same)
