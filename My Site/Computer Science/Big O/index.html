Notation Description
O(1) - Constant time|space
- algorithm runs the same amount of time (or needs the same amout of memor) regardless of the input size
O(log n) - Logarithmic time|space
- algorithm runs according to the log formula (if there's no number after the log we assume that is log2 = log base 2)
O(n) - Linear time|space
- algorithm runs the same amount of time (or need the same amount of memory) as the input size is
O(n log n) - Loglinear / Quasilinear time|space
- algorithm runs constant + logarithmic
O(n^c) - Polynomial time|space
- algorithm runs n (input size) to the power of c (constant)
O(c^n) - Exponential time|space
- algorithm runs c (constant) to the power of n (input size)
O(n!) - Factorial time|space
- algorithm runs factorial times

Description and Demonstration :

- the Big O notation O() measures the time and space needed by an algorithm (time and space complexity)

- it tells us how much time (operation) and space (memory) the algorithm needs relative to the passed input (which is n usually)

Big O simplification rules

- the Big O notation gives us an aproximation measure, this is because algorithms have a carp ton of operations, so describing the exact number of operations would result in a massive terms which would be useless because computers are soo fast that we only interested in the most task heavy operations anyways

- for this reason we use 2 simplification rules:

- 1) we drop every constant (or turn it to O(1))

- 2) we only keep the largest term 

    O(74568)                ->          O(1)                                           // single constant is always just '1'  
    O(512 * n)              ->          O(n)                                           // we drop constants 
    O(n/3)                  ->          O(n)
    O(5 * n * n)            ->          O(n^2)
    
    O(n + 1000)             ->          O(n)                                           // we keep only the largest term    
    O(n + n^2)              ->          O(n^2)
    O(n + 500 + n^3 + n^2)  ->          O(n^3)
    O(5n^2 + 100n + 17)     ->          O(n^2)
    O((n/3)^3 + 10n)        ->          O(n^3)

    function doubler(items){
        let newArray = [];
        
        for(let i = 0; i < items.length; i++){
            newArray.push(items[i]);                                                   // n 
            newArray.push(items[i]);                                                   // n 
        }
        
        return newArray;
    }
    
    // O(n + n)             ->      O(n)                                               // space complexity

    function zoo(n){                                                                   // in case of recursion each stack frame will take up 1 space (so a recursion is at least O(n) space complexity)    
        if (n == 0){
            console.log( 'liftoff!' );
            return
        }
        
        console.log(n);
        zoom(n - 1);
    }

        }

    // O(n/2)               ->      O(n)                                               // space complexity

analysis-1 

    function unique(array){
        let newArray = [];
        
        for (let i = 0; i < array.length; i++){                                        // called 'n' times 
            const ele = array[i];
            if (!newArray.includes(ele)){                                              // the 'includes()' method calls 'n' times as well 
                newArray.push(ele);
            }
        }
        
        return newArray;
    }
    
    unique(['cat', 'dog', 'rat', 'dog', 'cat', 'bird']);                               // -> ['cat', 'dog', 'rat', 'bird']
    
    // time complexity = O(n^2)    
    // space complexity = O(n)    
    
// make it better ---------------------------------------------------------------------
    function unique(array){
        let uniques = new Set();
        
        for (let i = 0; i < array.length; i++){                                        // called 'n' times 
            const ele = array[i];
            uniques.add(ele)                                                           // called '1' time 
        }
        
        return Array.from(uniques);
    }
    
    unique(['cat', 'dog', 'rat', 'dog', 'cat', 'bird']);                               // -> ['cat', 'dog', 'rat', 'bird']
    
    // time complexity = O(n^2)    
    // space complexity = O(n)

                console.log( str1[i] );
            }
        } else {

            }
        }
