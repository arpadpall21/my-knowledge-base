(Special) Computer Hardware

64bit VS 32bit processor

- a 32bit (x86) processor supports only up to 4GigaByte of RAM VS a 64bit processor supports up to 16Exabyte of RAM (basically a 64bit processor can handle larger data chunks)

- Old 32bit (and 16bit) processor names ended mostly with '86', that's why 32bit processors referenced as 'x86' (ex: Intel 8086 | Intel 80386 | Intel 80486 | AMD 8086 | AMD 386 | AMD 486 | ect...)

- the application must be written for a 64bit system in order to take advantage the 64bit benefits

Redundant Power Supply

- a single Power Supply unit containing multiple Power Supplys

- each Power Supply is hot swappable and can power the computer on its own, so if one fails the other will continue powering the computer

ECC memory (Error-Correcting Code memory)

- RAM memory which checks and corrects corrupted data automatically (looks like a normal RAM stick)

Secondary Storage

M.2 SSD

- M.2 Solid State Drives have slots directly on the motherboard, this way they dont need any cable which is a speed bottleneck for SSDs

- current [2020-02-17] M.2 slot speed (B key = PCIe X2) and (M key = PCIe X4)

RAID & SLED

- SLED (Single Large Expensive Disk) basically refers to one large storage device

- RAID (Redundant Array of Idependent Disks) Storage Technology using multiple disks, it is used for data redundancy, performance improvement, or both (RAID is not data backup)

Network Hardware

Modem

- a modem is basically a modulator / demodulator, it converts the analog signal to digital signal and vice versa

- the modem is required if the ISP provides analog signal

Hub

- creates a LAN (Local Area Network), connects local computers together

- simply sends the received packets to all other computers, this is impractical because it unnecessarily sends the message to unwanted destinations and charges the network

Switch

- creates a LAN (Local Area Network), connects local computers together

- considered "inteligent" because it recognizes computers on the network through MAC addresses, so it sends the message only to the destination

- can create multiple logical networks, so we can have multiple separated networks on the same Switch, these locical networks are called VLANs (Virtual Local Area Network),

Router

- a Router 'routs' network trafic between two (or more) networks, it creates a subnet and routs data on it

- it has 2 (or more) IP addresses, it receives the packetes on one side processes them and sends towards to the other network, this protocol is called NAT/PAT (Network Address Translation / Port Address Translation)

- it is the 'Gateway' between LAN and WAN usually

- it does port forwarding = incoming traffic from WAN to a specific port is redirected to one specific address inside the LAN

SOHO Router (Small Office Home Office)

- Soho Routers are designed for Homes and Small Offices, the are practical because the have built in DNS, DHCP, Modem, Switch and Firewal, however they are not as robust and safe as commercial routers

NAS and SAN (Network Attached Storage device) (Storage Area Network)

NAS

- storage device attached to the network, so all devices can connect to it on the network

SAN

- entire separate network for storage with data redundancy, backup and safety features
