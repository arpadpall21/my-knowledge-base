Description and Demonstration :

- Vite is a general frontned build tool and development server (we can use to build Vanilla Frontend, React, Vue)

- for example to build a React project:

- scaffolds the React project (npx create-vite -> project creation wizzard)

- install dependencies (npm install)

- start dev server (npm start)

- using Vite has the following development speed advantages over the tradional Create React App tool:

- doesn't use Webpack, instead uses the browsers native ES module loader (which make it much faster because there's no bunding)

- Hot Module Replacement, only replaces code parts that are changed

- In general it uses faster build tools that RCA

Configuration [link]

- Vite supports powerfull configurations and plugins through the vite.config.js|ts file (ex: we can configure the base URL for the application)

Example :

create React project 

    $ npx create-vite
      ? Project name: .                 // . means use current directory
      ? Select a framework:             // prompts which project to scaffold
        Vanilla
        Vue
        React
        ...
    
    $ npm install                       // installs dependencies
    $ npm run dev                       // start dev server

config file 

// vite.config.ts (file) ------------------------------
    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'
    
    // https://vitejs.dev/config/
    export default defineConfig({
      plugins: [react()],             // react project uses react plugin (scaffolded for us)
      base: '/app/',                  // app will serverd on this base URL in prod and dev as well
      server: {
        port: 3000                    // dev server port 
      }
    })
