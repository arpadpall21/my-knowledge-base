Remember This :

- Consistent style guide is important -> Consisentcy within a project is more important -> Consistency within a module is the most important

General Roules

- always use spaces instead of tabs

- lines should be limited up to 100 characters (not recommended but I prefer 120 characters)

- use single quotes ' for strings (use dobule quotes to avoid escaping (bad -> 'It\'s' good -> "It's"))

- long strings should not be broken, even if it exedes the allowed line length

- files should be terminated with a single new line character
