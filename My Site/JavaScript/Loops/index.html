The while loop 

                                                                                          1  1  3  3  1  0.10     
SYNTAX:     while(cond

                
 - conditional statement evaluated before each iteration 

                
 - if evaluated true runs a pass, otherwise stops the loop 

            
) {                                                             // if the condition (cond) evaluates true runs the code in the block {}    
                // some code...
            }
            
            
  // do ...while ----------------------------------------------------------------------   1  1  4  4  1  0.10      
            do {                                                                       // runs the code in the block once, then if the condition (cond) evaluates true runs the block again and again...    
                // some code...
            } while(cond

                
 - conditional statement evaluated before each iteration except the 1st iteration 

                
 - if evaluated true runs a pass, otherwise stops the loop 
  
            
)
DEMO 
    let i = 0;
    
// while ------------------------------------------------------------------------------
    while (i < 5){                                                                     // checks the condition before every pass 
        i++                                                                            // code insdie the block runs 5 times 
    } 
    
// do...white ------------------------------------------------------------------------- 
    do {
        i++                                                                            // -! the first pass is garanteed to run before the condition is checked

The break / continue and label statements

- the label statement provides an identifier for a loop, these labels only usable by break and continue statements to identify whitch statement to apply

- the break statement (can break any {} code blocks), prevents all further code to be executed in the current block and blocks the loop

- the continue statement (only usable by loops), prevents all further code to be executed in the current block but it desn't block the loop 

SYNTAX : 
  // break ----------------------------------------------------------------------------   1  1  3  3  1  0.10      
            for (;;){ 
                break labelName  
                
                // some code...                                                        // all further code in the block won't be executed and the loop is stoped    
            }
    
    
  // continue -------------------------------------------------------------------------   1  1  4  3  1  0.10       
            for (;;){
                continue labelName  
                
                // some code...                                                        // all further code after the continue statement is skipped but the loop continues executing  
            }
    
    
  // label ----------------------------------------------------------------------------   1  1  4  4  1  0.10      
            labelName: for (;;){                                                       // labels the loop     
                break labelName                                                        // if a label is declared the break is valid on that loop only    
                
                // some code...   
            }
            
            labelName: for (;;){
                continue labelName                                                     // if a label is declared the continue is valid on that loop only      
                
                // some code...  
            }
Example : 
    for(let i = 0; i < 10; i++){                                                       // loop runs only once up to the 'break' statement    
        console.log( 'before break' );                                                 // code executed once   
        
        break;                                                                         // breaks the loop 
        
        console.log( 'after break' );                                                  // code never executed 
    }
    
    
    for(let i = 0; i < 10; i++){                                                       // loop runs 10 times but the block executed only up to the continue statement    
        console.log( 'before continue' );                                              // code never executed 
        
        continue;                                                                      // skips all further code in the block but don't break the loop   
        
        console.log( 'after continue' );                                               // code never executed   
    }
    
// label statement --------------------------------------------------------------------
    outer: for (i = 0; i < 10; i++) {                                                  // outer loop labeled "outer" 
        for (j = 0; j < 5; j++) { 
            if (i === 4) {
                break outer;                                                           // the outer will be broken at 4th iteration   
            }
            
            console.log('i = ' + i + ', j = ' + j);
        }
    }

break VS continue

CODE : 
<script>
    var i, k = 0;
    var breakMemory = "";
    var continueMemory = "";
    
    for (i = 0; i < 10; i++){
        if (i === 5) {continue;}
        breakMemory += i + " / ";
    }
// ------------------------------------------------------
    while (k < 10){
        if (k === 5) {break;}                   // the "continue" statement cannot be used here because that will skip all the remaining code                                  
        continueMemory += k + " / ";
        k++;
    };
    document.write(
        "continue = " + breakMemory + "<br>" +
        "break = " + continueMemory
    );
</script>
