Remember This :

Description and Demonstration :

- JavaScript Date object tracks time in UTC internally, but typically accepts input and produces output in the local time of the computer it's running on. It has very few facilities for working with time in other time zones

- the internal representation of a Date object is a single number 'timestamp', it represents UTC milliseconds elapsed since 1970-01-01 00:00:00 UTC (without regard to leap seconds)

- the new date object underneath creates a timestamp which is counted from January 1st, 1970, 00:00:00 UTC (called "Zero Time")

- we can get UTC or local time from the date object (date objects have no timezone support)

ISO 8601 date standard

- the ISO 8601 date standard is a Universal Date Standard format, follows a strict standard, exemple: (2017-08-22T12:24:40Z) 

    YYYY-MM-DDTHH:MM:SS                          
    YYYY-MM-DDTHH:MM:SSZ                                                       // UTC time (Z stands for "zero meridian")
    YYYY-MM-DDTHH:MM:SS+|-HH:MM                                                // UTC time with adjusted time-zone
DEMO 
    1998                                                                               // the year is required only!
    1998-06                    
    1998-06-20                 
    1998-06-20T23:20:06        
    1998-06-02T23:20:06Z                                                               // UTC time 
    1998-06-02T23:20:06+02:00                                                          // time with adjusted time zone (relative to UTC)

The Date() / Date.now() / Date.parse() / Date.UTC() / valueOf() methods

DEMO 
    Date();                                                                             // -> 'Mon Mar 10 2025 11:29:04 GMT+0100 (Central European Standard Time)'
    
    new Date(1403256444443);                                                           // -> "Fri Jun 20 2014 12:27:24 GMT+0300 (heure d’été d’Europe de l’Est)"    // milliseconds (zero-time = counterd from January 1st, 1970, 00:00:00 UTC)   
    new Date(2014, 05, 20, 12, 27, 24);                                                // -> "Fri Jun 20 2014 12:27:24 GMT+0300 (heure d’été d’Europe de l’Est)"    // numbers          // -! 5 means June (months are indexed 0 = January)   
    new Date("2014-06-20T12:27:24");                                                   // -> "Fri Jun 20 2014 12:27:24 GMT+0300 (heure d’été d’Europe de l’Est)"    // ISO time 
    new Date("Tue Jun 20 2014 15:27:24 GMT+0300 (heure d’été d’Europe de l’Est)")      // -> "Fri Jun 20 2014 15:27:24 GMT+0300 (heure d’été d’Europe de l’Est)"    // date string 
    
    new Date("2014-06-20T12:27:24Z");                                                  // -> "Fri Jun 20 2014 15:27:24 GMT+0300 (heure d’été d’Europe de l’Est)"    // UTC time specified so converted back local time 
    
    Date.now();                                                                       // -> 1741602898564
    
    Date.parse("2018-06-07T19:56:46.567");                                           // -> 1528390606567      // ISO 8601 datetime parsed to milliseconds 
    Date.parse('Thu Oct 22 2020 13:50:46 GMT+0300 (Eastern European Summer Time)')   // -> 1603363846000      // datetime string parsed to milliseconds UTC      
    
    Date.UTC(2018, 06, 07, 19, 56, 46, 567);                                         // -> 1530993406567      // passed UTC time converted to milliseconds
    Date.UTC(2025);                                                                  // -> 1735689600000      // 2025 jan 1st
    
    const now = new Date();
    now.valueOf();                                                                    // -> 1741605077399

The toUTCString() / toString() / toLocalString() / toTimeString() / toLocaleTimeString / toDateString() / toLocaleDateString() methods

DEMO 
    const now = new Date();
    
    now.toUTCString();                    // -> 'Mon, 10 Mar 2025 12:06:27 GMT'
    now.toString();                       // -> 'Mon Mar 10 2025 12:53:13 GMT+0100 (Central European Standard Time)'
    now.toLocaleString();                 // -> '3/10/2025, 12:53:13 PM'
    
    now.toTimeString();                   // -> '12:53:13 GMT+0100 (Central European Standard Time)'
    now.toLocaleTimeString();             // -> '12:53:13 PM'
    
    now.toDateString();                   // -> 'Mon Mar 10 2025'
    now.toLocaleDateString();             // -> '3/10/2025'

Getter Methods getDate() / getDay() / getFullYear() / getHours() / getMilliseconds() / getMinutes() / getMonth() / getSeconds() / getTime() / getUTCDate() / getUTCDay() / getUTCFullYear() / getUTCHours() / getUTCMilliseconds() / getUTCMinutes() / getUTCMonth() / getUTCSeconds() / getTimezoneOffset()

DEMO 
    var myTimeNow = new Date();                                                        // this the locale time right now 
    myTimeNow;                                                                         // -> "Thu Apr 25 2019 18:02:30 GMT+0300 (heure d’été d’Europe de l’Est)"
        
    myTimeNow.getFullYear();                                                           // -> 2019
    myTimeNow.getMonth();                                                              // -> 3              // -! months are indexed (0 = January / 3 = April)   
    myTimeNow.getDate();                                                               // -> 25 
    myTimeNow.getDay();                                                                // -> 4              // -! weekdays are indexed (0 = Sunday / 4 = Thursday)    
    myTimeNow.getHours();                                                              // -> 18
    myTimeNow.getMinutes();                                                            // -> 2
    myTimeNow.getSeconds();                                                            // -> 30
    myTimeNow.getMilliseconds();                                                       // -> 0
    
// UTC --------------------------------------------------------------------------------
    myTimeNow.getUTCFullYear();                                                        // -> 2019
    myTimeNow.getUTCMonth();                                                           // -> 3              // -! months are indexed (0 = January / 3 = April)   
    myTimeNow.getUTCDate();                                                            // -> 25 
    myTimeNow.getUTCDay();                                                             // -> 4              // -! weekdays are indexed (0 = Sunday / 4 = Thursday)    
    myTimeNow.getUTCHours();                                                           // -> 15         
    myTimeNow.getUTCMinutes();                                                         // -> 2 
    myTimeNow.getUTCSeconds();                                                         // -> 30
    myTimeNow.getUTCMilliseconds();                                                    // -> 0
    
// ------------------------------------------------------------------------------------
    myTimeNow.getTime();                                                               // -> 1556204550000  // millisecond value of the date object (UTC time since Zero Time)   
    myTimeNow.getTimezoneOffset();                                                     // -> -180           // time difference between the local and UTC time (relative to local time) (removing 180 minutes from the local time we are getting the UTC time)

    myTimeNow.setUTCMilliseconds(554),        myTimeNow;                               // -> "Thu Apr 25 2019 18:02:30 GMT+0300 (heure d’été d’Europe de l’Est)"
                                              
    myTimeNow.setTime(1556204550000),         myTimeNow;                               // -> "Thu Apr 25 2019 18:02:30 GMT+0300 (heure d’été d’Europe de l’Est)"    // milliseconds to datetime 
                                              
// offset values ----------------------------------------------------------------------
    myTimeNow.setMonth(13),                   myTimeNow;                               // -> "Tue Feb 25 2020 18:02:30 GMT+0200 (heure normale d’Europe de l’Est)"  // 11 = December / 13 = Febuary of next year    
    myTimeNow.setMonth(-1),                   myTimeNow;                               // -> "Tue Dec 25 2018 18:02:30 GMT+0200 (heure normale d’Europe de l’Est)"  // 0 = January / -1 = December of previous year   
    
    myTimeNow.setHours(0),                    myTimeNow;                               // -> "Thu Apr 25 2019 00:02:30 GMT+0300 (heure d’été d’Europe de l’Est)"    // 0 first hour of today     
    myTimeNow.setHours(-24),                  myTimeNow;                               // -> "Wed Apr 24 2019 00:02:30 GMT+0300 (heure d’été d’Europe de l’Est)"    // 0 hour of the previous day   
    myTimeNow.setHours(80),                   myTimeNow;                               // -> "Sun Apr 28 2019 08:02:30 GMT+0300 (heure d’été d’Europe de l’Est)"    // 3 days an 8 hours ahead
