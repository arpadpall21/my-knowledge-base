Description and Demonstration :

- the Iterator object is an Array-like object which holds its items in strict order, we can access its items through the next() method or by using the for... of loop

- the Iterator object can be created by various tools, like generator functions or by various methods (ex: mapObj.entries()) 

    interatorObj
    
    interatorObj.next()                    // gets the next memeber as { value: val, done: bool }
    
    var numberMap = new Map([[1, "egy"], [2, "ketto"], [3, "harom"]]);
    var numberEntries = numberMap.entries();
    
    numberEntries.next();                                 // -> { value: [1, 'egy'], done: false }
    numberEntries.next();                                 // -> { value: [2, 'ketto'], done: false }
    numberEntries.next();                                 // -> { value: [3, 'harom'], done: false }
    numberEntries.next();                                 // -> { value: done: true }
    
    for (let member of numberEntries) {
      member;                                           // -> [ 1, 'egy' ]  |  [ 2, 'ketto' ]  |  [ 3, 'harom' ]
    }
