Primitives are immutable VS Objects are mutable

- Objects are addressed by reference not by value therefore they are mutable

- the same underlying object can be referenced by multiple variables

- a primitive can be referenced by a single variable 

// pbjects mutation -------------------------------------------------------------------
    var obj1 = {p1:'origVal'};                  
    var obj2 = obj1;                                                                   // both variables ('obj1' and 'obj2') reference the same underlying object from now 
    
    obj2.p1 = 'modVal';                                                                // the same underlying object is modified 
    obj1.p2 = 'newVal';
    
    obj1;                                                                              // -> {p1:'modVal', p2:'newVal'}     // same object 
    obj2;                                                                              // -> {p1:'modVal', p2:'newVal'}
    
// primitive unmutability -------------------------------------------------------------
    var p1 = 21;
    var p2 = p1;                                                                       // a copy is made so each variables have their unique primitives now    
    
    p1 = 11;                                                                           // each variables hold their unique values  
    p2 = 31;
    
    p1;                                                                                // -> 11                             // different primitives   
    p2;                                                                                // -> 31

Object Properties

- JavaScript Objects are containers for key/value pairs, the key is called property, property order is not garanteed in regular Objects

- in JavaScript there are two types of property accessor, the . (dot) and [] (square brackets)

- object keys are strings, any data passed as object key is converted to string 

SYNTAX:     objectName.propertyName               
            objectName['propertyName'] | objectName["propertyName"]                    // double or single qoute doesn't matter
DEMO 
    var myObj = {'prop1':'val_1', prop2:'val_2', 32:'val_3', true:'val_4', [2]:'val_5'} // -! non-string keys are converted to string
    
    myObj.prop1;                                                                       // -> 'val_1'
    myObj['true'];                                                                     // -> 'val_4'
    myObj["2"];                                                                        // -> 'val_5'    // the passed array is converted to string ([2] => '2')
        
// using variable value as object key -------------------------------------------------
    var x = 'key1'
    var y = 'key2';
    
    var myObj = {[x]:'val-1'};                                                         // -! ES6 
    myObj[y] = 'val-2'                                                                 // ES5
        
    myObj;                                                                             // -> {key1:'val-1', key2:'val-2'}

Getter and Setter properties

- it is possible to define a property behaviour when it gets or sets 

SYNTAX :    var myObj = {
                         get get_propId(){ return <code>; },             <-- getter property
                         set set_propId(value){ value<code>; }           <-- setter property
                        } 
            
            myObj.get_propId;                                            // called as property, function executed and code returned 
            myObj.set_propId = passValue;                                // set property, function executed and value passed
DEMO 
    var rabbits = {rabbit_A:"Lilu", rabbit_B:"sooty", rabbit_C:"rose",
                   get getAllRabbits(){ return this.rabbit_A + ", " + this.rabbit_B + ", " + this.rabbit_C},    // getter property set 
                   set setAllRabbits(value){ this.rabbit_A = value;
                                             this.rabbit_B = value;
                                             this.rabbit_C = value;}
                  };
    rabbits.getAllRabbits;              // returns "Lilu, sooty, rose" (getter property executes the function and returns the values)
    rabbits.setAllRabbits = "sweety";   // all property values are updated (setter property executes the function, set passed value to the targeted properties)
        
    rabbits.getAllRabbits;              // returns "sweety, sweety, sweety" (all values are updated)
