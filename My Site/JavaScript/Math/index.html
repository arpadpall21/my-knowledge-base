Math.fround(val:any) - returns the closest 32bit single precision floating point representation of val
Math.clz32(val:any) - returns the number of leading zero bits when val is represented as binary 32bit signed integer

ex:

- 8452 is represented as 00000000000000000010000100000100 32bit unsigned integer therefore it has 18 0bits before the first 1bit (Math.clz32(8452) returns 18)

- 9999999 is represented as 00000000100110001001011001111111 32bit unsigned integer therefore it has 8 0bits before the first 1bit (Math.clz32(9999999) returns 8)

Description and Demonstration :

- the Math buit-in object alows us to perform mathematical tasks on numbers but not on bigints

- the Math object is not a constructor

The random() method

DEMO 
    Math.random();                                                                     // -> 0.9563975899807617     // number between 0 to 0.99999999999999999

The floor() / ceil() / round() methods

DEMO 
// floor() / ceil() -------------------------------------------------------------------
    Math.floor(2.54);                                                                  // -> 2
    Math.ceil(1.14);                                                                   // -> 2
    
// round() ----------------------------------------------------------------------------
    Math.round(3.49);                                                                  // -> 3
    Math.round(3.50):                                                                  // -> 4

The min() / max() methods

DEMO 
// min() ------------------------------------------------------------------------------
    Math.min(0, 54, 7);                                                                // -> 0
    Math.min(NaN, 45, -75);                                                            // -> NaN        
    Math.min(-7854, -Infinity, 447);                                                   // -> -Infinity
    
// max() ------------------------------------------------------------------------------
    Math.max(-45, -47, -31, -12);                                                      // -> -12
    Math.max(false, 32, [56]);                                                         // -> 56         // all arguments can be converted to a number 
    Math.max(1, 2, 'three');                                                           // -> NaN        // cannot convert all arguments to a number, NaN returned

The pow() / sqrt() / cbrt() methods

DEMO 
// pow() ------------------------------------------------------------------------------
    Math.pow(2, 5);                                                                    // -> 32 
    Math.pow([2], '5');                                                                // -> 32                     // arguments are converted to numbers 
    
// sqrt() -----------------------------------------------------------------------------
    Math.sqrt(49);                                                                     // -> 7 
    Math.sqrt('1024');                                                                 // -> 32                     // value converted to a number 
    
// cbrt() -----------------------------------------------------------------------------
    Math.cbrt(343);                                                                    // -> 7 
    Math.cbrt('32768');                                                                // -> 32                     // value converted to a number

The sin() / cos() / tan() methods

Example : 
    Math.sin(0.3);                                                                     // -> 0.29552020666133955
    Math.cos(1.1);                                                                     // -> 0.4535961214255773
    Math.tan('0.48');                                                                  // -> 0.520610844191258      // argument converted to a number

The abs() / trunc() / sign() / fround() / clz32() methods

DEMO 
// abs() ------------------------------------------------------------------------------
    Math.abs('-1');                                                                    // -> 1
    Math.abs(-2);                                                                      // -> 2
    Math.abs(null);                                                                    // -> 0
    Math.abs('');                                                                      // -> 0
    Math.abs([]);                                                                      // -> 0
    Math.abs([2]);                                                                     // -> 2
    Math.abs([1,2]);                                                                   // -> NaN
    Math.abs({});                                                                      // -> NaN
    Math.abs('string');                                                                // -> NaN
    Math.abs();                                                                        // -> NaN
    
// trunc() / sing() -------------------------------------------------------------------
    Math.trunc('32.32');                                                               // -> 32                     // returns the integer part only  
    Math.trunc([142]);                                                                 // -> 142
    Math.trunc(-415.22);                                                               // -> -415
    
    Math.sign([84552.21]);                                                             // -> 1                      // positive value 
    Math.sign(-441.2);                                                                 // -> -1                     // negative value 
    Math.sign(0);                                                                      // -> 0
    
// fround() ---------------------------------------------------------------------------
    Math.fround(12.12);                                                                // -> 12.119999885559082     // closest 32bit single precision floating point representation    
    Math.fround(1.2);                                                                  // -> 1.2000000476837158
    
// clz32() ----------------------------------------------------------------------------
    Math.clz32(45);                                                                    // -> 26                     // 45 has 26 leading 0bits in 32bit unsigned integer binary representation     
        // 00000000000000000000000000101101                                                // 32bit signed integer binary representation of 45 
    Math.clz32(5548796);                                                               // -> 9
        // 00000000010101001010101011111100                                                // 32bit signed integer binary representation of 5548796
    Math.clz32(-125);                                                                  // -> 0                      // negative values awlays start with 1bit 
        // 11111111111111111111111110000011                                                // 32bit signed integer binary representation of -125
