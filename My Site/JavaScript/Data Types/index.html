Description and Demonstration:

- JavaScript data is dynamic (The same variable can be used to store different data)

- JavaScript has 7 Primitive Data types, any other data is considered as complex data (object)

Primitive Data

- a primitive data is also constructed with the corresponding Object Constructor, they themselves do not have any properties or methods but they can still use their Constructor's properties and methods

- in JavaScript primitives are immutable (we cannot have multiple references to the same primitive (like objects can)) and executes faster than objects

Primitives in JavaScript [Ordered: Alphabetically]
Data Description
string - represents a textual data
number - represents a numerical data
boolean - represents a logical data with two possible value true or false
undefined - means value without any real data
null - means no value (something what does not exist)
symbol - 6 unique object key 38 36 25 12 9 0.12
bigint - 11 big integer value 67 68 54 79 14 10.4.0

DEMO 
// undefined VS null ------------------------------------------------------------------
    var x;                                                                             // -> undefined              // undefined = data without value  
    var x = null;                                                                      // -> null                   // null = no data at all   
    
// primitives are immutable  ----------------------------------------------------------
    var str1 = "myString"; 
    var str2 = str1;                                                                   // primitive data is immutable (str2 variable has a copy of str1 variable's primitive)   
    
    str1;                                                                              // -> "myString";
    str2;                                                                              // -> "myString";
    
    str1 = "newString";
    
    str1;                                                                              // -> "newString";
    str2;                                                                              // -> "myString";            // each variable has its own primitive

Complex Data

- in JavaSript any non primitive data is considered as a Complex data (an object)

- a Complex data is basically composed of multiple primitive data

DEMO 
// these are all Complex datas -------------------------------------------------------- 
    {model:"A4", mark:"Audi", year:2010}                                               // an object is an object
    ["A4", "Audi", 2010]                                                               // an array is an object
    new Date()                                                                         // a date is an object
    function audi(){}                                                                  // a function is an object

Data Type checking and Conversion typeof / valueOf()

- if we operate on a "wrong" data type JavaScript will try to convert the value to a "right" data type, this can produce unexpected results sometimes

- we can convert a data type by using Constructors (ex: String()) or various methods, Careful! some conversion types might be tricky see conversion table above

DEMO 
// primitive data ---------------------------------------------------------------------
    typeof "myString"                                                                  // -> "string"
    typeof 21                                                                          // -> "number"
    typeof true                                                                        // -> "boolean"
    typeof undefined                                                                   // -> "undefined" 
    typeof null                                                                        // -> "object"       // !! this is a bug in JavaScript it should be "null"
    typeof Symbol                                                                      // -> "symbol"       // ES6
    typeof 21n                                                                         // -> "bigint"       // ES11
    
// complex data -----------------------------------------------------------------------
    typeof function myFunction(){}                                                     // -> "function"     // functions are complex datas but returned as "function"
    typeof {}                                                                          // -> "object"       // any other data will be returned as "object"

- data conversion 

// convert to string ------------------------------------------------------------------
    String(21 + 21);                                                                   // -> "42"           // using constructor 
    (21 + 21).toStirng;                                                                // -> "42"            
    String(Date());                                                                    // -> "Thu Jul 17 2014 15:38:19 GMT+0200 (W. Europe Daylight Time)"  // convert Date object to string 
    
    var number = 14;
    number.toPrecision(2);                                                             // -> "2"            // number method convert numbers to string 
    number.toFixed(1);                                                                 // -> "14.2"         
    number.toExponential();                                                            // -> "1.4e+1"
    
// convert to number ------------------------------------------------------------------ 
    Number('3.14');                                                                    // -> 3.14
    Number('21 21');                                                                   // -> NaN
    Number('');                                                                        // -> 0
    Number('text');                                                                    // -> NaN
    Number (new Date());                                                               // -> 1510859509048  // milliseconds after 1970 1st of January
    
    parseInt('3.14 is the pi Number');                                                 // -> 3              // returns the first integer of a string 
    parseFloat('3.14 is the pi Number');                                               // -> 3.14           // returns the first floating point number of a string 
    
// convert to boolean ----------------------------------------------------------------- 
    Boolean(5 == 5)                                                                    // -> true
    Boolean(3 > 1)                                                                     // -> true
    Boolean(0)                                                                         // -> false
    Boolean("false")                                                                   // -> true           // -! Careful!
