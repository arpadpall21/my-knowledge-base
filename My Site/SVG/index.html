Description and Demonstration :

- the SVG (Scalable Vector Graphics) is an XML languages developped by W3C since 1999, the version 1.1 (second edition) is the recommendation right now but version 2.0 is under heavy developement [2019-01-04]

- SVGs are high quality no matter the resolution and every element can be animated but they load slowly due to their size

- file extension is .svg / .svgz (compressed format), MIME type is image/svg+xml

- we can use programs like Photoshop and Inkscape (download InkScape) to draw SVG files for us

Display SVG on web page

- SVG files can be displayed in <img> / <iframe> / <object> elements

- in HTML5 we can directly embed SVG in the webpage between the <svg> tags 

    <img src="mySVG.svg" title="mySVG"> 
    <iframe src="mySVG.svg"></iframe> 
    <object data="mySVG.svg" type="image/svg+xml"></object>
    
    <svg xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink" preserveAspectRatio="xMidYMid" width="200" height="200" viewBox="0 0 200 200">
        <SVG code here>
    </svg>
