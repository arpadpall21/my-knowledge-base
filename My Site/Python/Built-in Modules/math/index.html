Description and Demonstration :

- mark the math module supports only floating points and integers, for complex number types use the cmath module

DEMO

- gcd() (Greatest Common Divisor) 

    math.gcd(2, 4)        // -> 2
    math.gcd(4, 18)       // -> 2
    math.gcd(25, 84)      // -> 1

- lcm() (Lowest Common Multiple) 

    math.lcm(6, 4)        // -> 12
    math.lcm(6, 3)        // -> 6

- reminder() 

    math.remainder(10, 3)   // -> 1.0
    math.remainder(5, 1.33) // -> -0.3200000000000003

- comparing nan values 

    math.nan == math.nan    // -> False     // 2 Not A Number values are not equal 
    math.nan is math.nan    // -> True      // but Python stores these value on the same memory location
