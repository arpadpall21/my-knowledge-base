Description and Demonstration :

- the os module provides functionalities for 3 important things:

- environment variable funcionalities

- file system operations

- process functionalities (for child processes see the subprocess)

- from V.3.7 Python uses UTF-8 encoding and ignores locale encoding

Example :

- umask() 

    os.umask(0O222)                     // -> 18              // new umask -> 222
                                                              // old umask -> 022  (decimal 18 -> octal 22)

- execl() 

    import os
    import sys
    
    os.execl(sys.executable, sys.executable, './test_2.py')     // replaces the current process by sys.executable ./test_2.py (uses this process' id to start the new process)

- fork() 

    import os
    
    id = os.fork()                              // forks a new process by using the current state of the current process 
    
    if id != 0:
        print( "Hello from parent process" )    // this code runs in the parent process    
        print( id )                             // id of the child process
    elif id == 0:
        print( "Hello from child process" )     // this code runs in the child process
