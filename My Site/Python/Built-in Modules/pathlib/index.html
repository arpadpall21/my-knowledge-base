Concrete Path Path [Ordered: Alphabetically]
Method / Property Description
Path(path:str, ...

- mutiple path segmenst can be passed (if no path is passed the current directory is assumed)

) 		- creates and returns a concretePath object
- on instantiation does not check if the path exist on the system

concretePath.chmod(chmod:octal, follow_symlinks=bol) - changes the access permission of the file|directory
concretePath.lchmod() - same as chmod() but if the path points to a symbolic link than the simbolic link's access permissions are modified (not the original file's)
concretePath.glob(globPattern) - where globPattern matches returns all path segments in a genObj (as concretePath)
concretePath.owner() - returns the owner of the concretePath
concretePath.group() - returns the group of the concretePath
concretePath.exists() - returns True if the file|directory exists, False otherwise
concretePath.is_dir() - returns True if the path is a directory, False otherwise
concretePath.is_file() - returns True if the path is a file, False otherwise
concretePath.is_mount() - returns True if the path is a mount point, False otherwise
concretePath.is_symlink() - returns True if the path is a symbolic link, False otherwise
concretePath.is_socket() - returns True if the path is a socket, False otherwise
concretePath.is_fifo() - returns True if the path is a FIFO, False otherwise
concretePath.is_block_device() - returns True if the path is a block device, False otherwise
concretePath.is_char_device() - returns True if the path is a character device, False otherwise
concretePath.touch(args...) - creates a new file (file that is specified in the Path(path:str) constructor)
concretePath.mkdir(args...) - creates a new directory (directory that is specified in the Path(path:str) constructor)
concretePath.unlink() - removes the targeted file
concretePath.rmdir() - removes an empty directory (fore removing directory structure use shutil.rmtree())
concretePath.read_text(encoding=enc:str) - returns the content of the file as a string
concretePath.read_bytes() - returns the full content of the file as a bytes
concretePath.write_text(str, encoding=enc:str) - writes text to the file overwrites the file (creates the new file if doesn't exist)

concretePath.open(args) - opens the targeted file for reading (used as context manager) (does the same as open())
concretePath.hardlink_to(filePath:str) 3.10

concretePath.replace(newName:str)
concretePath.rename(newName:str) 		- renames the targeted file|directory (will move the file from -> to) 3.8 returns a new concretePath

concretePath.samefile(path:str|pathObj) - returns True if concretePath points to the same file as path

Useful Links :

pathlib (docs.python.org)

Remember This :

- for lower level file system access use the os.path module

- for complex file system operations (like: copying, removing entire fs structures) use the shutil module (pathlib cannot remove an entire directory structure)

Description and Demonstration :

- the pathlib module provides an object oriented file system access, for both POSIX and Windows paths

- path objects provided by the pathlib module can be used anywhere wher an os.PathLike object can be used

- pathlib provides us with 2 different path types, these are basically for access level distinctions

- Pure Path provides only computational operations (no file system access) (used when only file system computation is needed)

- Concrete Path provides full file system access operations subclass of Pure Path

PurePath

The / operator

DEMO 
    from pathlib import PurePath
    
    
    current_dir = PurePath()
    current_file = PurePath(__file__)
    root = PurePath('/')
    
    
    json_path = PurePath() / 'test.json'                        // test.json in the current directory  
    
    json_in_other_dir = PurePath('../other_dir/test.json')      // all these do the same
    json_in_other_dir2 = PurePath() / '../other_dir/test.json'
    json_in_other_dir3 = PurePath() / '..' / 'other_dir' / 'test.json'

The parts / parent / root / name / suffix / suffixes / stem / is_absolute() / is_relative_to() / join_path() / match() properties and methods

DEMO 
// parts / parent / root -----------------------------------------------
    print( PurePath('/var/log').parts )                   // -> ('/', 'var', 'log')
    print( PurePath('/var/log').parent )                  // -> '/var'
    print( PurePath('/var/log').root )                    // -> '/'
    
    
// name / suffix / suffixes / stem -------------------------------------
    print( PurePath('/var/log/log.py').name )             // -> 'log.py'
    print( PurePath('/var/log/log.py').suffix )           // -> '.py'
    print( PurePath('/var/log/log.log.py').suffixes )     // -> ['.log', '.py']
    print( PurePath('/var/log/log.py').stem )             // -> 'log'
    
    
// is_absolute() / is_relative_to() ------------------------------------
    print( PurePath('/var/log/log.py').is_absolute() )    // -> True
    print( PurePath('./log/log.py').is_absolute() )       // -> False
    
    org_path = PurePath('/var/log/test')
    print( org_path.is_relative_to('/var') )              // -> True
    print( org_path.is_relative_to('/log') )              // -> False
    
    
// join_path() ---------------------------------------------------------
    print( PurePath('/var').joinpath('log').joinpath('log.py') )   // -> /var/log/log.py (purePath obj)
    
    
// match() -------------------------------------------------------------
    print( PurePath('/var/log/log.py').match('*.py') )        // -> True
    print( PurePath('/var/log/log.py').match('**/log/**') )   // -> True

Path (Concrete Path)

The cwd() / home() methods

DEMO 
    from pathlib import Path
    
    
    print( Path.cwd() )            // -> current working dir (path obj)
    print( Path.home() )           // -> user's home dir (path obj)

The stat() / lstat() methods

DEMO 
    from pathlib import Path
    
    
    print( Path('test.json').stat() )                   // -> os.stat_result (object)
    
    print( oct(Path('test.json').stat().st_mode) )      // -> 0o100644
    Path('test.json').chmod(0o444)
    print( oct(Path('test.json').stat().st_mode) )      // -> 0o100644

The touch() / unlink() / mkdir() / rmdir() methods

DEMO 
    from pathlib import Path
    
    
    Path('new_file.txt').touch()      # creates the new file 
    Path('new_file.txt').unlink()     # removes the created file
    
    Path('new_dir').mkdir()           # creates the new directory
    Path('new_dir').rmdir()           # removes the created directory

The read_text() / read_bytes() / write_text() / write_bytes() methods

DEMO

    Path('test.txt').write_text('some content')
    Path('test2.txt').write_bytes(b'some content')

The hardlink_to() / symlink_to() / readlink()

DEMO 
    from pathlib import Path
    
    
// symlink ------------------------------------------------------
    Path('symlink.json').symlink_to('test.json')          # creates a symlink to test.json
    print( Path('symlink.json').readlink() )              # -> test.json (obj)        # resolves where the symlink is pointing to    
    
    
// hardlink -----------------------------------------------------
    print( Path('test.json').stat().st_nlink )            // -> 1     one hardlink
    
    Path('test_1.json').hardlink_to('test.json')          # creating hardlinks 
    Path('test_2.json').hardlink_to('test.json')
    Path('test_3.json').hardlink_to('test.json')
    
    print( Path('test.json').stat().st_nlink )            // -> 4

The replace() / rename() methods

DEMO 
    from pathlib import Path
    
    
    Path('./new_dir/test.json').rename('new_name.json')     // -> moves and renames the file
    Path('./new_dir/test.json').replace('new_name.json')    // -> moves and renames the file

The absolute() / resolve() methods

DEMO 
    from pathlib import Path
    
    
    Path.cwd().absolute()                      // -> absolute path (purePath obj)
    Path('symlink.json').absolute().parts      // -> [..., 'symlink.json']   # the symlink is the last segment (not the original file)
    
    Path('symlink.json').resolve().parts       // -> [..., 'org.json']      # the original file is the last segment (to which the symlink points)

The samefile() method

DEMO 
    from pathlib import Path
