Conditional Statements

The if...else statement 

    if cond:                                                                            // if the cond evaluated truthy executes the below code(s)   
        codeBlock
    elif cond:                                                                          // if the above cond is falsy this cond is evaluated 
        codeBlock
    ...                                                                                 // we can have multiple elif statements    
    
    else:                                                                               // if all the above cond are false the below code is executed    
        codeBlock
    
// -------------------------------------------------------------------------------------
    if cond: exp                                                                        // if cond is truthy then exp is executed (shorthand statement)    
    
// -------------------------------------------------------------------------------------
    exp1 if cond esle exp2                                                              // if cond is truthy then exp1 is executed (ternary operator | conditional expression)    
                                                                                        // if cond is falsy then exp2 is executed
DEMO 
    x = 5
    
    if x > 7:
        print('if')
    elif x > 5:
        print('elif')
    else:
        print('else')                                                                   // 'else' printed
    
// -------------------------------------------------------------------------------------
    if 0 == 0: print('OK')                                                              // 'OK' printed 
    
    if 0 == 1: print('OK')                                                              // 'OK' not printed  
    
// -------------------------------------------------------------------------------------
    x = 10
    pirnt('less than 10') if x < 10 else print('more than 10')                          // 'less than 10' is printed 
    
    x = 11
    pirnt('less than 10') if x < 10 else print('more than 10')                          // 'more than 10' is printed

The match statement 3.10

- the match statemnt is like the switch statement in JavaScript, but here we don't use break (once a match found there's no further execution) 

    macth exp:                                                                          // the result of the exp is matched against the below case statement 
        case val | val ...

            
 - the or | operator allows us to specify multiple possible cases 

            
 - ex: case 1 | 2 | 3: 

        
 :                                                           // if the result of the cond is val then the below code block is executed (and the match code block stop executing further)
            codeBlock
        ...                                                                             // we can have as many case statements as we like (that's the point here)
        case _:                                                                         // default case 
            defVal
DEMO 
    match 8 + 2:
        case 7 | 8 | 9:
        print(9)
            print(9)
        case 10:
            print(10)                                                                   // '10' printed         // furnter match code block execution is stopped    
        case 11 | 12 | 13:
            print(11)
        case 10:                                                                        // never checked because once there's a match the match code block stops executing    
            print('10 again')
        case _:
            print('fallback value')
