Method Description
textwrap.wrap(str, args...) - formats the passed str and returns the result in a list where each line is an item
textwrap.fill(str, args...) - formats the passed str and returns the result as str (same as wrap() but the result is returned as str)
|
textwrap.indent(str, prefix:str, args...) - adds prefix a the start of each line, returns the result as str
textwrap.dedent(str) - removes white spaces from the start of each line, returns the result as str
|
textwrap.shorten(str, width:int, args...) - 3.4 truncates str to widht width, returns the result as str

Description and Demonstration :

- The textwrap lib provides features to format strings

The wrap() and fill() methods

DEMO 
    import textwrap
    
    txt = """
        Lorem ipsum dolor sit amet,
        consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut
        labore et dolore magna aliqua.
    """

    textwrap.wrap(txt, width=20, max_lines=3, placeholder="...")    // -> ['     Lorem ipsum', 'dolor sit amet,', 'consectetur...']
    textwrap.fill(txt, width=20, max_lines=3, placeholder="...")    // -> '     Lorem ipsum\ndolor sit amet,\nconsectetur...'

The indent() and dedent() methods

DEMO 
    import textwrap
    
    txt = """
        Lorem ipsum dolor sit amet,
        consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut
        labore et dolore magna aliqua.
    """
    
    textwrap.indent(txt, '>')             // -> '\n>    Lorem ipsum dolor sit amet,\n>    consectetur adipiscing elit,\n>    sed do eiusmod tempor incididunt ut\n'  
    textwrap.dedent(txt)                  // -> '\nLorem ipsum dolor sit amet,\nconsectetur adipiscing elit,\nsed do eiusmod tempor incididunt ut\n'

The shorten() method

DEMO 
    import textwrap
    
    txt = """
        Lorem ipsum dolor sit amet,
        consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut
    """
    
    textwrap.shorten(txt, width=20, placeholder="...")      // -> 'Lorem ipsum dolor...'

The TextWrapper class

- The TextWrapper class basically does the same as wrap() or fill() methods, but we can preconfigure then readjust format settings 

    wrapper = TextWrapper(args...)
    wrapper.attr = val       // readjust specific setting
    
    wrapper.wrap(str)        // returns the result as list where each item is a line 
    wrapper.fill(str)        // returns the result as str
DEMO 
    import textwrap
    
    txt = """
        Lorem ipsum dolor sit amet,
        consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut
    """
    
    wrapper = textwrap.TextWrapper(width=20)      // initial setting
    wrapper.max_lines = 3                         // readjust settings
    wrapper.placeholder = "..."
    
    wrapper.fill(txt)                             // -> '     Lorem ipsum\ndolor sit amet,\nconsectetur...'
