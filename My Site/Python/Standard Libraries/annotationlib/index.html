Description and Demonstration :

The annotationlib module provides tools for introspecting annotations on modules, classes, and functions 

    get_annotations(obj|fn

      
 - may be a callable, class, module, or other object with __annotate__ or __annotations__ attributes. Passing any other object raises TypeError 

    
)        // returns a dictionary with the types 
    
    from annotationlib import get_annotations
    
    hello: str = "Hello"
    
    def repeat(data: str, cycles: int) -> str:
        result: str = ""
        for cycle in range(cycles):
            result += data
        return result
    
    get_annotations(hello)                 // -> TypeError: 'Hello' does not have annotations
    get_annotations(repeat)                // -> {'data': <class 'str'>, 'cycles': <class 'int'>, 'return': <class 'str'>}
