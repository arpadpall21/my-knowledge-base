Remember This :

- in Python the local time is controlled by the TZ system variable, we manually set the TZ value which tells python the local timezone (remember using time.tzset() to 'activate' the new timezone)

Description and Demonstration :

- arithmetic and comparison operations are supported betwen datetime objects

- datetime object can be 'naive' or 'aware'

'naive' has no timezone informations, assumes to contain the local time

'aware' has timezone informations

- to check if the datetime object is aware or naive call datetime.tzinfo, if the returned value is None the datetime object is naive, aware otherwise

Creating aware datetime objects

- we can implement our own tzinfo by creating a subclass of the tzinfo class and implement the utcoffset() / dst() / tzname() methods on it

- we can also create tzinfo implementation by using the timezone class (that's the cheapest way)

- then the created tzinfo implementation is passed as argument in datetime.datetime or datetime.time classes on creation so they are 'aware' of the implemented timezone

- or use the 'easy way' zoneinfo module (this handles timezone and DST alike)

Example :

- get local VS get utc 

    from datetime import datetime, timezone
    
    
    local = datetime.now()
    utc = datetime.now(tz=timezone.utc)
    
    print( local.hour )                   // -> 8
    print( utc.hour )                     // -> 6

- checking if datetime is 'aware' or 'naive' 

    from datetime import datetime, timezone
    
    
    naive = datetime.now()
    aware = datetime.now(tz=timezone.utc)
    
    print( naive.tzinfo )         # // -> None 
    print( aware.tzinfo )        # // -> UTC    (<datetime.timezone> obj)

- datetime.timedelta() 

    import datetime
    
    timeNow = datetime.datetime.now()
    
    timedelta = datetime.timedelta(
        days=13,
        weeks=1,
        hours=5,
        minutes=5
    )
    
    print( timedelta )                // -> returns a time object (passed time)
    print( timeNow + timedelta )      // -> the returned object is the current time added to the passed time (timedelta)

- proleptic Gregorian ordinal fromordinal() / toordinal() 

    import datetime
    
    date1 = datetime.date.fromordinal(1)
    print( date1 )                    // -> 0001-01-01  // date object 
    
    today = datetime.date.today()
    print( today.toordinal())         // -> 738314    // 738314 elpased since 1st of January year 0

- creating aware time object 

    import datetime
    import time
    
    tz = datetime.timezone( datetime.timedelta(hours=10), name="some tz name")
    t = datetime.time(20, 51, 00, tzinfo=tz)
    
    print( t.utcoffset() )      // -> 10:00:00  // timedelta object 
    print( t.tzname() )         // -> 'some tz name'
