Method Description
copy.copy(obj) - creates and returns shallow copy of obj (only 1st (direct) members are copied, any 2nd or deeper level members will be referenced from the original obj)
- does not copy module, method, stack trace, stack frame, file, socket, window, or any similar types
copy.deepcopy(obj) - creates and returns full copy of obj
- does not copy module, method, stack trace, stack frame, file, socket, window, or any similar types

Description and Demonstration :

- the copy lib creates shallow or deep copies of objects 

    import copy
    
    
    original = {"A": 1, "B": {"C": 2}}
    original_copy = copy.copy(original)           // only the 1st level objects are copied (any deeper level objects are referenced from the original)
    original_deep_copy = copy.deepcopy(original)  // all levels copied
    
    original["A"] = 2              // set 1st level member's value
    original["B"]["C"] = 3          // set 2nd (deep) level member's value
    
    original                       // -> {'A': 2, 'B': {'C': 3}}
    original_copy                  // -> {'A': 1, 'B': {'C': 3}}    // A 1st level member copied but B referenced from the original object   
    original_deep_copy             // -> {'A': 1, 'B': {'C': 2}}    // all members are copied
