Method / Property Description
zoneinfo.ZoneInfo(timezone:str) - returns a datetime.tzinfo implementation of the passed timezone
- to automatically figure out local timezone use zoneinfo.ZoneInfo("Europe/Budapest")
zoneinfo.available_timezones() - returns available timezones in a set that can be passed to zoneinfo.ZoneInfo()

Description and Demonstration :

- the zoneinfo (Python v.3.9+) is used to create 'aware' datetime objects (its a concrete implementation to support IANA timezone database (PEP 615) 

    import datetime
    import zoneinfo
    
    tz = zoneinfo.ZoneInfo("Europe/Budapest")
    dt = datetime.datetime(2022, 6, 12, 12, 9, tzinfo=tz)     // aware datetime object created (current time zone considered -> Budapest)
    
    dt.utcoffset()                     // -> 2:00:00     // timedelta object
    dt.dst()                           // -> 1:00:00     // timedelta object
    dt.tzname()                        // -> 'CEST'      // string
