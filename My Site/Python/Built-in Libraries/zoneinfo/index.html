Remember This :

Description and Demonstration :

- the zoneinfo module is a concrete implementation to support IANA timezone database (PEP 615)

- we use zoneinfo instances to create 'aware' datetime objects (pass the as tzinfo)

- this module does not provide direct zone data, instead 'asks' the system timezone database for it (if not available falls back to tzdata package)

- we can manually configure the desired timezone by passing the TZPATH system variable (for the desired timezone) (looked at compile time)

DEMO 
    import datetime
    import zoneinfo
    
    tz = zoneinfo.ZoneInfo("Europe/Budapest")
    
    dt = datetime.datetime(2022, 6, 12, 12, 9, tzinfo=tz)     // aware datetime object created 
    
    print( dt.utcoffset() )     // -> 2:00:00     // timedelta object
    print( dt.dst() )           // -> 1:00:00     // timedelta object
    print( dt.tzname() )        // -> 'CEST'      // string
