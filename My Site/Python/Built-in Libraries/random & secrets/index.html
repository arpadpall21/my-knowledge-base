random generation

random.randint(st(inc):int, end(inc):int) - returns a random int from the specified range
random.choice(iterable) - returns a random member from the iterable
random.shuffle(iterable) - suffles the iterable members in place (returns None)
random.sample(iterable, newLen:int) - picks newLen random elements from iterable and returns them in a new list (does not modify the original iterable)
random.random() - returns a random float between 0.0 - 1.0 range
random.uniform(st(inc):float, end(inc):float) - returns a random float between the specified range (the end may or may not be included, depending on float-point rounding)

Description and Demonstration :

- the random module provides tools for pseudo-random generation (deterministic optionally)

- deterministic meaning we can provide a starting value so we can get the same random result every time

- the random module is not suited for security or cryptographic uses, use the secrets module for that purpose

- the secrets uses the most secure random generation tool that OS can provide

random methods

Deterministic pseudo-random generation 

    import random
    
    print( random.random() )        // if no seed specified the current system time is used as start seed value
    
    random.seed(2.5)                // seeding random generation start value (further random generation will be deterministic)    
    
    print( random.random() )        // 0.41877545666909954     # after seeding 2.5 this random sequence generation is deterministic    
    print( random.random() )        // 0.9404475600150654  
    print( random.random() )        // 0.17813953767439972  
    
    
// --------------------------------------------------------------------
    st = random.getstate()          // generates and returns random current state value
    random.setstate(st)             // setting a previously captured state (further random generation will be deterministic)    
    
    print( random.random() )        // 0.8432052740270409      # after setting state this random sequence generation is deterministic    
    print( random.random() )        // 0.25779457143664086   
    print( random.random() )        // 0.6684710416662717

random methods 

    import random
    
    print( random.randbytes(10) )                 // -> b'\xf5\xb1e"JX\xb7\x91\xf1\xd8'   # random 10 byte sequence
    
    print( random.randrange(10) )                 // -> 1     # random number picked from the range
    print( random.randrange(0, 10, 2) )           // -> 4     # random number picked from the range (step restricted)
    
    print( random.randint(0, 2) )                 // -> 0     # random number picked from the range
    
    print( random.choice(('a', 'b', 'c')) )       // -> 'a'   # a memeber one memeber randomly returned 
    
    list = ['a', 'b', 'c', 'd', 'e']
    print( random.shuffle(list) )                 // shuffles the passed list elements in place
    print( list )                                 // -> ['c', 'd', 'e', 'a', 'b']     # shufled list 
    
    list = ['a', 'b', 'c', 'd', 'e']
    print( random.sample(list, 3) )               // -> ['d', 'e', 'a']     # new list (picks 3 members from the original and returns them in a new suffled list)
    print( list )                                 // ['a', 'b', 'c', 'd', 'e']   # original 
    
    print( random.random() )                      // -> 0.023294117934377523      # returns a random float between 0.0 - 1.0     
    
    print( random.uniform(1.0, 1.1) )             // -> 1.0381047643824899        # random float betwen 1.0 - 1.1

secret methods 

    import secrets
    
    
    list = ['a', 'b', 'c', 'd']
    print( secrets.choice(list) )             // -> 'd'     # ranodom member 
    print( secrets.randbelow(10) )            // -> 5       # random int between 0 - 10 (10 excluded)
    print( secrets.randbits(8) )              // -> 115     # random integer in 1 byte range (0-255)    
    
    print( secrets.token_bytes(8) )           // -> b'\x1b&\xf0\xc0\xe6\xe9\xa4\xf1'
    print( secrets.token_hex(8) )             // -> 'fcc5708a3a5d3881'
    print( secrets.token_urlsafe(8) )         // -> 'X_5HBgBizvk'
    
    print( secrets.compare_digest('test', 'test'))    // -> True    # method always takes the same time to make the comparison (timing attack protection)
