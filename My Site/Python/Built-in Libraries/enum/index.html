Description and Demonstration :

- Python Enum is supported from v3.4, it's an enumeration datype containing set of symbolic names bound to unique values

- We use enums with their symbolic members (member type = Enum type), but each member has its name and value

- Enums can be iterated over (members are stored in definition order) 

    from enum import Enum
    
    
    class MyEnum(Enum):               // creating enum 
        MEMBER = val
        ...
    
    
    MyEnum.MEMBER                     // using enum
    MyEnum['MEMBER']
    
    MyEnum.MEMBER.name                // accessing member name (returns 'MEMBER')
    MyEnum.MEMBER.value               // accessing member value 
    
    type(MyEnum.MEMBER) == MyEnum     // True (the member type = enum itself)
    
    
    
// functional enum creation --------------------------------------------
    MyEnum = Enum(className:str, [MEMBER:str, ...])     // does the same as class MyEnum(Enum): (enum member values will be 1 2 ...)
  
    from enum import IntEnum, StrEnum
    
    
    class MyEnum(IntEnum):            // same as Enum, but member values can only be int  
        MEMBER = int
        ...
    
    
    class MyEnum(StrEnum):            //  3.10  same as Enum, but member values can only be str 
        MEMBER = str
        ...
Example : 
    from enum import Enum
    
    
    class Color(Enum):
        RED = 1                 # internal values must be uniqe
        GREEN = 2
        BLUE = 3
    
    
    Color.RED                   # we use enums as values (one of the enumerated member) 
    Color['RED']
    
    
# accessing member name & value ---------------
    Color.RED.name              // -> 'RED'
    Color.RED.value             // -> 1
    
    
# iterating over enum members -----------------
    for val in Color:
        val                     // -> [<Color.RED: 1>, <Color.GREEN: 2>, <Color.BLUE: 3>]
    
    
# member type is the enum ---------------------
    type(Color.BLUE)            // -> <enum 'Color'>
    type(Color.BLUE) == Color   // -> True

Functional enum creation 

    from enum import Enum
    
    
    Color = Enum('Color', ['RED', 'GREEN', 'BLUE'])
    
    for member in Color:
        member                  // -> 1  |   2   |  3       // member values are mapped to integers

The IntEnum / StrEnum enums 

    from enum import IntEnum, StrEnum
    
    
    class Number(IntEnum):
        ONE = 1
        TWO = 2
        THREE = 3
    
    
    class NumberStr(StrEnum):
        ONE = 'one'
        TWO = 'two'
        THREE = 'three'
    
    
    type(Number.TWO.value)          // -> <class 'int'>   
    type(NumberStr.TWO.value)       // -> <class 'str'>
