Method Description
json.load(file-like, args...) - parses the passed file-like (containing JSON) into Python dict and returns it
json.loads(json:str|bytes|bytearray, args...) - parses the passed json into Python dict and returns it
|
json.dump(dict, file-like, args...) - serializes the passed dict into JSON and writes it in file-like
json.dumps(dict, args...) - serializes the passed dict into JSON string and returns it

Description and Demonstration :

- the json module allow to parse JSON to Python or serialize Python to JSON data

DEMO 
    import json
    
    json.load(open('./test.json', 'r'))                   # reads the 'test.json' JSON file parses its content to Python data and returns it 
    
    jsonData = '{"test":"val1", "arr":[1, 2, 3.1]}'
    json.loads(jsonData)                                  // -> {'test': 'val1', 'arr': [1, 2, 3.1]}     # parses the passed JSON string into Python data and returns it    
    
    
    pythonData = {"test":"val1", "arr":[1, 2, 3.1]}
    json.dump(pythonData, open('./test2.json', 'w'))      # serializes the passed Python dictionary into JSON and writs it in the 'test2.json' file   
    
    json.dumps(pythonData)                                // -> '{"test":"val1", "arr":[1, 2, 3.1]}'     # serializes the passed Python dictionary into JSON string and returns it
