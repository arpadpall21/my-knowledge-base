Description and Demonstration:

- Python is an interpreted programming languages that can be treated as procedural, OOP or functional programming language, it was developed by Guido van Rossum (released in 1991)

- Python uses simple English language-like syntax, aims to write more with less typing (uses the minimum possible symbols as possible, like it doesn't use parenthesies)

- Python is synchronous but it has libraries what we can use for asynchronous programming (asyncio)

Interpreters

- (Python) interpreters translate (Python) code into machine readable code, so different interpreters do this differently (they implement the Python script (code))

- the default Python interpreter is CPython, but there many interpreters out there (ex: IronPython, Jython, PyPy, PythonNet, Stackless, Python)

- ex: CPython itself is written in C programming language, it translates (interpretes) Python code into machine readable code

Variables and Scoping (global / nonlocal)

- python is case-sensitive, variable names can contain A-z 0-9 _ but cannot start with any number

- there are 2 scope types in Python, global scope and function scope (local scope), because of the Python syntax if we want to set upper scope variables we must use the nonlocal or global keywords

- each module has its own global scope (there's not shared global scope here like in JavaScript's global or window) 

 // variables --------------------------------------------------------------------------
    varName = val                                                                       // a variable is created when we assign a value to it   
    varName                                                                             // -! this will raise a NameError
    
    var1 = var2 = var3 = val                                                            // we can assign one value to multiple variables in one statement     
    var1, var2, var3 = val1, val2, val3                                                 // we can assign values to multiple variables in one statement (this is basically unpacking)   
    
 // scopes -----------------------------------------------------------------------------

// global keyword -----------------------------------------------------------------------
    def fn():
        def nestFn():
            global x                                                                     // from now on the x variable references the global x variable (this is how we set upper scope variables)    
            x = "I'm global"                                                             // assing value to the global variable 
        nestFn()
    fn()
    
    x                                                                                    // -> "I'm global"
    
// nonlocal keyword ---------------------------------------------------------------------
    x = 'global'
    def fn():
        x = 'local-1'
        
        def nestFn():
            nonlocal x                                                                   // from now on the x variable references the the parent scope's x variable (this is how we set upper scope variables)    
            x                                                                            // -> 'local-1'
        nestFn()
    fn()
