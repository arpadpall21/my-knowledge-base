Description and Demonstration :

- in Python everything is considered as object, this is important becasue primitives declared the same actaully points to the same memory location 

// immutable data types --------------------
    str                  // string
    int                  // integer
    float                // float (double precision floating point number)
    bool                 // boolean
    byte                 // bytes
    tuple                // tuple
    frozenSet            // frozen set
    None                                  // None is undefined or null data    
    
// mutable data types ----------------------
    list                 // list
    dict                 // dictionary
    set                  // set
    byteArr              // byte array
    
    objA = {"p1":1}
    objB = {"p1":1}
    
    objA is objB                                                                       // -> False      // objects point to different memory location 
    
    
    strA = 'abc'
    strB = 'abc'
    
    strA is strB                                                                       // -> True      // -! both primitives point to the same memory location    
    
    id(strA)                                                                           // -> 140279067851312      // same memory location 
    id(strB)                                                                           // -> 140279067851312
    
    
    strA = 'abc'
    strB = ''.join(['a', 'b', 'c'])                                                    // declared differently so it's not sotred on the same memory location 
    
    strA is strB                                                                       // -> False      // -! primitives does not point to the same memory location    
    
    id(strA)                                                                           // -> 140279067851312      // -! different memory location 
    id(strB)                                                                           // -> 140279067884537

Hashable data

- dictionary keys are converted to hashes for faster lookup, so any hashable data can be used as dictionary key

- the hash() function retuns the hash value of the passed data

DEMO 
    hash(21)                                                                           // -> 21                         // hashes of the data
    hash(21.21)                                                                        // -> 484227031934877717
    hash('some string')                                                                // -> -2369675691354808467
    hash(True)                                                                         // -> 1
    
    hash((21, 'str', False))                                                           // -> 1262170736549504929        // tuples are hashable if they contain hashable data only 
    hash((21, 'str', [1, 2]))                                                          // -! TypeError                  // unhashable data     
    
    hash([1, 2])                                                                       // -! TypeError                  // unhashable data     
    hash({'p1':'val1'})                                                                // -! TypeError                  // unhashable data     
    hash({1, 2})                                                                       // -! TypeError                  // unhashable data
