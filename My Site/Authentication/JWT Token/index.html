Description and Demonstration :

- a JSON Web Token (JWT) is an open standard (RFC 7519) for securely transmit information between parties as JSON

- information contained by the token is considered trusted because it's cryptographically protected (simple secret, private/public key)

- there is a good official JWT library support here... 

    header.body.signature      // encoded token format
    
    
// decoded token ---
    {                                               // header
      "alg": "alg",
      "typ": "JWT"
    }
    {                                                 // body 
      "sub": "subject",
      all body fields...
    }
    alg(header:base64UrlEnc.body:base64UrlEnc, secret)   // supported algorithms
