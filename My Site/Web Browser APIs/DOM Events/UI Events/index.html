Description and Demonstration :

UIEvent()

- the UIEvent() constructor is derived form the Event() constructor

- the UIEvent() is the parent constructor for the MouseEvent() / KeyboardEvent() / FocusEvent() / InputEvent() / TouchEvent / WheelEvent constructors, all these event objects can inherit its properties and methods

- this is how you can create a User Interfrace Object (Not Tested!) MDN create UIEvent Object

SYNTAX :    event = new UIEvent(type, init)    // NOT TESTED!

UI Events 

SYNTAX :    abort                              // fires when the loading of a media is aborted  
            error                              // fires when an error occurred during the loading of a media file
            load                               // fires when an object has loaded
            resize                             // fires when the document view is resized 
            scroll                             // fires when an element's scrollbar is being scrolled 
            select                             // fires after the user selects some text 
            unload                             // event occurs once a page has unloaded (browser windows has been closed) (Tested Not Working property, [2018-10-03])

UI Event Methods and Properties

- the UIEvent() constructor has some of the same properties/methods as the Event() does, like (cancelBubble / cancelable) 

SYNTAX :    eventObj.detail                    // returns the number of clicks on the same area under a short time
            eventObj.view                      // returns the "window" object in which the event happens
            eventObj.sourceCapabilities        // returns the InputDeviceCapabilities interface <- Experimental Technology
    
    <p onclick="demo(event)"> TEST </p>
<script>
    var demo = function(event) {
        event.detail;                          // -> 2                          // clicked twice on the element (in a short time)
        event.view;                            // -> Window                     // this window object
        event.sourceCapabilities               // -> InputDeviceCapabilities    // interface returned (tested in Chrome 2018-10-05)
    }
</script>
