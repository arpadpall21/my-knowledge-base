Remember This :

- the onmouseover and onmouseout Will Fire on child elements as many times they enter or leave the child elements but the onmouseenter and onmouseleave only react on the parent element

- the onclick fires Only when the left mouse button released, but the onmouseup Will fire to any buttons!

- the mousewheel event is deprecated, use wheel event instead!

Mouse Events 

SYNTAX :    onclick                                // fires when mouse clicks on the element (action happens when mouse button released!)  REATCS ONLY TO THE LEFT BUTTON!
            onmouseup                              // fires when ANY MOUSE BUTTON RELEASED
            onmousedown                            // fires when mouse clicks over the element (action happens when button pressed!)
            oncontextmenu                          // fires when right click to open the context menu
            ondblclick                             // fires when mouse double clicks
            onmousemove                            // fires every time when mouse moves on the element
            onmouseenter                           // fires when the mouse moves onto the element
            onmouseleave                           // fires when the mouse leaves the element
            onmouseover                            // fires when mouse moves onto the elment OR ELEMENT's CHILDS
            onmouseout                             // fires when mouse leaves the element OR ELEMENT's CHILDS

Mouse Event Properties and Methods 

SYNTAX :    eventObj.altKey                        // returns a boolean value if the "Alt" key was pressed when mouse event was trigered
            eventObj.ctrlKey                       // returns a boolean value if the "Ctrl" key was pressed when mouse event was trigered
            eventObj.shiftKey                      // returns a boolean value if the "Shift" key was pressed when mouse event was trigered                      
            eventObj.getModifierState("keyArg")       // returns a boolean value if the set key was pressed when mouse event was trigered
                - "getArg" - button argument to set (MDN Geko keyArg)
            
            eventObj.which                         // returns the value of the pressed button on the mouse (1 left / 2 middle / 3 right) (Not stanard)
            eventObj.button                        // returns the value of the pressed button on the mouse (0 left / 1 middle / 2 right / 3 / 4) (IE8 and earlier versions  (1 left / 4 middle / 2 right))                
            eventObj.buttons                       // returns the value or the sum value of the pressed buttons on the mouse (0 no button / 1 primary / 2 secondary / 4 auxiliary / 8 4th button / 16 5th button) (Ex: 1 + 2 returns 3)
            
            eventObj.clientX                       // returns the horizontal position of the mouse relative to the current viewport
                eventObj.x                         // alias clientX (Not stanard)
            eventObj.clientY                       // returns the vertical position of the mouse relative to the current viewport
                eventObj.y                         // alias clientY (Not stanard)
            eventObj.offsetX                       // returns the horizontal position of the mouse relative to the element (Not stanard but supported by every browser [2018-11-27])
            eventObj.offsetY                       // returns the vertical position of the mouse relative to the element (Not stanard but supported by every browser [2018-11-27])
            eventObj.pageX                         // returns the horizontal position of the mouse relative to the document
            eventObj.pageY                         // returns the vertical position of the mouse relative to the document
            eventObj.screenX                       // returns the horizontal position of the mouse relative to the user's screen
            eventObj.screenY                       // returns the vertical position of the mouse relative to the user's screen
            eventObj.movementX                     // returns the horizontal distance between the current and the pervious mouse position (in pixel)
            eventObj.movementY                     // returns the vertical distance between the current and the pervious mouse position (in pixel)
            
            eventObj.relatedTarget                 // returns the related element what trigered the current event

<script>
    document.getElementById("demo").addEventListener("click", mouseMove);
    
    function mouseMove(ev) {
        ev.altKey;                          // -> false                     // alt key not perssed while clicked
        ev.ctrlKey;                         // -> false                     // ctrl key not perssed while clicked
        ev.shiftKey;                        // -> false                     // shift key not perssed while clicked
        ev.getModifierState("CapsLock");    // -> true                      // "CapsLock" was active while clicked
    
    // mouse buttons ---------------------------------    
        ev.which;                           // -> 1                         // left mouse button pressed
        ev.button;                          // -> 0                         // left mouse button pressed
        ev.buttons;                         // -> 0                         // left mouse button pressed
    
    // mouse position --------------------------------    
        ev.clientX;                         // -> 16                        // position on the viewport
        ev.clientY;                         // -> 636                       
        ev.x;                               // -> 16                        // position on the viewport 
        ev.y;                               // -> 636                       
        ev.offsetX;                         // -> 9                         // position on the element
        ev.offsetY;                         // -> 8                         
        ev.pageX;                           // -> 16                        // position on the document
        ev.pageY;                           // -> 4279                      
        ev.screenX;                         // -> 86                        // position on the screen
        ev.screenY;                         // -> 866                       
        ev.movementX;                       // -> -1                        // distance relative to the previous mouse position
        ev.movementY;                       // -> 0                         
    }
</script>

WheelEvent()

- wheel events are trigered when the mouse wheel (or similar device) is used (not to be confused with the scroll event)

- the WheelEvent() constructor is inherited form the MouseEvent() which is inherited from the UIEvent() which is inherited from the Event() constructor

- this is how you create a WheelEvent Object (Not Tested!) MDN Create WheelEvent Obj 

SYNTAX :    event = new WheelEvent(type, init)     // NOT TESTED!

Wheel Event

- the mousewheel event is deprecated, use wheel event instead! 

SYNTAX :    wheel                                  // fires when mouse wheel used / in touchpad when page is scrolled or zoomed in/out

Wheel Event Properties 

SYNTAX :    eventObj.deltaX                        // returns the scrolled amount on the "X" axis
            eventObj.deltaY                        // returns the scrolled amount on the "Y" axis
            eventObj.deltaZ                        // returns the scrolled amount on the "Z" axis
            
            eventObj.deltaMode                     // returns the a number which depicts the measurement unit of the delta properties (0 pixel / 1 line / 2 page)
    
<script>    
    document.body("wheel", action);
    
    function action(event) {
        event.deltaX;                              // horizontal position (not tested because of the lack of mouse support)
        event.deltaY;                              // -> 100    // chrome scrolls 100 px per wheel click
        event.deltaZ;                              // Z axis scroll amout (not tested because of the lack of mouse support)
        
        event.deltaMode                            // -> 0      // 0 = pixels (wheel scroll measured in pixels)
    }
</script>
