DRAG EVENTS
Event Description DOM
drag fires when an element is being dragged 3
dragend fires when the user has finished dragging an element 3
dragenter fires when the dragged element enters the drop target 3
dragleave fires when the dragged element leaves the drop target 3
dragover fires when the dragged element is over the drop target 3
dragstart fires when the user starts to drag an element 3
drop fires when the dragged element is dropped on the drop target 3
DRAGEVENT PROPERTY
Property Description
event = new DragEvent(type, init) create DragEvent object (Not Tested!)
eventObj.dataTransfer returns the DataTransfer object which handles the draged data

DataTransfer Object methods and properties (Check the Browser Compatibility!)
Property / Method Description
object = new DataTransfer() creates a dataTransfer object
dataTransferObj.clearData("format/key") clears the targeted "format/key" (or all data if no "format/key" specified) form the current DataTransfer object
dataTransferObj.dropEffect = "effectName" restricts the drop effect set by the effectAllowed property in the drop zone (none(no drop allowed) / copy / move) (supported by dragenter / dragover events)
dataTransferObj.effectAllowed gets or sets the effect that is allowed in the drag operation (uninitialized(default[all]) / copy / move) (supported by dragstart event)
dataTransferObj.files returns the FileList object which handles droped or selected exterior files [file API (link)]
dataTransferObj.getData("format/key") returns the specified value form the current DataTransfer object
dataTransferObj.items returns the DataTransferItemList object which is a list containing DataTransferItem objects for each transfered data (this object has the kind and the type properties) set by the setData() method at the dragstart event
dataTransferObj.setData("format/key", "data") sets one or multiple data in the dataTransfer object
dataTransferObj.setDragImage(img, xOffset, yOffset) sets a custom image during drag operation (only works with the dragstart event)
dataTransferObj.types returns an array containing the format/key set by the setData() method at the dragstart event

the DataTransfer object

- the DataTransfer object holds data (mostly usded in Drag and Drop operation) but it can be created and handled in other datatransfer operations 

SYNTAX :    object = new DataTransfer()                            // creates a dataTransfer object

DataTransfer object methods and properties 

SYNTAX :    dataTransferObj.setData("format/key", "data")          // sets one or multiple data in the dataTransfer object          <- in my tests (on 2018-10-06 in Chrome) it works mostly as key/value pairs  
                - "format/key" : specifies the data format to transfer (it's "text" the best to use) (MDN Draggable Data Types [link])
                - "data" : the text to transfer
                
            dataTransferObj.getData("format/key")                  // returns the specified value form the current DataTransfer object
            dataTransferObj.clearData("format/key")                // clears the targeted "format" (key) or all data if no "format" (key) specified form the current DataTransfer object
                - "format/key" / "format/key" : datatype (key) 
                
            dataTransferObj.setDragImage(img, xOffset, yOffset)    // sets a custom image during drag operation (only works with the dragstart event)  
                - xOffset and yOffset : defines the coordinates where the set image is grabbed by the cursor 
              
            dataTransferObj.effectAllowed                          // gets or sets the effect that is allowed in the drag operation (uninitialized(default[all]) / copy / move) (supported by dragstart event)
            dataTransferObj.dropEffect = "effectName"              // restricts the drop effect set by the effectAllowed property (none(no drop allowed) / copy / move) (supported by dragenter / dragover events)
            dataTransferObj.files                                  // returns the FileList object which handles droped or selected exterior files [file API (link)]  
            dataTransferObj.types                                  // returns an array containing the format/key set by the setData() method at the dragstart event  
            dataTransferObj.items                                  // returns the DataTransferItemList object which is a list containing DataTransferItem objects for each transfered data (this object has the kind and the type properties) set by the setData() method at the dragstart event  
              
          // There are some more Geko specific methods/properties what are not tested here!

<script>
// create an image element to the "setDragImage()" method
    var myImg = new Image();
    myImg.src = "red.jpg";
    
// dragStart
    var dragItem = function(event) {
    // setData --------------------------------------------------- 
        event.dataTransfer.setData("text", event.target.id);                                        // multiple data set
        event.dataTransfer.setData("customKey", "data_2");                                          
    // setDataImage ---------------------------------------------- 
        event.dataTransfer.setDragImage(myImg, 25, 25);                                             // drag image set (the mouse cursor grabs the image 25px form the left and top)
    // clearData -------------------------------------------------    
        event.dataTransfer.clearData("text");                                                       // this would erase the "text" format/key only
        event.dataTransfer.clearData();                                                             // this would clear the whole "dataTransfer" object's content
    // effectAllowed ---------------------------------------------    
        event.dataTransfer.effectAllowed = "move";                                                  // this item can droped only on zones which has the "dropEffect='move'" effect declared
    }
    
// item enters in the drop zones
    document.getElementById("dropZone_A").ondragover = function(event) {
        event.preventDefault();                                                                      <--  default behaviour must be prevented in each listener!            
        event.dataTransfer.dropEffect = "move";                                                     // on this zone the dragged item can be droped
    }
    document.getElementById("dropZone_B").ondragover = function(event) {
        event.preventDefault();                                                                      <--  default behaviour must be prevented in each listener! 
        event.dataTransfer.dropEffect = "copy";                                                     // the dragged item cannot be droped because this zone accepts elements which has the "copy" effect only!  
    }
    
// drop
    var dropIt = function(event) {
        event.target.appendChild(document.getElementById(event.dataTransfer.getData("text")));      // append the draged item
    // getData ---------------------------------------------------    
        event.dataTransfer.getData("text");                     // -> "item"                        // returns the value set under the "text" format/key
        event.dataTransfer.getData("customKey");                // -> "data_2"                      // returns the value set under the "customKey" format/key
    // files -----------------------------------------------------    
        event.preventDefault();                                                                        <- default behaviour prevented
        event.dataTransfer.files;                               // -> FileList                      // returns the "FileList" object which handles exterior files droped or selected 
    // types -----------------------------------------------------
        event.dataTransfer.types;                               // -> ["text/plain", "customkey"]   // returns an array containing formats/keys
    // items -----------------------------------------------------
        event.items;                                            // -> DataTransferItemListÂ {0: DataTransferItem, 1: DataTransferItem}
    }                                                                                               // there are 2 "DataTransferItem" objects, each has the "type" and "kind" property containing information about the set datas

        event.preventDefault();                                              // prevent default behaviour
        var collector = "";
        
        for(let i = 0; i < event.dataTransfer.files.length; i++) {           // looping through FileList ojbect's content and getting information
            collector += "File Name = " + event.dataTransfer.files[i].name + 
                " / Size = " + event.dataTransfer.files[i].size +
