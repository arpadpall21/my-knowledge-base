Description and Demonstration :

- a DOM event is a something that happes to an HTML element when something or the user invokes it, the Browser notifies JavaScript about these events

- event listens can be registered to these events so we can fire a function when certain event happens

- DOM events are propagated (also happening) on other elements:

- (by Default) bubbling up on the DOM tree (happens on all elements all the way up to the root)

- capturing down on the DOM tree (happens on all elements all the way down to leaf elements)

Registering Event Listeners

- an event listener or event handler is basically a function that triggers on a registered event

The composedPath() / composed / bubbles / stopPropagation() / cancelBubble / stopImmediatePropataion() / target / srcElement / currentTarget / eventPhase methods

DEMO 
    <p id="demo"> myParagraph <u>childElement</u> </p>
<script>
    document.getElementById('demo').addEventListener("click", parentFunc);             // listener set up 
    document.getElementById('demo').addEventListener("click", parentFuncB);            // second listener set up with the same event type
    
    function parentFunc(e) {
        e.composedPath();                                                              // -> [u#demoChild, p#demo, body, html, document, Window]   // when clicked on the child element (event propagation chain)
        e.compsed;                                                                     // -> true                                                  // the click event crosses the "Shadow-DOM" boundary
        e.bubbles;                                                                     // -> true                                                  // bubbling event type (propagates the event)
        //  ------------------------------
        e.target;                                                                      // -> <u>childElement</u>                                   // when clicked on the child element (returns the element what fired this event)
        e.target;                                                                      // -> <p id="demo"> myParagraph <u>childElement</u> </p>    // when clicked on this element (returns the element what fired this event)     
            e.srcElement                                                               // does the same as "e.target" property
        e.currentTarget;                                                               // -> <p id="demo"> myParagraph <u>childElement</u> </p>    // no matter where is clicked (always returns this (current) element!)      
        //  ------------------------------
        e.eventPhase;                                                                  // -> 2                                                     // when clicked on this element (target phase) this element produces the event
        e.eventPhase;                                                                  // -> 3                                                     // when clicked on the child element (bubbling phase) other element propagates the event and this element is in the bubbling phase to that event    
        e.eventPhase;                                                                  // -> 1     (if the listener would set up "capturing") (capturing phase)
        //  ------------------------------
        e.stopPropagation()                                                            // event propagation is prevented, the child element event is no fired!
        e.cancelBubble = true;                                                         // does the same as "stopPropagatoin()" method 
        e.cancelBubble;                                                                // -> ture                                                  // because the event propagation is prevented                       
        e.stopImmediatePropagation();                                                  // prevents the second listener to fire of the (same event type)
    }
    
    function parentFuncB(e) {}                                                         // this function won't execute because the same event type propagation is prevented above "e.stopImmediatePropagation()"
</script>
