Scrolling & Sizing
window.innerHeight - returns the browser's browsing window height (scrollbar included)
window.innerWidth - returns the browser's browsing window width (scrollbar included)
window.outerHeight - returns the browser application height
window.outerWidth - returns the browser application width
|
window.scrollX - returns horizontal scroll position of the document (counted from left)
- alias for pageXOffset
window.scrollY - returns vertical scroll position of the document (counted from the top)
- alias form pageYOffset
|
window.scroll(x:nr, y:nr | option:obj)
window.scrollTo(x:nr, y:nr | option:obj) 		- set the scrolling coordinates to the document (option argument support 61 36 48 79 14 )

- set the scrolling coordinates to the document relative to the current scroll position (option argument support 61 36 48 79 14 )

Timers
window.setTimeout(fn, delayMs:nr, arg, ...) - runs the passed function after delayMs (milliseconds), returns a timeoutId:nr that can be used to cancel the trigger
window.clearTimeout(timeoutId:nr) - cancels the timeout identified by timeoutId:rn
|
window.setInterval(fnr, delayMs:nr, arg, ...) - executes the passed function every delayMs (milliseconds), returns a timeoutId:nr that can be used to cancel the timout execution
window.clearInterval(intervalId) - cancels the interval execution identified by timeoutId:rn
|
window.requestAnimationFrame(fn) - used to animate DOM elements, runs the fn repedeadly as fast as the browser's refresh rate
- returns a frameId:nr tat can be used to cancel the call 10 webkit / 24 4 moz / 23 15 webkit / 15 10 6 webkit / 6.1
window.cancelAnimationFrame(FrameId:nr) - cancels an animation frame identified by FrameId:nr 24 11 moz / 23 15 10 6 webkit / 6.1

Encode URL Characters
window.encodeURI(str) - url encodes the passed str so it can be used as URL
- percent encodes non url-safe caracters (reserved charactes are encoded if they are not part of the url structure)
window.decodeURI(str) - decodes the previously encoded encodeURI(str) into human readable format
|
window.encodeURIComponent(str) - url encodes all non url-safe and reserved characters (used to url encode url parts)
window.decodeURIComponent(str) - decodes the previously encoded encodeURIComponent(str) into human readable format
|
window.btoa(str) - encodes the passed str into base64 (only can encode latin1 characters)
window.atob(base64:str) - decodes a base64 encoded string into utf-8 (previously encoded by btoa(str))

    window.getComputedStyle(thisElement).color;                                        // -> 'rgb(255, 0, 0)'    // gets the currently rendered style (read only)
    window.getComputedStyle(thisElement).fontSize;                                     // -> '24px'
        
    thisElement.style.border = '2px black solid';                                      // we can set inline styles like this   
    thisElement.style.fontWeight = 'bold';
    thisElement.style.animationPlayState = 'running';
</script>

Alert-Box alert()

- the Alert Box alert() pop-up window blocks the page until the "OK" button is pressed

- the Alert Box is used to comunicate a "must-read" message to the user

Confirm-Box confirm()

- the Confirm Box confirm() pop-up window blocks the page until confirmation button is pressed ("OK" returns true VS "Cancel" returns false)

- the Confirm-Box is used if you want the user accept or reject something 

    var confirm = window.confirm("Are you sure to accept");     // blocks the page upon trigered and asks for a confirmation
                                                                   the "OK" button returns true VS the "Cancel" button returns false

Prompt-Box prompt()

- the Prompt Box prompt() pop-up window blocks the page until confirmation button is pressed and a text is entered in the text-box (optional) ("OK" returns the input value VS "Cancel" returns null)

- the Prompt-Box is used to collect an input from the user 

    var promptInput = window.prompt("Enter a value", "Default Value");      // blocks the page upon trigered and asks for an input
                                                                               the "OK" button returns the entered value VS the "Cancel" button returns null

Encode URL characters

- any non URL safe character must be encoded if we want to use it in a URL (also called percend encoding) 

    a-z A-Z 0-9 $ - _ . , ' + ! * ( )         // url safe characters (can be used in URL without encoding)
    : ; / ? @ = &                             // reserved URL characters (encoded only if we're using it as custom character) (ex: ? used to start query string, in this case won't be encoded)
    
    otherChar                // must be url (percent) encoded

The encodeURI() / encodeURIComponent() / decodeURI() / decodeURIComponent() / btoa() / atob() methods

DEMO : 
// encodeURI() / decodeURI() / encodeURIComponent() / decodeURIComponent() ---------------
    const validUri = window.encodeURI('https:/\/someDomain/somePath?one=1&two=ket')       // -> 'https://someDomain/somePath?one=1&two=ket%C5%91'
    window.decodeURI(validUri)              // -> 'https://someDomain/somePath?one=1&two=ket'
    
    const encodeAll = window.encodeURIComponent('https:/\/someDomain/somePath?one=1&two=ket')   // -> 'https%3A%2F%2FsomeDomain%2FsomePath%3Fone%3D1%26two%3Dket%C5%91'     // encode all non url-safe caracters (reserved characters included)   
    window.decodeURIComponent(encodeAll)             // -> 'https://someDomain/somePath?one=1&two=ket'
