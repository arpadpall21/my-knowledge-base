Description and Demonstration :

- any HTML element can be transformed into a popover element by adding the popover attribute

- a popover element is hidden by default, when opened centered on top of the UI

- the Popover API is used to create popovers, if we want to create modals we should use the <dialog> element instead 

    <HTMLElement popover> ... </HTMLElement>            // creates a popover element of any html elements
    
    HTMLpopoverElement.showPopover()       // controlling the popover element programatically
    HTMLpopoverElement.hidePopover()
    HTMLpopoverElement.togglePopover()
    
    
// simplest way to open|close a popover element via button ------
    <button popovertarget="popoverId"></button>        // button open|closes the targeted popover
    <HTMLElement id="popoverId" popover></HTMLElement>
    
    
// events -----------------
    beforetoggle                      // triggered before the popover element toggles
    toggle                            // triggered after the popover element toggled

Simples way to control a popover via button

Popover content
Example: 
    <button popovertarget="mypopover">Toggle the popover</button>
    <div id="mypopover" popover>Popover content</div>

Control Popover via script

Popover content
Example: 
    <button onMouseUp="tgPopover()">Show popover for 3 seconds</button>
    <div id="my-popover-scripted" popover>Popover content</div>
    <script>
        function tgPopover() {
          const popoverElement = document.getElementById('my-popover-scripted');
          popoverElement.showPopover();
          setTimeout(() => popoverElement.hidePopover(), 3000);
        }
    </script>

Popover Events

- check console to see triggered events

Popover content
Example: 
    <button popovertarget="my-popover-event-test">Toggle the popover</button>
    <div id="my-popover-event-test" popover>Popover content</div>
    <script>
        const popoverElement = document.getElementById('my-popover-event-test');
        
        popoverElement.addEventListener('beforetoggle', () => console.log('before toggle event fired'))
        popoverElement.addEventListener('toggle', () => console.log('toggle event fired'))
    </script>
