Property / Method Description
Sizing
htmlElement.clientHeight - returns the element's height in pixels (content + padding)
htmlElement.clientWidth - returns the element's width in pixels (content + padding)
|
htmlElement.offsetHeight - returns the element's height in pixels (content + padding + border)
htmlElement.offsetWidth - returns the element's width in pixels (content + padding + border)
|
htmlElement.scrollHeight - returns the element's height in pixels (content + padding + overflowing content)
htmlElement.scrollWidth - returns the element's width in pixels (content + padding + overflowing content)
|
htmlElement.clientTop - returns the element's top border thickness in pixels
htmlElement.clientLeft - returns the element's left border thickness in pixels
|
htmlElement.getBouncingClientRect() - returns a DOMRect object that describes the size and the position of the element relative to the viewport
|
Scrolling
htmlElement.scrollLeft - gets or sets (in pixels) the horizontal scroll position of the content within the element
htmlElement.scrollTop - gets or sets (in pixels) the vertical scroll position of the content within the element
|
htmlElement.scroll(x:nr, y:nr | option:obj)
htmlElement.scrollTo(x:nr, y:nr | option:obj) 		- set the scrolling coordinates to the document 61 36 48 79 10.1
htmlElement.scrollBy(x:nr, y:nr | option:obj)
- set the scrolling coordinates to the element relative to the current scroll position 61 36 48 79 10.1
|
htmlElement.scrollIntoView(options...) - scrolls the ancestor elements in a way that this element is visible to the user

Size clientHeight / clientWidth / offsetHeight / offsetWidth / scrollHeight / scrollWidth / getBouncingClientRect() / clientTop / clientLeft properties & methods

DEMO 
// CSS --------------------------------------------------------------------------------
    #test1 {
        width: 150px;
        height: 50px;
        background-color: lightcoral;
        border: solid 10px lightskyblue;
    }
    
// HMTL -------------------------------------------------------------------------------
    <div id='test1' onclick="show()"> 
        <p> testtesttesttesttesttesttest </p>                                          // element's conent is overflowing both horizontally and vertically   
        <p> test test test test test test test test test test </p>
    </div>
<script>
    let element = document.getElementById('test1');
    
    element.clientHeight;                                                              // -> 50                 // (content + padding)
    element.clientWidth;                                                               // -> 150                // (content + padding)
    
    element.offsetHeight;                                                              // -> 70                 // (content + padding + border)
    element.offsetWidth;                                                               // -> 170                // (content + padding + border)
    
    element.scrollHeight;                                                              // -> 74                 // (content + padding + overflowing content)
    element.scrollWidth;                                                               // -> 181                // (content + padding + overflowing content)
    
    element.clientTop;                                                                 // -> 10                 // top border width    
    element.clientLeft;                                                                // -> 10                 // left border width    
    
    element.getBoundingClientRect();                                                   // -> {    
                                                                                       // ->   x: 10,           // horizontal position relative to the viewport   
                                                                                       // ->   left: 10,        // same as 'x'
                                                                                       // ->   y: 845,          // vertical position relative to the viewport   
                                                                                       // ->   top: 845,        // same as 'y'
                                                                                       // ->   bottom: 965,     // the bottom of the element counted from top the viewport     
                                                                                       // ->   right: 530,      // the right side position of the element counted from left the viewport    
                                                                                       // ->   width: 520,      // element with (content + padding + border)   
                                                                                       // ->   height: 120      // element height (content + padding + border)
                                                                                       // -> }

</script>
