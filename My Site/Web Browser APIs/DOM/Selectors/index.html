Element Selectors
Property / Method Description
document.getElementById(id:str) - returns the htmlElement with the specified id attribute
document.getElementsByClassName(classId:str) - returns an htmlCollection of the elements with the specified class attribute
document.getElementsByTagName(htmlTag:str) - returns an htmlCollection of the specified elements
|
document.querySelector(cssSelector:str) - returns the the first htmlElement targeted by the query selector
document.querySelectorAll(cssSelector:str) - returns a nodeList targeted by the query selector
|
document.activeElement - returns the currently focused htmlElement

Collectors
Property Description
document.documentElement - returns the <html> element
document.head - returns the <head> element
document.body - returns the <body> element
document.title - gets or sets the title of the document (the text contained by the <title> element)
|
document.links - returns all the document's <a> and <area> elements that have the href attribute in a htmlCollection
document.images - returns all the document's <img> elements in a htmlCollection
document.embeds - returns all the document's <embed> elements in a htmlCollection
document.anchors - returns all <a> elements that has a name attribute in a htmlCollection
|
document.scripts - retunrs all the document's <script> elements in a htmlCollection
document.forms - returns all the document's <form> elements in a htmlCollection

Element Selectors

The getElementById() / getElementsByClassName() / getElementsByTagName() / querySelector() / querySelectorAll() / activeElement methods and property

DEMO 
    <p id="par"> premier paragraphe </p> 
    <p class="par"> deuxieme paragraphe </p>
    <p class="par"> troisieme paragraphe </p>
<script>
// getElementsById() / querySelector() ------------------------------------------------
    document.getElementById("par");                                                    // -> <p id="par"> premier paragraphe </p>
    document.querySelector(".par");                                                    // -> <p class="par"> deuxieme paragraphe </p>  // fist item returned ONLY!
        
// getElementsByClassName() / getElementsByTagName() / querySelectorAll() -------------
    document.getElementsByClassName("par")                                             // -> [p.par, p.par]                            // HTML Collection
    document.getElementsByClassName("par")[0]                                          // -> <p class="par"> deuxieme paragraphe </p>  // returns the first item from the HTML collection
    
    document.getElementsByTagName("p")                                                 // -> [p.par, p.par]                            // HTML Collection
    document.getElementsByTagName("p")[1]                                              // -> <p class="par"> troisieme paragraphe </p> // returns the second item from the HTML collection
    
    document.querySelectorAll(".par")                                                  // -> [p.par, p.par]                            // Node List
    document.querySelectorAll(".par")[1]                                               // -> <p class="par"> troisieme paragraphe </p> // returns the second item from the Node List
    
// activeElement ----------------------------------------------------------------------
    document.activeElement                                                             // -> returns the currently focused element (only fucusable elements! ex: input)
</script>

HTML Collection VS Node List

The length / item() / namedItem() / entries() / keys() / values() / forEach() property and methods

- an HTMLCollection is an Object containing multiple HTML elements, collected by an HTML collector like getElementsByTagName()

- a Node List is an Object containing multiple Nodes, collected by a Node collector like querySelectorAll()

DEMO 
    <p> elso elem </p>
    <p id="2nd"> masodik elem </p>
    <p name='3rd'> harmadik elem </p>
    <p> negyedig elem </p>
<script>
    var myHTMLCollection = document.getElementById('demoList').getElementsByTagName("li");  // HTMLCollection 
    var myNodeList = document.getElementById('demoList').querySelectorAll("li");            // NodeList
    
// length -----------------------------------------------------------------------------
    myHTMLCollection.length;                                                           // -> 4      // collection has 4 elements
    myNodeList.length;                                                                 // -> 4      // list has 4 nodes
    
// item() / namedItem() ---------------------------------------------------------------
    myHTMLCollection.item(1);                                                          // -> <p id="2nd"> masodik elem </p>     // element indexed  
    myNodeList.item(2);                                                                // -> <p name='3rd'> harmadik elem </p>  // element indexed  
    
    myHTMLCollection.namedItem('2nd');                                                 // -> <p id="2nd"> masodik elem </p>     // element referenced by id attribute (HTMLCollection only!)  
    myHTMLCollection.namedItem('3rd')                                                  // -> <p name='3rd'> harmadik elem </p>  // element referenced by name attribute (HTMLCollection only!)   
     
// [ ] --------------------------------------------------------------------------------
    myNodeList[2];                                                                     // -> <p name='3rd'> harmadik elem </p>  // node list items can be indexed only   
    
    myHTMLCollection[1];                                                               // -> <p id="2nd"> masodik elem </p>     // HTML collection items can be indexed
    myHTMLCollection['2nd'];                                                           // -> <p id="2nd"> masodik elem </p>     // HTML collection items can be referenced by id attribute 
    myHTMLCollection['3rd'];                                                           // -> <p name='3rd'> harmadik elem </p>  // HTML collection items can be referenced by name attribute    
    
// entries() / keys() / values() ------------------------------------------------------
    var getEntries = myNodeList.entries();                                             // returns an iterable object containing the lists key value (index node) pairs 
    getEntries.next();                                                                 // -> {value:[0, <p> elso elem </p>], done:false}   
        
    var getKeys = myNodeList.keys();                                                   // returns an iterable object containing the lists keys    
    getKeys.next();                                                                    // -> {value:0, done:false}    
        
    var getValues = myNodeList.values();                                               // returns an iterable object containing the lists values    
    getValues.next();                                                                  // -> {value:<p> elso elem </p>, done:false}   
    
// forEach() -------------------------------------------------------------------------- 
    myNodeList.forEach(function(value, key, nodeList){                                 // runs a pass for each node in the list 
        value;                                                                         // -> <p> elso elem </p> | ...       // node 
        key;                                                                           // -> 0                  | ...       // index
        nodeList;                                                                      // -> NodeList(4)        | ...       // the node list itself   
    });
</script>

    var HTMLCollection = document.getElementById('testList').children;
    var nodeList = document.getElementById('testList').querySelectorAll('li');
    
console.log( '// item() / namedItem() ------------------------------------' );

Traversing

The parentNode / childElementCount / children / childNodes / firstElementChild / firstChild / lastElementChild / lastChild / nextElementSibling / nextSibling / previousElementSibling / previousSibling / elements properties

DEMO 
    <div id="parent">
        <p> child paragraph-1 </p>
        <p id="child"> child paragraph-2 </p>       <!-- comment -->
        <p> child paragraph-3 </p>
    </div>
    <form id="demo">
        <input type="text" value="myText">
        <input type="number" value="21">
        <input type="submit value="Submit">
    </form>
<script>
    var parent = document.getElementById("parent");
    
// parentNode / childElementCount -----------------------------------------------------
    document.getElementById('child').parentNode;                                       // -> "<div id="parent">"
    parent.childElementCount;                                                          // -> 3
    
// children / childNodes --------------------------------------------------------------
    parent.children;                                                                   // -> [p, p#child, p, child: p#child]                           // HTMLCollection
    parent.children[2];                                                                // -> <p> child paragraph-3 </p>
    parent.childNodes;                                                                 // -> [text, p, text, p#child, text, comment, text, p, text]    // NodeList
    parent.childNodes[5];                                                              // -> <!-- comment -->
    
// firstElementChild / firstChild / lastElementChild / lastChild ----------------------
    parent.firstElementChild                                                           // -> <p> child paragraph-1 </p>
    parent.firstChild                                                                  // -> #text                                                     // the first Node is returned!
    parent.lastElementChild                                                            // -> <p> child paragraph-3 </p>
    parent.lastChild                                                                   // -> #text                                                     // the last Node is returned!
    
// nextElementSibling / nextSibling / previousElementSibling / previousSibling --------
    document.getElementById("child").nextElementSibling                                // -> <p> child paragraph-3 </p>
    document.getElementById("child").nextSibling                                       // -> #text                                                     // the next Node is returned!
    document.getElementById("child").previousElementSibling                            // -> <p> child paragraph-1 </p>
    document.getElementById("child").previousSibling                                   // -> #text                                                     // the previous Node is returned!
    
// elements ---------------------------------------------------------------------------
    document.forms['demo'].elements.lenth;                                             // -> 3 
    document.forms.demo.elements[0].type;                                              // -> "myText"
    document.forms['demo'].elements[1].value = 100;                                    // sets the value to 100
</script>
