Properties Description
screen.pixelDepth - returns the number of color bits used to display (HTML uses 24bit "True Colors")
screen.width - returns the full width of the screen as a number (pixels)
screen.height - returns the full width of the screen as a number (pixels)
|
screen.availWidth - returns the available with (in pixels) for the web content
screen.availHeight - returns the available height (in pixels) for the web content
|
screen.colorDepth - returns the number of bits used to display one color (HTML uses 24bit "True Colors")
|

Description and Demonstration :

- the screen Object contains informations about the user's screen

Example :

CODE : 
    <p id="screenInfo"></p>
<script>
    var fullScreenWidth = screen.width;
    var fullScreenHeight = screen.height;
    var availableScreenWidth = screen.availWidth;
    var availableScreenHeight = screen.availHeight;
    var colorDepth = screen.colorDepth;
    
    document.querySelector('#screenInfo').innerHTML = 
        " Full Screen Width Size = " + fullScreenWidth + "<br>" +
        " Full Screen Height Size = " + fullScreenHeight + "<br>" +
        " Available Screen Width Size = " + availableScreenWidth + "<br>" +
        " Availabel Screen Height Size = " + availableScreenHeight + "<br>" +
        " Screen Color Depth = " + colorDepth + " bits";
</script>
