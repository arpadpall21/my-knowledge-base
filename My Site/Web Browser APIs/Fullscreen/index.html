MySite > Web Browser APIs > Fullscreen
Method / Property Description
element.requestFullscreen() - request the user agent to display the element on full screen
- returns a promise that is resolved on successful fullscreen
document.exitFullscreen() - request the user agent to exit from full screen
- returns a promise that is resolved on successful exit fullscreen
|
element.fullscreenEnabled - returns true if user agent allows fullscreen mode, otherwise returns false
element.fullscreenElement - returns true if fullscreen mode is enabled, otherwise returns null
|
events
fullscreenchange - triggers when fullscreen mode is toggled on|off
fullscreenerror - triggers on fullscreen toggle error

Description and Demonstration :

- The Fullscreen allows us to display an element in fullscreen

Hello World!

(auto disables after 5 seconds) 
// HTML ----------------------------------------
    <div id="full-scr">
      <p> Hello World! </p>
    </div>
    <button onMouseup="fullscreen()"> Enable Fullscreen </button>
    
    <script>
    function fullscreen() {
      if (!document.fullscreenElement && document.fullscreenEnabled) {
        document.getElementById('full-scr').requestFullscreen()
          .then(() => console.log('Fullscreen enabled'));
        
        setTimeout(() => {
          document.exitFullscreen()
            .then(() => console.log('Fullscreen disabled'));
        }, 5000)
      }
    }
    </script>
  
// CSS ----------------------------------------
    #full-scr {
      display: flex;
      justify-content: center;
      align-items: center;
      background-color: palegreen;
    }
