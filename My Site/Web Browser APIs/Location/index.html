Methods / Properties Description
location.assign(URL:str) - loads the new page (document) in the current window (original page will be stored in History [can navigate back])
location.replace(URL:str) - replaces the current page (document) with the specified one (original page won't be stored in History [cannot use the back button to navigate back!])
location.reload(bol

false (Default) - reloads from cache

true - forces to make a new request to the server

) 		- reloads the current page (document)
|
location.host - gets or sets the host + port parts of the current URL
location.href - gets or sets the full URL of the current URL
location.origin - gets or sets the protocol + host + port parts of the current URL
location.hostname - gets or sets the host part of the current URL
location.port - gets or sets the port part of the current URL (as a string)
location.protocol - gets or sets the protocol part of the current URL
location.pathname - gets or sets or path part of the current URL
location.search - gets or sets the querystring part of the current URL
location.hash - gets or sets document fragment part of the current URL

Description and Demonstration :

- the location object holds information on the current URL

DEMO 
// href / host / hostname / pathname / protocol / origin / port / search / hash -------
    location.href;                                                                     // -> 'http://localhost:1000/Web%20Browser%20APIs/Location%20API/index.html'
    location.host;                                                                     // -> 'localhost:1000'
    location.hostname;                                                                 // -> 'localhost'
    location.pathname;                                                                 // -> '/Web%20Browser%20APIs/Location%20API/index.html'
    location.protocol;                                                                 // -> 'http:'
    location.origin;                                                                   // -> 'http://localhost:1000'
    location.port;                                                                     // -> '1000'
    location.search;                                                                   // -> ''
    location.hash;                                                                     // -> ''
    
// assing() / replace() / reload() ----------------------------------------------------
    window.location.assign("https://www.w3schools.com");                               // new page (document) will be loaded in the current window
    window.location.replace("https://www.w3schools.com");                              // new page (document) will replace the current one
    
    window.loaction.reload(false);                                                     // reloads the page (does the same thing than the borwser's "reload" button) 
    window.location.reload(true);                                                      // reloads the page from the server
