Correlation

- Correlation is a measure of dependence between two boolean values (mostly used in probability calculation)

- the result closest to 1 means positive correlation, closest -1 means negative correlation, and 0 means no correlation at all 

    function correlation(table) {
        return (table[3] * table[0] - table[2] * table[1]) /
            Math.sqrt((table[2] + table[3]) *
            (table[0] + table[1]) *
            (table[1] + table[3]) *
            (table[0] + table[2]));
    }
    correlation(true true, true false, false true, false false);
        true true : argument specifies the number of times when both bolleans are "true"
        true false & false true  : argument specifies the number of times when one argument is "ture" and the other one is "false" ()
                                   these two arguments are interchangable
        false false : argument specifies the number of times when both bolleans are "false"

    }
    correlation([14, 21, 74, 54]);      
        // -> -0.1467546654051646
        // 14 - times when both booleans "true"
        // 21 and 74 - times when one "true" and the other is "false" (these values are interchangables)
        // 54 - times when both booleans are "false"

Live Example :

- we need a small exercies to describe this

We are going to measure the correlation (relation) between the fact that BoB fall asleep because of drinking beer in 30 days, so we know that...

  1. - 11 days he drank beer and felt asleep (both situations are "true")
  2. - 3 days he drank beer but didn't fall asleep (one situation is "true" the other one is "false")
  3. - 7 days he did not drink beer but felt asleep (one situation is "true" the other one is "false")
  4. - 9 days he did not drink beer neighter left asleep (both situations are "false")
  • a result closest to 1 means higher chance that Bob fall asleep because of the beer
  • a result closest to -1 means higher chance that Bob DID NOT fall asleep because of the beer
  • if the result is 0 that means that the beer has ho effect on Bob's sleepiness
