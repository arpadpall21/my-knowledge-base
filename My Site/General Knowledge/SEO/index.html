Description and Demonstration :

- SEO (Search Engine Optimization) is a technique of enhancing a website's visibility and ranking on search engine result pages (there's a whole business branch built on top of SEO)

- search engines use crawlers (spiders) to collect data about pages on the internet, the collected data is processed and indexed

Concepts:

- keywords : use clear highliy ranked relevant keywords

- backlinks : links to your site refered by other websites (search engines like this)

Sitemap XML and robot.txt

Sitemap XML

- a map of the website on XML format, specifies different endpoints their creation date, refresh and priority [link]

- sitemaps should be exposed (ex: https://mySite.com/sitemap.xml), this url can be submitted to serach engines

The robot.txt file

- specifies pages of your site that should not be listed by search engines, the robot.txt file should be exposed on the root domain (ex: https://mySite.com/robot.txt)

- if you don't provide any robot.txt that means you allow serach engines to crawl all your pages

- robot.txt is not a security feature, it's just a convention in order to facilitate serach engine listing (ex: malicious crawlers ingnore it) 

// robot.txt (example) -------------------------
    User-agent: userAgent|*       // search engine user agents (* = all)
    Allow|Disallow: urlPath       // allows|disallow the page to be crawled by the serach engine (ex: /public/)   
    ...
    
    Sitemap: sitemapUrl           // sitemap url (ex: https://www.example.com/sitemap2.xml or /sitemap.xml absolute url recommended)
    ...                           // other sitemap urls
